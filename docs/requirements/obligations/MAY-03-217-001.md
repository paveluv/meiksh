## Extracted Text

> A set of values that may be stored in a floating type but that are neither Inf nor valid floating-point numbers. Not all systems support NaN values.

## Context

**Definition only — no shell relevance.** Defines "NaN (Not a Number)" as values storable in floating types that are neither Inf nor valid numbers. Not all systems support NaN.

**Shell impact:** None for the standard shell. The shell's arithmetic expansion uses integer arithmetic only. If `meiksh` implements floating-point arithmetic as an extension (like ksh), NaN handling would become relevant. For POSIX compliance, no action required.
