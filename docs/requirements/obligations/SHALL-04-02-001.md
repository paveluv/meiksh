## Extracted Text

> Functions that suspend the execution of the calling thread shall not cause the execution of other threads to be indefinitely suspended.

## Context

**Definition only — no shell relevance.** States that functions suspending the calling thread SHALL NOT cause other threads to be indefinitely suspended. This is a concurrent execution guarantee for the System Interfaces.

**Shell impact:** None for the shell command language. If `meiksh` uses threads internally, this is an OS/libc guarantee it can rely on. No action required.
