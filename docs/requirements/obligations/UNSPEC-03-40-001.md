## Extracted Text

> A character that, in the output stream, should cause printing (or displaying) to occur one column position previous to the position about to be printed. If the position about to be printed is at the beginning of the current line, the behavior is unspecified. It is the character designated by '\b' in the C language. It is unspecified whether this character is the exact sequence transmitted to an output device by the system to accomplish the backspace function. The backspace defined here is not necessarily the ERASE special character.

## Context

**Definition — moderate shell relevance.** Defines the backspace character `<backspace>` as `\b` (0x08). It should cause printing to move one column position back. Behavior at the beginning of a line is unspecified. It is unspecified whether this byte is the exact sequence sent to the terminal. The POSIX backspace is not necessarily the ERASE special character.

**Shell impact:** The shell must recognize `\b` in escape-sequence contexts: `printf '\b'`, `echo` with XSI interpretation, and `$'\b'` quoting. The shell writes byte 0x08; terminal handling is outside shell scope. The interactive line editor must not confuse backspace character output with the ERASE key.
