## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: [XSI] Causes signal delivery to occur on an alternate stack.

## Context

This is the [XSI] extension defining the semantics of `SA_ONSTACK`: when set in `sa_flags`, signal delivery occurs on the alternate signal stack rather than the normal process stack. The alternate stack is set up via `sigaltstack()`. For `meiksh`, this is not typically relevant since the shell does not generally need alternate signal stacks. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 522–523).
