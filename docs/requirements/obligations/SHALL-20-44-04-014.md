## Extracted Text

> (The letter ell.) List the commands rather than invoking an editor on them. The commands shall be written in the sequence indicated by the first and last operands, as affected by -r, with each command preceded by the command number.

## Context

This obligation (duplicate extraction of the `-l` option description) requires that when `fc -l` is used, commands are listed to stdout instead of being opened in an editor. The output sequence is determined by `first` and `last` operands and the `-r` flag. Each command must be preceded by its history command number (unless `-n` suppresses it). The format is `"%d\t%s\n"` per line. Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
