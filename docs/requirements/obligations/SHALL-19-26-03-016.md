## Extracted Text

> The shell shall write to standard error a trace for each command after it expands the command and before it executes it. It is unspecified whether the command that turns tracing off is traced.

## Context

**Behavior:** The `-x` (`xtrace`) option: the shell must write to **standard error** a trace for each command **after** expanding it and **before** executing it. The trace output is prefixed by the expanded value of `PS4` (default `+ `). It is **unspecified** whether the command that turns tracing off (`set +x`) is itself traced.

**Applies to:** All shell modes.
