## Extracted Text

> If the utility utility is invoked, the exit status of time shall be the exit status of utility; otherwise, the time utility shall exit with one of the following values:: The utility specified by utility was found but could not be invoked.

## Context

**Behavior:** This is the description for exit status 126 of the `time` utility. When the utility operand names an executable file that was found via PATH search (or directly by path), but the system could not invoke it, `time` shall exit with status 126.

**Conditions:** Typical causes include: the file lacks execute permission, the file is not a recognized executable format, or exec() fails with an error other than ENOENT.

**Error handling:** The shell should write a diagnostic message to stderr before exiting with 126, indicating that the utility was found but could not be executed.
