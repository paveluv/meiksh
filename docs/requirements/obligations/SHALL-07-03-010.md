## Extracted Text

> The escape character shall default to <backslash>, which is the character used in all examples shown in this volume of POSIX.1-2024.

## Context

**Behavior:** The default escape character in locale definition files is backslash (`\`). This is used for encoding constants and line continuation within the file.

**Shell relevance:** The shell does not parse locale definition files. This is `localedef` input syntax. No shell implementation action needed.
