## Extracted Text

> The following exit values shall be returned:: 0

## Context

**Behavior:** The cd built-in must also return exit status 0 in a second case. This is the exit value label "0" corresponding to the description in SHALL-20-14-14-005 (directory changed successfully, -e is NOT in effect, -P IS in effect, but PWD could not be determined). Even though PWD is incorrect, the exit is still 0 because -e was not specified.

**Applies to:** cd -P invocations where the chdir succeeds but the physical path cannot be determined, and -e is not used.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 245.
