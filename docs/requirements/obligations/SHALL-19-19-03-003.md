## Extracted Text

> The dot special built-in shall support XBD 12.2 Utility Syntax Guidelines, except for Guidelines 1 and 2.

## Context

**Behavior:** The `.` (dot) special built-in must support XBD 12.2 Utility Syntax Guidelines, **except** for Guidelines 1 and 2. Guideline 1 requires utility names of 2-9 characters; Guideline 2 requires names from the portable character set. The `.` command is a single special character, which violates these guidelines by design.

**Implementation:** The dot built-in should accept the `--` end-of-options marker and pass remaining arguments as positional parameters to the sourced file.

**Applies to:** All shell modes.
