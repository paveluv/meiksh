## Extracted Text

> The following commands shall be recognized in command mode:: Undo the last command that changed the edit line. This operation shall not undo the copy of any command line to the edit line.

## Context

**Behavior:** In vi command mode, `u` undoes the last command that changed the edit line. This is a single-level undo (pressing `u` again re-does the change, toggling).

**Restriction:** This operation shall NOT undo the copy of a command line to the edit line (when a history entry replaces the edit line per SHALL-20-110-13-05-002, that copy is not undoable).

**Implementation:** Maintain a single undo buffer that stores the previous state of the edit line before each modification. On `u`, swap the current line with the undo buffer. History-to-edit-line copies do not populate the undo buffer.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 620–621.
