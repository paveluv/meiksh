> An <asterisk> is a pattern that shall match multiple characters, as described in 2.14.2 Patterns Matching Multiple Characters.

## Context

**Duplicate of SHALL-19-14-01-011: `*` matches multiple characters per 2.14.2.**

Same obligation as SHALL-19-14-01-011. An unquoted, unescaped `*` not inside a bracket expression **shall** match multiple characters as described in 2.14.2 Patterns Matching Multiple Characters. See SHALL-19-14-01-011 for full details.
