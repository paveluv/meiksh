## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: Transmit a STOP character, intended to suspend input data.

## Context

**Behavior:** This is the description of TCIOFF: **transmit a STOP character, intended to suspend input data**. When `tcflow(fd, TCIOFF)` is called, a STOP character (typically ^S) is sent to the terminal to signal that the sender should pause data transmission.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** See SHALL-14-77-03-09-009. This sends the XOFF character to request the terminal pause sending input.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 484.
