> The following environment variables shall affect the execution of command:

## Context

**Behavior:** The `command` built-in must consult and be affected by the following environment variables: LANG, LC_ALL, LC_CTYPE, LC_MESSAGES, NLSPATH (XSI), and PATH. The shell implementation must read these variables from the current execution environment during `command` execution.
