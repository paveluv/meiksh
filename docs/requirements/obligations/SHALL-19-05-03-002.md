## Extracted Text

> The following variables shall affect the execution of the shell:

## Context

This is the introductory statement for the list of shell variables that have special meaning. It establishes that the variables enumerated in the following obligations (ENV, HOME, IFS, LANG, LC_ALL, LC_COLLATE, LC_CTYPE, LC_MESSAGES, LINENO, NLSPATH, PATH, PPID, PS1, PS2, PS4, PWD) are mandatory for the shell to recognize and handle with their defined semantics.

**Required behavior:**
- The shell must recognize each of the variables listed in subsequent obligations and implement their specified effects on shell execution.
- The shell must not ignore or fail to act on any of these variables when they are set.

**Conditions:** Applies universally to all shell execution contexts.

See 2.5.3 "Shell Variables" in `utilities/V3_chap02.html`.
