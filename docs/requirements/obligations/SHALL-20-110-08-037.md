## Extracted Text

> [UP] Determine a decimal number representing the limit to the number of previous commands that are accessible. If this variable is unset, an unspecified default greater than or equal to 128 shall be used. The maximum number of commands in the history list is unspecified, but shall be at least 128. An implementation may choose to access this variable only when initializing the history file, as described under HISTFILE . Therefore, it is unspecified whether changes made to HISTSIZE after the history file has been initialized are effective.

## Context

**Behavior:** This is a duplicate extraction of the HISTSIZE variable description (same source lines as SHALL-20-110-08-009). The full behavior specification is identical:

1. **When set:** Decimal number limiting accessible previous commands.
2. **When unset:** Default >= 128 shall be used.
3. **Minimum:** History list shall support at least 128 commands.
4. **Initialization:** May be read only once at history init time; later changes may have no effect.

**Feature gate:** [UP] (User Portability) option.
