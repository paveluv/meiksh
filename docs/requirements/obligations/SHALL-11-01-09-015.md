## Extracted Text

> If the ISIG flag is set, receipt of the SUSP character shall cause a SIGTSTP signal to be sent to all processes in the foreground process group for which the terminal is the controlling terminal, and the SUSP character shall be discarded when processed.

## Context

**Behavior:** The SUSP special character (typically ^Z) — duplicate/alternate extraction of the same spec text as SHALL-11-01-09-006:
1. Recognized on input only when ISIG is set in `c_lflag`.
2. Generates SIGTSTP sent to all processes in the foreground process group.
3. The SUSP character shall be **discarded** when processed.

**Conditions:** ISIG must be set in `c_lflag`. The terminal must be the controlling terminal.

**Shell relevance:** See SHALL-11-01-09-006 for full details. ^Z suspends the foreground job; the shell detects the stop and reports it for job control.

**Spec reference:** POSIX.1-2024, §11.1.9 Special Characters (tag_11_01_09), `basedefs/V1_chap11.html` line 246.
