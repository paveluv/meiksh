## Extracted Text

> The readonly special built-in shall be a declaration utility. Therefore, if readonly is recognized as the command name of a simple command, then subsequent words of the form name=word shall be expanded in an assignment context. See 2.9.1.1 Order of Processing.

## Context

**Behavior:** `readonly` is a **declaration utility**, identical to `export` in this regard. When recognized as the command name of a simple command, subsequent `name=word` forms are expanded in an **assignment context**. See 2.9.1.1 Order of Processing.

**Applies to:** All shell modes.
