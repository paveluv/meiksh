## Extracted Text

> The following commands shall be recognized in command mode:: Repeat the most recent / or ? command, reversing the direction of the search. If there is no previous / or ?, the terminal shall be alerted and the current command line shall remain unchanged.

## Context

**Behavior:** In vi command mode, `N` repeats the most recent `/` or `?` search command but reverses the direction. If there is no previous `/` or `?` command, the terminal shall be alerted and the command line remains unchanged.

**Implementation:** Same as `n` but reverse the search direction before executing.
