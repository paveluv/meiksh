## Extracted Text

> When not inside a bracket expression, the following shall match a single character:: an ERE ordinary character

## Context

When not inside a bracket expression, an ERE ordinary character (any character that is not an ERE special character) must match exactly itself. An ordinary character is any character in the supported character set except the ERE special characters (`.`, `[`, `\`, `(`, `)`, `*`, `+`, `?`, `{`, `|`, `^`, `$`). For example, `a` matches the character `a`, `5` matches `5`, etc. The ERE engine must treat each such character as a single-character-matching atom.
