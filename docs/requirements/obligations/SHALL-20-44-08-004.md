## Extracted Text

> The following environment variables shall affect the execution of fc:: HISTFILE

## Context

The `fc` builtin must recognize the `HISTFILE` environment variable, which names the pathname of the command history file. If `HISTFILE` is not set, the shell may use `$HOME/.sh_history` as the default. See SHALL-20-44-08-005 and SHALL-20-44-08-019 for the full behavioral requirements including initialization, access control, and deletion ordering. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
