## Extracted Text

> The following environment variables shall affect the execution of sh:: MAIL

## Context

**Behavior:** The MAIL environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-023).

**Applies to:** Interactive shells (User Portability [UP] option).

**Details:** MAIL provides the pathname of the user's mailbox file for incoming mail notification. The shell checks this file's creation/modification time at intervals defined by MAILCHECK. MAIL is only consulted when MAILPATH is not set. See SHALL-20-110-08-023 for the full behavior specification.
