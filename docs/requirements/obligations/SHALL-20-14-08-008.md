## Extracted Text

> The following environment variables shall affect the execution of cd:: LC_ALL

## Context

**Behavior:** The `LC_ALL` environment variable must affect `cd` execution. LC_ALL is recognized as an internationalization override variable for the cd built-in. When set to a non-empty string, it overrides all other LC_* and LANG variables.

**Applies to:** Locale determination for cd.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 188.
