## Extracted Text

> The following exit values shall be returned:: 1

## Context

**Behavior:** `getopts` shall return exit status 1 when the end of options is encountered. This terminates the typical `while getopts ...` loop. End of options occurs when:
- A `--` argument is found (not an option-argument).
- A non-option argument (not starting with `-`) is found.
- All options have been processed.

**Spec reference:** POSIX.1-2024, `getopts` EXIT STATUS, line 206 of `utilities/getopts.html`.
