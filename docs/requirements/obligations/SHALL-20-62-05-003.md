## Extracted Text

> The following operand shall be supported:: Specifies the jobs for which the status is to be displayed. If no job_id is given, the status information for all jobs shall be displayed. The format of job_id is described in XBD 3.182 Job ID.

## Context

**Behavior:** The `job_id` operand specifies which jobs to display:
- If one or more `job_id` operands are given, display status only for those specific jobs.
- If no `job_id` is given, display status for all jobs (per SHALL-20-62-04-007: all running, suspended, and changed-status jobs).

**Job ID format:** Defined in XBD 3.182 — includes `%n`, `%string`, `%?string`, `%%`/`%+`, `%-`.

**Error handling:** If a specified `job_id` does not match any known job, the implementation should report an error.

**Spec reference:** POSIX.1-2024, `jobs` OPERANDS, lines 69-70 of `utilities/jobs.html`; XBD 3.182.
