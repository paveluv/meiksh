## Context

**Behavior:** BRE anchoring: A BRE can be limited to matching expressions that begin or end a string using `^` (beginning) and `$` (end) anchors. These are considered BRE anchors in specific contexts described in the following sub-obligations.

**Shell relevance:** Shell pattern matching does not use `^` and `$` as anchors. In `case` patterns, the entire string must match the pattern (implicit full-string anchoring). In filename expansion, patterns are matched against each filename component. This BRE anchoring concept does not apply to shell patterns.
