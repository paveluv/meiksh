## Extracted Text

> The standard error shall be used only for diagnostic messages and the warning message specified in EXIT STATUS.

## Context

The `shift` special builtin (section 2.27 in utilities/V3_chap02.html#tag_19_27) must use standard error **only** for diagnostic messages **and** the warning message specified in the EXIT STATUS section. The `shift` builtin produces no stdout output.

Unlike most other builtins which say "only for diagnostic messages", `shift` specifically also allows a **warning** message on stderr. Per the EXIT STATUS section, if the operand `n` is greater than `$#` (the number of positional parameters), `shift` writes a warning to stderr and returns exit status >0.

Diagnostic messages that may be written to stderr include:
- Operand `n` is not a valid non-negative integer
- Warning that `n` exceeds the count of positional parameters

See 2.27 shift in utilities/V3_chap02.html#tag_19_27.
