## Extracted Text

> Otherwise, the following exit values shall be returned:: The utility specified by command_name was found but could not be invoked.

## Context

**Behavior:** Description for exit status 126: The utility was found in PATH (or as a built-in) but could not be invoked. Common causes:
- File lacks execute permission.
- File is not a valid executable (bad magic number, wrong architecture).
- File is a directory.

The shell must distinguish "found but not invokable" (126) from "not found" (127).

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section EXIT STATUS (tag_20_22_14), line 206.
