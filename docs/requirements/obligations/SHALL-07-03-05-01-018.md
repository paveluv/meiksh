## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: d_fmt

## Context

The `d_fmt` keyword shall be recognized in an LC_TIME locale definition. It introduces the definition of the date format string.

**Shell relevance:** This format is used by `%x` in `date`. In the POSIX locale: "%m/%d/%y". See XBD 7.3.5.1.
