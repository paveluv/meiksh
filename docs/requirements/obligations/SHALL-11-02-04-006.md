## Extracted Text

> If HUPCL is set, the modem control lines for the port shall be lowered when the last process with the port open closes the port or the process terminates. The modem connection shall be broken.

## Context

**Behavior:** When HUPCL is set in `c_cflag`, the modem control lines for the port shall be lowered (causing a hangup) when the last process with the port open closes it or terminates. The modem connection shall be broken.

**Relevance to meiksh:** The `stty` builtin must support setting/clearing HUPCL (via `hupcl`/`-hupcl` or `hup`/`-hup` operands). This is relevant for serial terminals and modem connections but typically has no effect on pseudo-terminals.

**Spec reference:** POSIX.1-2024, §11.2.4 Control Modes (tag_11_02_04), `basedefs/V1_chap11.html` line 1073.
