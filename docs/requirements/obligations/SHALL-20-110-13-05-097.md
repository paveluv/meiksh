## Extracted Text

> The following commands shall be recognized in command mode:: Put a copy of the current contents of the save buffer before the current cursor position. The current cursor position shall be moved to the last character put from the save buffer. A count shall indicate how many copies of the save buffer shall be put.

## Context

**Behavior:** In vi command mode, `P` puts (pastes) the contents of the save buffer BEFORE the current cursor position. The cursor is moved to the last character put from the save buffer. With a count, put that many copies of the save buffer contents.

**Implementation:** Insert save buffer contents before cursor position, repeated `count` times. Set cursor to the last inserted character.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 616–618.
