## Extracted Text

> The following environment variables shall affect the execution of alias:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

`LANG` **shall** provide a default value for internationalization variables that are unset or null when executing `alias`. The precedence is: `LC_ALL` overrides all; individual `LC_*` variables override `LANG` for their category; `LANG` is the fallback. See XBD 8.2 for the full precedence rules. Reference: alias utility ENVIRONMENT VARIABLES, XBD 8.2.
