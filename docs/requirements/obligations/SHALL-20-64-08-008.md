## Extracted Text

> The following environment variables shall affect the execution of kill:: LC_MESSAGES

## Context

**Behavior:** The `LC_MESSAGES` environment variable must be recognized by the `kill` built-in. It determines the locale for formatting diagnostic messages written to stderr. See SHALL-20-64-08-009.
