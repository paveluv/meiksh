## Context

**Non-interactive shell notification (optional, parent):** For every case where an interactive shell writes a job status message "immediately prior to writing the next prompt for input" (deferred notifications for suspended, completed, or terminated background jobs), a **non-interactive** shell **may** also write the same message. The permitted timings are listed in MAY-19-11-002 through MAY-19-11-004.

This is entirely optional. The implementation may choose to support job notifications in non-interactive mode (useful for scripts that use job control), or may suppress them entirely for non-interactive shells.
