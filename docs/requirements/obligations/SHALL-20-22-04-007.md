## Extracted Text

> The following options shall be supported:: -V

## Context

**Behavior:** The `command` built-in must support the `-V` option. When specified, `command -V name` writes a human-readable string to stdout indicating how the shell would interpret `name`, without executing it. The output format includes the command category (function, alias, built-in, reserved word, etc.).

**Applies to:** `command -V command_name` invocations.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section OPTIONS (tag_20_22_04), line 87.
