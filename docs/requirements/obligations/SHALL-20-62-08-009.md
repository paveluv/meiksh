## Extracted Text

> The following environment variables shall affect the execution of jobs:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error and informative messages written to standard output.

## Context

**Behavior:** Describes `LC_MESSAGES` effect on `jobs`: determines locale for both diagnostic messages to stderr and informative messages to stdout. Note that for `jobs`, LC_MESSAGES affects stdout output too (the state strings like "Running", "Done", "Stopped"), not just stderr diagnostics.

**Spec reference:** POSIX.1-2024, `jobs` ENVIRONMENT VARIABLES, line 97 of `utilities/jobs.html`.
