## Extracted Text

> Specify the name of an existing locale which shall be used as the definition of this category. If this keyword is specified, no other keyword shall be specified. Note: This is a localedef keyword, unavailable through nl_langinfo().

## Context

The `copy` keyword in LC_MESSAGES specifies an existing locale whose LC_MESSAGES definition is reused. If `copy` is specified, no other keyword shall be specified in the LC_MESSAGES block. Note: `copy` is a `localedef` keyword, not available through `nl_langinfo()`.

**Shell relevance:** Informational for locale infrastructure. See XBD 7.3.6.
