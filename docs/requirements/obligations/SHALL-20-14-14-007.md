## Extracted Text

> The following exit values shall be returned:: Either the -e option or the -P option is not in effect, and an error occurred.

## Context

**Behavior:** Exit status >0 must be returned when:
1. Either `-e` or `-P` (or both) is NOT in effect (i.e., not using `cd -eP`), AND
2. An error occurred (e.g., directory not found, permission denied, empty string operand).

This covers the common error cases: `cd nonexistent`, `cd ""`, `cd` with unset HOME, etc. The specific non-zero value is not mandated (any value >0 is acceptable).
