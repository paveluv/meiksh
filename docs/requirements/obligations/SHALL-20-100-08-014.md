## Extracted Text

> The following environment variables shall affect the execution of read:: PS2

## Context

**Behavior required:** The `PS2` environment variable must affect the execution of the `read` builtin. PS2 provides the continuation prompt string.

**Condition:** When `read` encounters line continuation (backslash-newline without `-r`) in an interactive shell with terminal stdin.

**Implementation detail:** When `read` needs to prompt for a continuation line (see SHALL-20-100-03-003), it writes the value of PS2 to stderr. The default value of PS2 is `"> "`. The shell should expand PS2 (performing parameter expansion, etc.) before displaying it, consistent with how PS1 is handled.

**Spec reference:** Section ENVIRONMENT VARIABLES of `utilities/read.html`, line 169.
