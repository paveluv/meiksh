## Context

**Behavior:** The `sh` utility shall return exit status 126 under the condition described in SHALL-20-110-14-007. This is the exit value label "126" in the exit status definition list.

**Applies to:** The `sh` utility when invoked with a command_file that cannot be executed due to ENOEXEC.

**Details:** Exit status 126 indicates the command_file was found but could not be executed. Specifically, an ENOEXEC error occurred — the file exists and has execute permission but is not in a recognized executable format (e.g., a binary for a different architecture, or a script without a valid `#!` line that the system exec cannot handle). See SHALL-20-110-14-007 for the detailed condition.
