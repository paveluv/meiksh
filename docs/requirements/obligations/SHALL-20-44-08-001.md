## Extracted Text

> The following environment variables shall affect the execution of fc:

## Context

The `fc` builtin must honor the following environment variables: `FCEDIT` (default editor), `HISTFILE` (history file path), `HISTSIZE` (history size limit), `LANG`, `LC_ALL`, `LC_CTYPE`, `LC_MESSAGES`, and `NLSPATH` (XSI). Each variable has specific effects on `fc` behavior as described in companion obligations SHALL-20-44-08-002 through SHALL-20-44-08-020. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
