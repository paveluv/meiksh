## Extracted Text

> The character used to indicate that the characters following shall be interpreted in a special way, as defined later in this section. This shall default to <backslash> ('\\'), which is the character used in all the following text and examples, unless otherwise noted.

## Context

**Behavior:** The `<escape_char>` declaration in a charmap file defines the escape character used for encoding constants (e.g., `\x41`, `\d65`, `\101`). It defaults to backslash (`\`).

**Shell relevance:** The shell does not parse charmap files. This escape character is specific to charmap file syntax and is unrelated to the shell's backslash escape processing. No shell implementation action needed beyond awareness that charmap files use `\` as escape by default.
