## Extracted Text

> The <termios.h> header shall define the following symbolic constant:: Size of the array c_cc for control characters.

## Context

**Behavior:** NCCS is described as the **size of the array `c_cc` for control characters**. The value of NCCS is implementation-defined but must be at least large enough to hold all required subscript indices.

**Conditions:** NCCS defines the dimension of `c_cc` in `struct termios`.

**Shell relevance:** meiksh uses `c_cc[NCCS]` for all special character and MIN/TIME parameter access. The actual value of NCCS varies by platform (commonly 11, 16, 20, or 32) but is always sufficient for the standard subscripts.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 69.
