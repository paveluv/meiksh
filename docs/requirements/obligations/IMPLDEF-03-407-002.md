## Extracted Text

> If the initial user program field is null, an implementation-defined program is used.

## Context

**Definition — moderate shell relevance.** States that if the "initial user program" field in the user database is null, an implementation-defined program is used. The initial user program is typically the user's login shell.

**Shell impact:** If the user database entry (e.g., `/etc/passwd`) has an empty shell field, the login system uses an implementation-defined default (usually `/bin/sh`). This means `meiksh` may be invoked as the default shell even when not explicitly configured. The shell itself does not read this field; the login system does. However, `meiksh` should be prepared to function as the login shell for any user.
