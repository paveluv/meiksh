## Extracted Text

> The following commands shall be recognized in command mode:: Enter insert mode after the current cursor position. Characters that are entered shall be inserted before the next character.

## Context

**Behavior:** In vi command mode, `a` enters insert mode with the insertion point AFTER the current cursor position. Characters typed are inserted before the next character.

**Implementation:** Set mode to insert, set insertion point to cursor_pos + 1.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 534–535.
