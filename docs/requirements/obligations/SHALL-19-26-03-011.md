## Extracted Text

> This option shall be supported if the implementation supports the User Portability Utilities option. When this option is enabled, the shell shall perform job control actions as described in 2.11 Job Control. This option shall be enabled by default for interactive shells.

## Context

**Behavior:** The `-m` (`monitor`) option (requires User Portability Utilities option):
1. When enabled, the shell performs job control actions as described in 2.11 Job Control.
2. This option must be **enabled by default** for interactive shells.
3. Job control includes: placing commands in separate process groups, managing foreground/background jobs, reporting job status changes.

**Applies to:** Primarily interactive shells, but can be set in non-interactive shells.

**Spec reference:** `set` DESCRIPTION in `utilities/V3_chap02.html`, lines 4836-4837. Cross-ref: 2.11 Job Control.
