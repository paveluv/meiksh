## Extracted Text

> The following commands shall be recognized in command mode:: Repeat the most recent f, F, t, or T command. Any number argument on that previous command shall be ignored. However, reverse the direction of that command.

## Context

**Behavior:** In vi command mode, `,` repeats the most recent `f`, `F`, `t`, or `T` command but reverses the search direction. Any count from the original command is ignored; the new count applies.

**Implementation:** Same as `;` but reverse the direction: `f`↔`F`, `t`↔`T`. This is a motion command.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 531–532.
