## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `exec` special builtin (section 2.21 in utilities/V3_chap02.html#tag_19_21) must use standard error **only** for diagnostic messages. It must not write any non-diagnostic output to stderr.

Diagnostic messages that may be written to stderr include:
- Command not found
- Permission denied
- Redirection errors (e.g., file not found for `exec 3< nofile`)

See 2.21 exec in utilities/V3_chap02.html#tag_19_21.
