## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: d_t_fmt

## Context

The `d_t_fmt` keyword shall be recognized in an LC_TIME locale definition. It introduces the definition of the date-and-time format string.

**Shell relevance:** This format is used by `%c` in `date`. In the POSIX locale: "%a %b %e %H:%M:%S %Y". See XBD 7.3.5.1.
