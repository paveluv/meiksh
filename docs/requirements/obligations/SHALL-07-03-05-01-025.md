## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: Define the appropriate time representation in the 12-hour clock format with am_pm, corresponding to the %r conversion specification. The operand shall consist of a string and can contain any combination of characters and conversion specifications. If and only if the 12-hour format is not supported in the locale, the string shall be empty.

## Context

The `t_fmt_ampm` keyword defines the appropriate time representation in 12-hour clock format with AM/PM, corresponding to the `%r` conversion specification. The operand shall consist of a string that can contain any combination of characters and conversion specifications. If and only if the 12-hour format is not supported in the locale, the string shall be empty.

**POSIX locale value:** "%I:%M:%S %p".

**Shell relevance:** Used by `date +%r`. See XBD 7.3.5.1.
