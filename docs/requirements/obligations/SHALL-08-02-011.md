## Extracted Text

> Specify the pathname to the root directory of the messages object hierarchy the gettext and ngettext utilities use during the search for messages objects. If present, it shall replace the default root directory pathname. [XSI] NLSPATH has precedence over TEXTDOMAINDIR .

## Context

**Behavior:** The `TEXTDOMAINDIR` variable specifies the pathname to the root directory of the messages object hierarchy used by `gettext` and `ngettext` utilities. If set, it shall replace the default root directory pathname for message object searching. However, `NLSPATH` (XSI) takes precedence over `TEXTDOMAINDIR`.

**Shell relevance:** The shell must propagate `TEXTDOMAINDIR` to child processes. If the shell itself uses gettext-based message catalogs for its diagnostics, it should respect this variable (but give `NLSPATH` higher priority per the XSI rule).
