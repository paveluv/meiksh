## Extracted Text

> The following environment variables shall affect the execution of cd:: LANG

## Context

**Behavior:** The `LANG` environment variable must affect `cd` execution. LANG provides a default value for internationalization variables (LC_*) that are unset or null. For cd, this primarily affects the locale used for diagnostic messages (via LC_MESSAGES) and character interpretation (via LC_CTYPE). The shell must consult LANG as a fallback when specific LC_* variables are not set.

**Applies to:** Locale determination for cd diagnostic messages and character interpretation.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 184. See also XBD 8.2 Internationalization Variables.
