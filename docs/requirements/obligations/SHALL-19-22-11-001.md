## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `exit` special builtin (section 2.22 in utilities/V3_chap02.html#tag_19_22) must use standard error **only** for diagnostic messages. It must not write any non-diagnostic output to stderr.

Diagnostic messages that may be written to stderr include:
- Invalid exit status argument (not a valid number)
- In interactive shells, a warning about stopped jobs when exit is attempted (though this specific warning behavior is covered in other obligations)

See 2.22 exit in utilities/V3_chap02.html#tag_19_22.
