## Extracted Text

> The following commands shall be recognized in command mode:: Delete all characters from the current cursor position to the end of the line. The deleted characters shall be placed in the save buffer.

## Context

**Behavior:** In vi command mode, `D` deletes all characters from the current cursor position to the end of the line and places them in the save buffer. Equivalent to `d$`.
