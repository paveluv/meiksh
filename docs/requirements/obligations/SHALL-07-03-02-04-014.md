## Extracted Text

> An ellipsis can be used as an operand if the collating element was an ellipsis, and shall be interpreted as the value of each character defined by the ellipsis.

## Context

When both the collating element and its weight operand are ellipses in the LC_COLLATE collation order section, the weight for each character in the ellipsis range is its own collation position (i.e., each character in the range gets a unique weight equal to its position). This is the standard way to define a range of characters with unique, code-point-ordered weights. The shell is affected because this is the mechanism used in many locale definitions (including the POSIX locale) to establish the collation order for ASCII character ranges, which directly determines bracket expression range matching behavior.

See: `basedefs/V1_chap07.html` §7.3.2.4 Collation Order.
