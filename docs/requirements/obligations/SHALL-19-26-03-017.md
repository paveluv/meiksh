## Extracted Text

> The default for all these options shall be off (unset) unless stated otherwise in the description of the option or unless the shell was invoked with them on; see sh.

## Context

**Behavior:** The default state for all `set` options is **off** (unset), unless:
1. The option's description states otherwise (e.g., `-m` is on by default for interactive shells).
2. The shell was invoked with the option enabled (e.g., `sh -e script.sh` starts with `-e` on).

**Applies to:** All shell modes.
