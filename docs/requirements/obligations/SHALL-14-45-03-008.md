## Extracted Text

> The <signal.h> header shall define the following data types:: [CX] Integer or structure type of an object used to represent sets of signals.

## Context

This describes `sigset_t` semantics (CX extension): it can be either an integer or structure type, used to represent signal sets. The implementation must not assume `sigset_t` is a specific size or type—it must use the POSIX signal set manipulation functions (`sigemptyset`, `sigfillset`, `sigaddset`, `sigdelset`, `sigismember`) rather than direct bit manipulation. The shell must manipulate signal sets only through these standard APIs.
