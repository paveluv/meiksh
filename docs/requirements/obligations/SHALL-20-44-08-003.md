## Extracted Text

> The following environment variables shall affect the execution of fc:: This variable, when expanded by the shell, shall determine the default value for the -e editor option's editor option-argument. If FCEDIT is null or unset, ed shall be used as the editor.

## Context

This describes the `FCEDIT` variable's behavior for `fc`. The implementation must: (1) Expand `FCEDIT` using normal shell expansion when determining the default editor. (2) Use the expanded value as the editor name (resolved via `PATH`). (3) If `FCEDIT` is null (empty string) or unset, the implementation must use `ed` as the editor. The priority chain is: `-e editor` argument > `FCEDIT` value > `ed`. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
