## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use as flags in the c_iflag field. The c_iflag field describes the basic terminal input control.

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for the `c_iflag` field (input modes). These include at minimum: BRKINT, ICRNL, IGNBRK, IGNCR, IGNPAR, INLCR, INPCK, ISTRIP, IXANY, IXOFF, IXON, PARMRK.

**Conditions:** Must be defined as bitmask constants suitable for bitwise-OR combination.

**Shell relevance:** meiksh uses these flags when configuring terminal input:
- ICRNL: CR-to-NL translation (essential for Enter key)
- IXON: XON/XOFF output control (may disable for line editor)
- ISTRIP: strip 8th bit (should be off for UTF-8)
- INLCR, IGNCR: NL/CR handling
The `stty` builtin must support all these flag names.

**Spec reference:** POSIX.1-2024, `<termios.h>` Input Modes (tag_14_77_03_02), `basedefs/termios.h.html` lines 220–221.
