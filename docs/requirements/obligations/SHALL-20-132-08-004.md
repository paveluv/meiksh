## Extracted Text

> The following environment variables shall affect the execution of umask:: LC_ALL

## Context

**Behavior:** If LC_ALL is set to a non-empty string value, it shall override all other internationalization variables (LANG, LC_CTYPE, LC_MESSAGES, etc.) for the `umask` utility.

**Conditions:** LC_ALL has the highest precedence among locale variables. When set, all locale categories use the value of LC_ALL.
