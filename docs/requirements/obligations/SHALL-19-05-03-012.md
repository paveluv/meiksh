## Extracted Text

> The following variables shall affect the execution of the shell:: The value of this variable overrides the LC_* variables and LANG , as described in XBD 8. Environment Variables.

## Context

This defines the semantics of the `LC_ALL` variable.

**Required behavior:**
- When `LC_ALL` is set (to a non-empty value), its value overrides ALL individual `LC_*` variables (`LC_CTYPE`, `LC_COLLATE`, `LC_MESSAGES`, etc.) and `LANG`.
- This provides a single-variable mechanism to force all locale categories to the same value.

**Conditions:** Applies whenever the shell determines locale settings. `LC_ALL` has the highest precedence in the internationalization variable hierarchy.

See 2.5.3 "Shell Variables" (`LC_ALL`) and XBD 8 "Environment Variables" in `basedefs/V1_chap08.html`.
