## Extracted Text

> The following environment variables shall affect the execution of getopts:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

**Behavior:** `LANG` provides default values for any internationalization variables (LC_CTYPE, LC_MESSAGES, etc.) that are unset or null. The precedence order is: LC_ALL overrides everything, then specific LC_* variables, then LANG as fallback. See XBD 8.2 for the full precedence rules.

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, lines 140-141 of `utilities/getopts.html`; XBD 8.2.
