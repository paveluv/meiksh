## Extracted Text

> Each AND-OR list shall be expanded and executed in the order specified.

## Context

**Behavior:** Each AND-OR list in a sequential list MUST be expanded and executed in the order they appear. No reordering or parallel execution is permitted.

**Applies to:** Sequential lists (`cmd1; cmd2; cmd3`). Each AND-OR list is fully expanded and executed before the next one begins.

**Implementation detail:** Execute AND-OR lists left-to-right. Word expansion for each list happens just before its execution, not ahead of time. This ensures that side effects of earlier commands (e.g., variable assignments) are visible to later commands' expansions.

**Spec reference:** Section 2.9.3.2, line 2185 in `utilities/V3_chap02.html`.
