## Extracted Text

> The string that shall be used as the local currency symbol.

## Context

The `currency_symbol` value defines the local currency symbol string (e.g., "$" for US dollars). The shell does not directly use this value for formatting. The shell must correctly propagate the LC_MONETARY environment variable so programs can access this through `localeconv()`.

See: `basedefs/V1_chap07.html` §7.3.3 LC_MONETARY.
