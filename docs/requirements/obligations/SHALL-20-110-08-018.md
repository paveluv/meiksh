## Extracted Text

> The following environment variables shall affect the execution of sh:: LC_CTYPE

## Context

**Behavior:** The LC_CTYPE environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-019).

**Applies to:** All shell modes (interactive and non-interactive).

**Details:** LC_CTYPE determines character classification: how bytes are interpreted as characters, which characters are letters (alpha class), and the behavior of character classes in pattern matching. See SHALL-20-110-08-019 for the full description.
