## Extracted Text

> The following operands shall be supported:: Assign the value of string to the alias alias-name.

## Context

When the `alias-name=string` operand form is used, the shell **shall** assign the value of `string` to the alias named `alias-name`. The string becomes the replacement text used during alias substitution (XCU 2.3.1). The string can contain any characters and may include spaces, special characters, or even other alias names. If an alias with that name already exists, it is redefined. Reference: alias utility OPERANDS.
