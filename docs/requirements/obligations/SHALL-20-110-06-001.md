## Extracted Text

> The standard input shall be used only if one of the following is true:

## Context

**Behavior:** The shell's standard input (fd 0) shall only be consumed for reading commands under specific conditions enumerated below (see 001–004). Outside of these conditions, the shell must not read from stdin.

**Conditions:** This is the parent rule; the three qualifying conditions are:
1. The `-s` option is specified (see SHALL-20-110-06-002).
2. Neither `-c` nor any operand (script filename) is specified (see SHALL-20-110-06-003).
3. A command within the script itself reads stdin, e.g., the `read` builtin without redirection (see SHALL-20-110-06-004).

**Applies to:** Both interactive and non-interactive shells. When a script file is provided as an operand (and `-s` is absent), the shell reads commands from that file, NOT from stdin.
