## Extracted Text

> The <signal.h> header shall define the following symbolic constant. The value shall be suitable for use in #if preprocessing directives:

## Context

The `<signal.h>` header must define `SIG2STR_MAX` as a symbolic constant suitable for `#if` preprocessing. This constant specifies the maximum size of a buffer needed to hold a signal name string returned by `sig2str()`, including the terminating null byte. For `meiksh`, this is relevant when converting signal numbers to names (e.g., for `kill -l`, `trap -l`, or formatting signal names in job status output). The shell should use `SIG2STR_MAX` to size buffers for signal name conversion.
