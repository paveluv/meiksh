## Extracted Text

> The <sys/times.h> header shall define the clock_t type as described in <sys/types.h>.

## Context

The `<sys/times.h>` header shall define the `clock_t` type as described in `<sys/types.h>`. This is an arithmetic type capable of representing times returned by `times()`. The units are clock ticks (as determined by `sysconf(_SC_CLK_TCK)`). For `meiksh`, `clock_t` is the type used to store and compute CPU time differences when implementing the `time` reserved word and `times` builtin. The shell must convert `clock_t` values to seconds by dividing by the clock ticks per second. See POSIX `<sys/times.h>` DESCRIPTION (basedefs/sys_times.h.html lines 45–46).
