## Extracted Text

> The following operand shall be supported:: job_id

## Context

**Behavior:** The `jobs` utility must accept one or more `job_id` operands, which specify which jobs to display status for. The format of `job_id` is defined in XBD 3.182 Job ID.

**Job ID formats (from XBD 3.182):**
- `%n` — job number n
- `%string` — job whose command begins with string
- `%?string` — job whose command contains string
- `%%` or `%+` — current job
- `%-` — previous job

**Spec reference:** POSIX.1-2024, `jobs` OPERANDS, line 68 of `utilities/jobs.html`.
