## Extracted Text

> The application shall ensure that the value of n is a positive decimal integer. If n is greater than the number of enclosing loops, the outermost enclosing loop shall be used. If there is no enclosing loop, the behavior is unspecified.

## Context

**Behavior:** The application must ensure the value of `n` for `continue` is a **positive decimal integer**. If `n` is greater than the number of enclosing loops, the **outermost** enclosing loop is used (continue returns to the top of the outermost loop). If there is **no** enclosing loop, the behavior is **unspecified**.

**Applies to:** All shell modes.

**Spec reference:** `continue` DESCRIPTION in `utilities/V3_chap02.html`, line 3459.
