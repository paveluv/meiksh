## Extracted Text

> The following environment variables shall affect the execution of fc:: HISTSIZE

## Context

The `fc` builtin must recognize the `HISTSIZE` environment variable, which specifies the maximum number of previous commands accessible in the history. See SHALL-20-44-08-007 and SHALL-20-44-08-020 for the full behavioral requirements including default values and initialization. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
