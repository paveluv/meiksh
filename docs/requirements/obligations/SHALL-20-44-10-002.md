## Extracted Text

> If both the -l and -n options are specified, the format of each command shall be:

## Context

When `fc -l -n` (or `fc -ln`) is used, the output format for each command must be `"\t%s\n", <command>`. The command number is suppressed but the leading tab character is preserved. This ensures consistent alignment whether or not numbers are shown. The format applies to the first line of each command; continuation lines of multi-line commands use the same `"\t%s\n"` format (see SHALL-20-44-10-003). Reference: POSIX.1-2024 §fc, STDOUT (tag_20_44_10).
