## Extracted Text

> This variable shall be set as specified in the DESCRIPTION. If an application sets or unsets the value of PWD , the behavior of cd is unspecified.

## Context

**Behavior:** Duplicate extraction of the PWD description (same as SHALL-20-14-08-019). This obligation has two parts:

1. **Shall set PWD:** The cd built-in must set PWD as specified in the DESCRIPTION (step 10):
   - Without `-P`: set to the logical canonicalized curpath value from step 9.
   - With `-P`: set to the physical path (equivalent to `pwd -P` output).
2. **Unspecified on application modification:** If an application sets or unsets PWD directly (rather than letting the shell manage it), the behavior of cd is entirely unspecified.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), lines 204-205.
