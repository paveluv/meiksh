## Extracted Text

> The following variables shall affect the execution of the shell:: LINENO

## Context

This is the heading obligation for the `LINENO` shell variable. It introduces `LINENO` as one of the variables that shall affect execution. The detailed semantics are in SHALL-19-05-03-020 and the duplicate SHALL-19-05-03-038.

**Required behavior:**
- The shell must recognize `LINENO` as a shell variable with special meaning.

See 2.5.3 "Shell Variables" (`LINENO`) in `utilities/V3_chap02.html`.
