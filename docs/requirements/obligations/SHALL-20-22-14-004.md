## Extracted Text

> When the -v or -V options are specified, the following exit values shall be returned:: >0

## Context

**Behavior:** Exit status >0 must be returned by `command -v` or `command -V` when the command_name could not be found or an error occurred. This is the failure case for the information mode.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section EXIT STATUS (tag_20_22_14), line 199.
