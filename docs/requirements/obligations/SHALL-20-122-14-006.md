## Extracted Text

> If the utility utility is invoked, the exit status of time shall be the exit status of utility; otherwise, the time utility shall exit with one of the following values:: 127

## Context

**Behavior:** When the `time` utility cannot find the specified utility at all, `time` shall exit with status 127. This means the PATH search (or direct path lookup) failed to locate an executable file matching the utility operand.

**Conditions:** The utility name was not found in any directory listed in PATH, or if an absolute/relative path was given, the file does not exist (ENOENT).

**Distinction from 126:** Exit code 127 means "not found" while 126 means "found but not invocable."
