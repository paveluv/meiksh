## Extracted Text

> The following commands shall be recognized in command mode:: Move to the first occurrence of the character 'c' that occurs after the current cursor position. If the cursor was positioned on the last character of the line, the terminal shall be alerted and the cursor shall not be advanced. If the character 'c' does not occur in the line after the current cursor position, the terminal shall be alerted and the cursor shall not be moved.

## Context

**Behavior:** In vi command mode, `fc` moves the cursor to the first occurrence of character `c` that occurs AFTER the current cursor position (searching rightward).

**Edge cases:**
- If cursor is on the last character: alert the terminal (bell), do NOT advance.
- If character `c` does not occur after cursor: alert the terminal, do NOT move.
- With count: find the count-th occurrence of `c` after cursor.

**Implementation:** Search forward from cursor+1 for character `c`. On count > 1, find the count-th match. If not found, ring bell. This is a motion command.
