## Extracted Text

> The following environment variables shall affect the execution of kill:: LC_ALL

## Context

**Behavior:** The `LC_ALL` environment variable must be recognized by the `kill` built-in. When set to a non-empty string value, it overrides all other internationalization variables (LANG, LC_CTYPE, LC_MESSAGES, etc.). See SHALL-20-64-08-005.
