## Extracted Text

> The following environment variables shall affect the execution of umask:: [XSI] Determine the location of messages objects and message catalogs.

## Context

**Behavior:** This is the description for NLSPATH's effect on `umask`. On XSI-conformant systems, NLSPATH determines where message objects and message catalogs are located for use by the `umask` utility when producing locale-dependent messages.

**Conditions:** Only applicable on XSI-conformant systems. This is not a base POSIX requirement.

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_132_08) in `utilities/umask.html`, lines 110-111.
