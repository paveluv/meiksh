## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcsetattr():: TCSAFLUSH

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCSAFLUSH` for use with `tcsetattr()`. When used, terminal attribute changes take effect after all pending output has been transmitted, and additionally all pending (unread) input is discarded.

**Conditions:** Used as the `optional_actions` parameter to `tcsetattr()`.

**Shell relevance:** meiksh uses TCSAFLUSH when switching to a new mode where stale input from the old mode should be discarded. For example, when entering the line editor after executing a command — any typeahead that arrived during command execution under the old terminal mode should be flushed. Also useful when the `stty` builtin changes modes.

**Spec reference:** POSIX.1-2024, `<termios.h>` Attribute Selection (tag_14_77_03_08), `basedefs/termios.h.html` line 464.
