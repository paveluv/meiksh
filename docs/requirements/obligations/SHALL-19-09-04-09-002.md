## Extracted Text

> The compound-list-1 shall be executed, and if it has a non-zero exit status, the while command shall complete. Otherwise, the compound-list-2 shall be executed, and the process shall repeat.

## Context

**Behavior:** Execution of `while compound-list-1 ; do compound-list-2 ; done`:

1. Execute compound-list-1 (condition).
2. If its exit status is **non-zero**, the `while` command completes.
3. Otherwise, execute compound-list-2 (body).
4. Go to step 1 (repeat).

The condition is always re-evaluated before each iteration. The body may modify variables or state that affects the condition.

**Applies to:** All shell modes.

**Spec reference:** Section 2.9.4.5 The while Loop in `utilities/V3_chap02.html`, lines 2333-2334.
