## Extracted Text

> The <signal.h> header shall define the following data types:: pid_t

## Context

The `<signal.h>` header must define `pid_t` for process identification. For `meiksh`, `pid_t` is fundamental—used in `kill()` for sending signals, `waitpid()` for reaping children, `getpid()`/`getppid()` for process identity, and throughout job control. The shell stores `pid_t` values for tracked jobs and child processes.
