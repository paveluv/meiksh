## Extracted Text

> The following environment variables shall affect the execution of sh:: FCEDIT

## Context

**Behavior:** The FCEDIT environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-005).

**Applies to:** Interactive shells using the `fc` built-in command (User Portability [UP] option).

**Details:** FCEDIT provides the default editor for the `fc` built-in's `-e` option. When FCEDIT is null or unset, `ed` is used as the default editor. See SHALL-20-110-08-005 for the full behavior specification.
