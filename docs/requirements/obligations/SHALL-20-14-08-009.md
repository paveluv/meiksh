## Extracted Text

> The following environment variables shall affect the execution of cd:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

**Behavior:** This is the description of LC_ALL's role for cd. When LC_ALL is set to a non-empty string value, it overrides all other internationalization variables (LANG, LC_CTYPE, LC_MESSAGES, etc.) for the cd built-in. The shell must check LC_ALL first when determining any locale category.

**Applies to:** All locale-sensitive behavior of cd.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 189.
