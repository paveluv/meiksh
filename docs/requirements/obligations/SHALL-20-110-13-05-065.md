## Extracted Text

> The following commands shall be recognized in command mode:: Enter insert mode after the end of the current command line.

## Context

**Behavior:** In vi command mode, `A` enters insert mode after the end of the current command line. Equivalent to `$a`.

**Implementation:** Move cursor to end of line (past the last character), then enter insert mode.
