## Extracted Text

> The following environment variables shall affect the execution of cd:: The name of the directory, used when no directory operand is specified.

## Context

**Behavior:** HOME is the directory used as the default target when no `directory` operand is given to `cd`. When the user types `cd` with no arguments:
- If HOME is set to a non-empty value, cd behaves as if that value was the directory operand (step 2).
- If HOME is empty or undefined, the behavior is implementation-defined and no further cd steps are taken (step 1).

**Error handling:** The shell must not crash or produce undefined behavior if HOME is unset; the implementation-defined behavior should be chosen and documented.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 183.
