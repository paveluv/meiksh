## Extracted Text

> The initial numerical group ID is used by the newgrp utility. Any other circumstances under which the initial values are operative are implementation-defined.

## Context

**Definition — low shell relevance.** States that the "initial numerical group ID" from the user database is used by the `newgrp` utility. Other circumstances under which initial values are operative are implementation-defined.

**Shell impact:** The shell inherits its real/effective GID from the login process, which derives it from the user database. The `newgrp` utility (which may replace the current shell) uses this field. `meiksh` does not need to read the user database directly for this purpose — the OS sets up GIDs during login.

**Cross-ref note:** refs include `../utilities/newgrp.html` — this is a shell-relevant utility (changes group and re-invokes the shell), but it is an external utility not a builtin. Consider extraction if `newgrp` obligations matter for `meiksh`.
