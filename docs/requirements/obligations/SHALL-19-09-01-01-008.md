## Extracted Text

> Each variable assignment shall be expanded for tilde expansion, parameter expansion, command substitution, arithmetic expansion, and quote removal prior to assigning the value.

## Context

**Behavior:** Standalone restatement of step 4 from SHALL-19-09-01-01-005. Each variable assignment MUST be expanded with tilde expansion, parameter expansion, command substitution, arithmetic expansion, and quote removal before the value is assigned. Field splitting and pathname expansion are NOT applied to assignment values.

**Applies to:** All variable assignments in simple commands.

**Implementation detail:** See SHALL-19-09-01-01-005 for full details.
