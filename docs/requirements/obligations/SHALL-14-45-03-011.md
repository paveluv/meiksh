## Extracted Text

> [CX] The <signal.h> header shall define the pthread_attr_t type as described in <sys/types.h>.

## Context

The `<signal.h>` header must provide `pthread_attr_t` (CX extension) for thread attribute specification in signal event notification (`sigevent` structure's `sigev_notify_attributes`). For `meiksh`, this type is not directly used unless the shell implements threaded signal notification. This is a system header requirement—the shell depends on the platform providing this type when using `SIGEV_THREAD` notification.
