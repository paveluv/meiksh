## Extracted Text

> When -p is specified, export shall write to the standard output the names and values of all exported variables, in the following format:

## Context

**Behavior:** When `export -p` is specified, `export` must write to standard output the names and values of all exported variables in the format:

```
export name="value"
```

For exported variables that are set. For exported variables that are unset:

```
export name
```

**Applies to:** All shell modes.
