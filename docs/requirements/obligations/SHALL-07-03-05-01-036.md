## Extracted Text

> Specify the name of an existing locale which shall be used as the definition of this category. If this keyword is specified, no other keyword shall be specified.

## Context

When the `copy` keyword is used in LC_TIME, the named locale shall provide the complete LC_TIME definition. If `copy` is specified, no other keyword shall be specified in the same LC_TIME block. This is the exclusivity constraint on the `copy` keyword.

**Shell relevance:** Informational for locale infrastructure. See XBD 7.3.5.1.
