## Extracted Text

> Interactive Shell: Unrecoverable read error when reading commands -- shall exit4

## Context

An interactive shell SHALL also exit when an unrecoverable read error occurs while reading commands. See 2.8.1 Consequences of Shell Errors in utilities/V3_chap02.html.

**Behavior required:**
- When the shell is interactive and encounters an unrecoverable error reading command input, the shell SHALL exit.
- A shell diagnostic message IS required.
- This is the ONLY error category where both interactive and non-interactive shells are required to exit.

**Note:** This is different from EOF on the terminal (which may or may not exit depending on `ignoreeof` option). An unrecoverable read error is a hard I/O failure, not simply end-of-input.
