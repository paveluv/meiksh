## Extracted Text

> If ISTRIP is set, valid input bytes shall first be stripped to seven bits; otherwise, all eight bits shall be processed.

## Context

**Behavior:** When ISTRIP is set in `c_iflag`, valid input bytes shall be stripped to seven bits (the high bit is cleared). When ISTRIP is not set, all eight bits of each input byte shall be processed without modification.

**Relevance to meiksh:** The `stty` builtin must correctly set/clear the ISTRIP flag. The shell does not perform bit stripping itself; the OS terminal driver handles it. The shell must faithfully pass through this flag when configuring terminal attributes.

**Spec reference:** POSIX.1-2024, §11.2.2 Input Modes (tag_11_02_02), `basedefs/V1_chap11.html` line 521.
