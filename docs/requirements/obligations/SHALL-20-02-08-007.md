## Extracted Text

> The following environment variables shall affect the execution of alias:: Determine the locale for the interpretation of sequences of bytes of text data as characters (for example, single-byte as opposed to multi-byte characters in arguments).

## Context

`LC_CTYPE` **shall** determine the locale for interpreting sequences of bytes of text data as characters in `alias` arguments. For example, single-byte versus multi-byte character handling in alias names and values depends on the `LC_CTYPE` setting. The shell must use this locale category when processing alias operands. Reference: alias utility ENVIRONMENT VARIABLES.
