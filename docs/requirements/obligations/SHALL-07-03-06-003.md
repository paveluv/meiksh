## Extracted Text

> The following keywords shall be recognized as part of the locale definition file.

## Context

The following keywords shall be recognized in the LC_MESSAGES locale definition file: `copy`, `yesexpr`, and `noexpr`.

**Shell relevance:** The shell relies on system locale data for yes/no pattern matching. See XBD 7.3.6.
