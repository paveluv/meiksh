## Extracted Text

> The following tokens shall be processed (in addition to those string constants shown in the grammar):: One or more decimal representations of the encoding of each byte in a single character. The decimal representation consists of an escape character followed by a character 'd' and two or more decimal digits.

## Context

DECIMAL_CHAR is defined as one or more decimal representations of the encoding of each byte in a single character. The format is an escape character followed by `d` and two or more decimal digits.

**Shell relevance:** Informational for locale infrastructure. See XBD 7.4.1.
