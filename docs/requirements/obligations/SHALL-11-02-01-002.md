## Extracted Text

> The tcflag_t and cc_t types are defined in the <termios.h> header. They shall be unsigned integer types.

## Context

**Behavior:** The types `tcflag_t` (used for terminal mode flags like `c_iflag`, `c_oflag`, `c_cflag`, `c_lflag`) and `cc_t` (used for special control characters in the `c_cc` array) shall be **unsigned integer types**, defined in `<termios.h>`.

**Conditions:** These types are required by the header definition.

**Shell relevance:** When meiksh manipulates terminal attributes, it should use these types correctly. Being unsigned, bitwise operations (OR, AND, complement) work as expected without sign-extension issues. The shell should use `tcflag_t` for flag variables and `cc_t` for control character values. No special casting is needed since they are unsigned.

**Spec reference:** POSIX.1-2024, §11.2.1 The termios Structure (tag_11_02_01), `basedefs/V1_chap11.html` lines 387–388.
