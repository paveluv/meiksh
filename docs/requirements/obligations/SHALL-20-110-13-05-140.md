## Extracted Text

> Repeat the most recent f, F, t, or T command. Any number argument on that previous command shall be ignored. Errors are those described for the repeated command.

## Context

**Behavior:** The `;` command shall repeat the most recent `f`, `F`, `t`, or `T` command in the same direction. The original count is ignored; the current count applies. Errors match the repeated command's rules.

**Details:** Same normative content as SHALL-20-110-13-05-059. This is a motion command.
