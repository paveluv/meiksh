## Extracted Text

> The following environment variables shall affect the execution of cd:: OLDPWD

## Context

**Behavior:** The `OLDPWD` environment variable must affect `cd` execution. OLDPWD contains the pathname of the previous working directory and is used when the `cd` operand is `-`. The shell must recognize OLDPWD as a variable that influences cd behavior.

**Applies to:** `cd -` invocations. Also, cd must SET OLDPWD to the old working directory value (the cwd before the cd) whenever PWD is set during cd execution.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 200.
