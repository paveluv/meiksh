## Extracted Text

> [UP] The processing of the LINENO shell variable shall be supported if the system supports the User Portability Utilities option. Set by the shell to a decimal number representing the current sequential line number (numbered starting with 1) within a script or function before it executes each command. If the user unsets or resets LINENO , the variable may lose its special meaning for the life of the shell. If the shell is not currently executing a script or function, the value of LINENO is unspecified.
