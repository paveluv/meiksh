## Extracted Text

> An address associated with a socket or remote endpoint, including an address family identifier and addressing information specific to that address family. The address may include multiple parts, such as a network address associated with a host system and an identifier for a specific endpoint.

## Context

**Definition only — no shell relevance.** Defines "socket address" as an address associated with a socket endpoint, including an address family and family-specific addressing info.

**Shell impact:** None. The shell does not natively support socket operations. Some shells provide `/dev/tcp` and `/dev/udp` extensions, but these are non-POSIX. No action required.
