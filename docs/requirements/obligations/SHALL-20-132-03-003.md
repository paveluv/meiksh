## Extracted Text

> If the mask operand is not specified, the umask utility shall write to standard output the value of the file mode creation mask of the invoking process.

## Context

**Behavior:** When `umask` is invoked with no operand (just `umask` or `umask -S`), it shall write the current file mode creation mask value to stdout. This is the "query" mode of `umask`.

**Output format:** The default output format is unspecified (but typically octal, e.g., `0022`). With `-S`, the output format is symbolic (e.g., `u=rwx,g=rx,o=rx`). The output must be a valid `mask` operand that can be passed back to `umask` to restore the same mask.

**No side effects:** In query mode, the umask value must not be changed; only the current value is printed.
