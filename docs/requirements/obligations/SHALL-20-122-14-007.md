## Extracted Text

> If the utility utility is invoked, the exit status of time shall be the exit status of utility; otherwise, the time utility shall exit with one of the following values:: The utility specified by utility could not be found.

## Context

**Behavior:** This is the description for exit status 127 of the `time` utility. When the utility operand could not be found via PATH search or by the specified path, `time` shall exit with status 127.

**Conditions:** The shell searched PATH for the utility name and found no matching executable file, or a specified path does not exist.

**Error handling:** The shell should write a diagnostic message to stderr indicating that the specified utility could not be found, before exiting with status 127.
