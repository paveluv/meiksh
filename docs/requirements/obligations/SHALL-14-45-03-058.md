## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: SIGSTKSZ

## Context

The `<signal.h>` header shall define the symbolic constant `SIGSTKSZ`. This is an [XSI] constant representing the default size in bytes for an alternate signal stack. It is a suggested size for allocating an alternate stack with `sigaltstack()`. For `meiksh`, this is not directly relevant since the shell does not use alternate signal stacks. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html line 552).
