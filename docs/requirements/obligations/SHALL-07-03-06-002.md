## Extracted Text

> The message catalog used by the standard utilities and selected by the catopen() function shall be determined by the setting of NLSPATH ; see 8. Environment Variables. The LC_MESSAGES category can be specified as part of an NLSPATH substitution field.

## Context

The message catalog used by standard utilities, selected by `catopen()`, shall be determined by the NLSPATH environment variable setting. The LC_MESSAGES category can be specified as part of an NLSPATH substitution field (e.g., `%L`).

**Shell relevance:** Diagnostic messages from the shell and its builtins may be localized through message catalogs found via NLSPATH. See XBD 7.3.6 and XBD 8.
