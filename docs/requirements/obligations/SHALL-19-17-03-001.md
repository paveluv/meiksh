## Extracted Text

> This utility shall do nothing except return a 0 exit status. It is used when a command is needed, as in the then condition of an if command, but nothing is to be done by the command.

## Context

**Behavior:** The `:` (colon) special built-in utility must do nothing except return exit status **0**. It is a no-op command used as a placeholder where a command is syntactically required but no action is needed (e.g., the `then` clause of an `if` command, or for side effects of variable expansion in its arguments).

**Arguments:** Arguments to `:` are expanded (side effects like variable assignment via `${var:=default}` do occur) but the results are discarded. Redirections are processed.

**Exit status:** Always 0.

**Applies to:** All shell modes. This is a special built-in.
