## Extracted Text

> The following environment variables shall affect the execution of getopts:: LC_ALL

## Context

**Behavior:** If `LC_ALL` is set to a non-empty string, it overrides all other internationalization variables (LANG, LC_CTYPE, LC_MESSAGES, etc.) for the execution of `getopts`. This is the standard override mechanism per XBD 8.2.

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, line 143 of `utilities/getopts.html`.
