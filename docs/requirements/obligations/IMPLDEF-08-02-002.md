## Extracted Text

> Additional criteria for determining a valid locale name are implementation-defined.

## Context

**Behavior:** Additional criteria for determining a valid locale name (beyond the standard format and slash-prefixed pathnames) are implementation-defined. The implementation may impose additional constraints on what constitutes a valid locale name.

**Shell relevance:** `meiksh` should document any additional locale name validation rules it applies. In practice, locale name validation is delegated to the C library's `setlocale()` function, so the shell inherits the host system's criteria.
