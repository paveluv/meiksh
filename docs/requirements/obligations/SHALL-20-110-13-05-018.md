## Extracted Text

> The following commands shall be recognized in command mode:: Insert the value of the alias named _letter. The symbol letter represents a single alphabetic character from the portable character set; implementations may support additional characters as an extension. If the alias _letter contains other editing commands, these commands shall be performed as part of the insertion. If no alias _letter is enabled, this command shall have no effect.

## Context

**Behavior:** In vi command mode, `@letter` shall insert the value of the alias named `_letter` (underscore prefix + the letter). The letter must be a single alphabetic character from the portable character set. If the alias value contains editing commands, those commands shall be executed as part of the insertion (macro expansion). If no alias `_letter` is defined/enabled, the command has no effect.

**Implementation:** Look up alias `_<letter>` in the alias table. If found, feed its value back into the command mode input as if the user typed those characters (macro replay). This enables user-defined editing macros.
