## Extracted Text

> The time utility shall invoke the utility named by the utility operand with arguments supplied as the argument operands and write a message to standard error that lists timing statistics for the utility. The message shall include the following information:

## Context

**Behavior:** The `time` utility shall invoke the utility named by its `utility` operand with the supplied `argument` operands, and write a message to standard error containing timing statistics. The message shall include at minimum: (1) elapsed (real) time, (2) user CPU time, and (3) system CPU time.

**Inputs:** The `utility` operand names the command to execute; `argument` operands are passed through.

**Output:** Timing statistics are written to stderr (never stdout). The format depends on whether `-p` is specified (see SHALL-20-122-11-002).

**Spec reference:** `utilities/time.html` lines 37-38.
