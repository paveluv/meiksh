## Extracted Text

> The following environment variables shall affect the execution of fg:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

This is the description of `LC_MESSAGES`'s effect on `fg`. The implementation must format all diagnostic messages from `fg` (errors about invalid job IDs, job control being disabled, etc.) according to the locale determined by `LC_MESSAGES`. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
