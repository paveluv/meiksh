## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use as values of objects of type speed_t.

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for baud rates as `speed_t` values. These include at minimum: B0, B50, B75, B110, B134, B150, B200, B300, B600, B1200, B1800, B2400, B4800, B9600, B19200, B38400. B0 is special — it signals hangup (drop DTR).

**Conditions:** These values are used with `cfsetispeed()`, `cfsetospeed()`, `cfgetispeed()`, `cfgetospeed()`.

**Shell relevance:** The `stty` builtin must support displaying and setting baud rates using these constants. Setting the baud rate to B0 causes the terminal to hang up (equivalent to dropping the modem connection). On pseudo-terminals, baud rate settings are typically ignored but the API must still work.

**Spec reference:** POSIX.1-2024, `<termios.h>` Baud Rate Selection (tag_14_77_03_04), `basedefs/termios.h.html` lines 350–351.
