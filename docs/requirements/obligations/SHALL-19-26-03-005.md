## Extracted Text

> Implementations shall support the options in the following list in both their <hyphen-minus> and <plus-sign> forms. These options can also be specified as options to sh.

## Context

**Behavior:** The implementation must support the standard `set` options in both `-` (enable) and `+` (disable) forms. These same options can also be specified as options to the `sh` command itself on invocation. The options include: `-a`, `-b`, `-C`, `-e`, `-f`, `-h`, `-m`, `-n`, `-o`, `-u`, `-v`, `-x`.

**Applies to:** All shell modes.
