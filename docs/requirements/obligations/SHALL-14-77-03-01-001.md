## Extracted Text

> The <termios.h> header shall define the following data types through typedef:

## Context

**Behavior:** `<termios.h>` **shall** define the data types `cc_t`, `speed_t`, and `tcflag_t` through `typedef`. These are the fundamental types used in the termios structure and terminal control API.

**Conditions:** These types must be available after including `<termios.h>`.

**Shell relevance:** meiksh uses these types when declaring variables for terminal attribute manipulation. `cc_t` for control character values, `speed_t` for baud rates, and `tcflag_t` for mode flags. The shell code should use these types rather than raw integer types for portability and clarity.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 41.
