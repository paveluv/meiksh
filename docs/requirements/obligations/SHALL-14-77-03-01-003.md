## Extracted Text

> The <termios.h> header shall define the following data types through typedef:: Used for terminal special characters.

## Context

**Behavior:** The `cc_t` type is described as being **used for terminal special characters**. This is the descriptive purpose of the `cc_t` typedef — it holds the byte values that represent special terminal control characters (INTR, QUIT, ERASE, KILL, EOF, EOL, SUSP, START, STOP) and the numeric parameters MIN and TIME.

**Conditions:** Part of the `cc_t` type definition in `<termios.h>`.

**Shell relevance:** When meiksh reads or writes control character values in the `c_cc` array, the values are of type `cc_t`. This type is typically `unsigned char` on most implementations.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 45.
