## Extracted Text

> The following operands shall be supported:: Replace the first occurrence of string old in the commands to be re-executed by the string new.

## Context

This is the description of the `old=new` operand for `fc -s`. The implementation must replace only the FIRST occurrence of the literal string `old` in the command text with the literal string `new`. This is not a global replacement — only the first match is substituted. After substitution, the modified command is added to the history and re-executed. If the string `old` does not appear in the command, the command is re-executed unchanged (no error). Reference: POSIX.1-2024 §fc, OPERANDS (tag_20_44_05).
