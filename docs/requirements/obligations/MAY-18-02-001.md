## Extracted Text

> Implementations may provide more liberal, or less restrictive, values than shown in Utility Limit Minimum Values. These possibly more liberal values are accessible using the symbols in Symbolic Utility Limits.

## Context

The shell implementation **may** provide more liberal (larger) limit values than the minimums specified in XCU Table "Utility Limit Minimum Values." For example, `{LINE_MAX}` may be larger than 2048. If the implementation provides higher limits, these actual values are accessible through the symbolic names in XCU Table "Symbolic Utility Limits" via the `getconf` utility. The shell is permitted but not required to exceed the minimum limits. Reference: XCU 1.2.
