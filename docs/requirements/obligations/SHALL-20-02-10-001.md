## Extracted Text

> The format for displaying aliases (when no operands or only name operands are specified) shall be:

## Context

When displaying aliases (no operands, or only bare name operands), the output format **shall** be: `"%s=%s\n", alias-name, value` — where each alias is on its own line, with the alias name followed by `=` and the quoted value. The exact format string from the spec is `"%s=%s\n", name, value`. Reference: alias utility STDOUT.
