## Extracted Text

> The <termios.h> header shall define the following data types through typedef:: Used for terminal baud rates.

## Context

**Behavior:** The `speed_t` type is described as being **used for terminal baud rates**. This is the descriptive purpose of the `speed_t` typedef — it holds baud rate values such as B0, B50, B75, ..., B38400, etc.

**Conditions:** Part of the `speed_t` type definition in `<termios.h>`.

**Shell relevance:** The `stty` builtin uses `speed_t` when getting/setting baud rates via cfgetispeed/cfsetispeed/cfgetospeed/cfsetospeed. On modern pseudo-terminals, baud rate is typically meaningless but the API must still work.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 47.
