## Context

**Non-interactive shell notification timing option 3:** A non-interactive shell may write the deferred job status message **before the shell exits**. This is the latest possible point—the notification would appear at shell shutdown. This is the least useful timing but is the simplest to implement (flush all pending notifications during the exit sequence).
