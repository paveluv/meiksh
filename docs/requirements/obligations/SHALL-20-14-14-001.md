## Extracted Text

> The following exit values shall be returned:

## Context

**Behavior:** The cd built-in must return the specific exit values defined in the EXIT STATUS section. The exit values depend on the combination of options (-e, -P, -L) and whether the directory change succeeded and PWD could be determined. The defined exit values are: 0 (success + PWD set), 0 (success + no -e + -P + PWD undetermined), >0 (error without both -e and -P), 1 (-e + -P + success + PWD undetermined), >1 (-e + -P + error).

**Applies to:** Every invocation of the cd built-in.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 239.
