## Extracted Text

> The following commands shall be recognized in command mode:: (Zero.) Move to the first character position on the current command line.

## Context

**Behavior:** In vi command mode, `0` (zero) moves the cursor to the first character position (column 0) on the current command line.

**Implementation:** Set cursor position to 0. This is a motion command.
