## Extracted Text

> The implementation shall support the functionality associated with the symbols CS7, CS8, CSTOPB, PARODD, and PARENB.

## Context

**Behavior:** The implementation **shall** support the functionality for these specific control mode flags:
- **CS7**: 7-bit character size
- **CS8**: 8-bit character size
- **CSTOPB**: send two stop bits (else one)
- **PARODD**: odd parity (else even)
- **PARENB**: parity enable

Other control mode flags (CS5, CS6, CLOCAL, HUPCL, CREAD) are defined but support is not mandated.

**Conditions:** These five must be functionally supported, not just defined as constants.

**Shell relevance:** The `stty` builtin must support setting these modes. CS8 is the typical character size for modern terminals. Parity settings (PARENB, PARODD) are primarily for serial connections but must still be configurable via `stty`.
