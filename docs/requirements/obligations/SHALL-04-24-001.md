## Extracted Text

> For functions called with a NaN argument, no errors shall occur and a NaN shall be returned, except where stated otherwise.

## Context

This is a C math library requirement about NaN argument handling. It is not relevant to a POSIX shell implementation — the shell does not use floating-point math. No shell action is required.

See: `basedefs/V1_chap04.html` §4.24.
