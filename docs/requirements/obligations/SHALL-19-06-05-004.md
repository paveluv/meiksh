## Extracted Text

> Each input field shall be considered in sequence, first to last, with the results of the algorithm described in this section causing output fields to be generated, which shall remain in the same order as the input fields from which they originated.

## Context

**Behavior:** Field splitting processes input fields sequentially, left-to-right. The output fields generated from each input field must maintain the same relative ordering as their source input fields. A single input field may produce zero, one, or multiple output fields, but those output fields appear in the output list at the position corresponding to their source input field.

**Implementation significance:** The field splitting algorithm must not reorder fields. Process each input field in order and append the resulting output field(s) to the output list.

**Spec reference:** See 2.6.5 Field Splitting in utilities/V3_chap02.html, lines 1215–1217.
