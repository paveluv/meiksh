## Extracted Text

> The following environment variables shall affect the execution of fc:: FCEDIT

## Context

The `fc` builtin must recognize the `FCEDIT` environment variable. `FCEDIT` specifies the default editor to be used when the `-e` option is not provided. The variable is expanded by the shell to determine its value. If `FCEDIT` is null or unset, `ed` must be used as the default editor. See SHALL-20-44-08-003 and SHALL-20-44-08-018 for the full behavioral description. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
