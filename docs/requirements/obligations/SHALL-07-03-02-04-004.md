## Extracted Text

> A <collating-symbol> can be used to define a position in the relative order for use in weights. No weights shall be specified with a <collating-symbol>.

## Context

A collating-symbol in the LC_COLLATE collation order section defines an abstract position in the collation sequence that can be used as a weight value for other elements, but does not itself represent a character. No weight operands may be specified on the line defining a collating-symbol — it serves purely as an ordering anchor. The shell is affected because collating symbols participate in equivalence class definitions and collation ordering, which affect bracket expression ranges in pattern matching.

See: `basedefs/V1_chap07.html` §7.3.2.4 Collation Order.
