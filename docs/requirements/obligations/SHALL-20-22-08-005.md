## Extracted Text

> The following environment variables shall affect the execution of command:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

**Behavior:** LC_ALL description for `command`. When LC_ALL is set to a non-empty value, it overrides LANG, LC_CTYPE, LC_MESSAGES, and all other LC_* variables for the `command` utility.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section ENVIRONMENT VARIABLES (tag_20_22_08), line 148.
