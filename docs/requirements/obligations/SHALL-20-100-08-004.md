## Extracted Text

> The following environment variables shall affect the execution of read:: LANG

## Context

**Behavior required:** The `LANG` environment variable must affect the execution of the `read` builtin. It provides a default value for locale variables that are unset or null.

**Condition:** When LC_* variables are unset or null, LANG serves as the fallback.

**Implementation detail:** For `read`, locale primarily affects: (1) LC_CTYPE for character interpretation during field splitting, (2) LC_MESSAGES for diagnostic messages and the PS2 prompt. LANG provides defaults for these when their specific variables are unset.
