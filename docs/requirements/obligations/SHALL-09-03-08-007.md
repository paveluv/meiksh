## Extracted Text

> A BRE anchored by both '^' and '$' shall match only an entire string. For example, the BRE "^abcdef$" matches strings consisting only of "abcdef".

## Context

**Behavior:** Duplicate of SHALL-09-03-08-004. A BRE with both `^` and `$` anchors matches only an entire string.

**Shell relevance:** Same as SHALL-09-03-08-004. Shell `case` patterns inherently do full-string matching.
