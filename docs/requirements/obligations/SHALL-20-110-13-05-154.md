## Extracted Text

> Undo the last command that changed the edit line. This operation shall not undo the copy of any command line to the edit line.

## Context

**Behavior:** The `u` command shall undo the last command that changed the edit line. This shall NOT undo the copy of a command line to the edit line (history promotion).
