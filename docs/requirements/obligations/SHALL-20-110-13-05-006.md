## Extracted Text

> The following commands shall be recognized in command mode:: Execute the current command line. If the current command line is not empty, this line shall be entered into the command history (see fc).

## Context

**Behavior:** In vi command mode, pressing `<newline>` shall execute the current command line. If the current command line is not empty, it shall be entered into the command history (accessible via `fc`).

**Conditions:** Applies in vi command mode. An empty command line is simply executed (no-op, no history entry). A non-empty line is both executed AND recorded in history.

**Implementation:** On newline: (1) if line is non-empty, append to history, (2) submit the line for execution by the shell parser, (3) reset the editing state for the next prompt.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 399–400. See also `utilities/fc.html` for command history.
