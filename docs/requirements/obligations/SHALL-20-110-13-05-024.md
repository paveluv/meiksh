## Extracted Text

> The following commands shall be recognized in command mode:: Invoke the vi editor to edit the current command line in a temporary file. When the editor exits, the commands in the temporary file shall be executed and placed in the command history. If a number is included, it specifies the command number in the command history to be edited, rather than the current command line.

## Context

**Behavior:** In vi command mode, `v` shall invoke the `vi` editor to edit the current command line in a temporary file. When the editor exits, the commands in the temporary file shall be executed and placed in the command history. If a number is provided, it specifies the command history number to edit rather than the current command line.

**Implementation:** (1) Write current command line (or history entry at `number`) to a temp file, (2) launch `$VISUAL`, `$EDITOR`, or `vi` on that file, (3) wait for editor to exit, (4) read back the file contents, (5) execute the contents as shell commands, (6) add to command history.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 454–456. See `utilities/vi.html` for the vi editor.
