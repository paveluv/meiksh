## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for use as the `action` argument to `tcflow()`. These constants control software flow control: TCIOFF, TCION, TCOOFF, TCOON.

**Conditions:** These constants must be defined and usable with `tcflow()`.

**Shell relevance:** meiksh may use `tcflow()` for software flow control management:
- **TCOOFF**: suspend output (like receiving STOP character)
- **TCOON**: resume output (like receiving START character)
- **TCIOFF**: transmit STOP character to suspend remote input
- **TCION**: transmit START character to resume remote input
The `stty` builtin may need to expose flow control settings.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` lines 479–480.
