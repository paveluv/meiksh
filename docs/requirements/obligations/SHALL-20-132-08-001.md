> The following environment variables shall affect the execution of umask:

## Context

**Behavior:** The `umask` utility shall be affected by the standard internationalization environment variables: LANG, LC_ALL, LC_CTYPE, LC_MESSAGES, and NLSPATH (XSI). These variables affect locale-dependent behavior such as diagnostic message formatting and character interpretation.

**Conditions:** These variables are consulted at the time `umask` is executed. Since `umask` is a shell builtin, the shell should use the current values of these variables from its execution environment.
