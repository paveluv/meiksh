## Extracted Text

> The <termios.h> header shall contain the definitions used by the terminal I/O interfaces (see 11. General Terminal Interface for the structures and names defined).

## Context

**Behavior:** The `<termios.h>` header **shall** contain all definitions used by the terminal I/O interfaces. This includes the `struct termios`, type definitions (`cc_t`, `speed_t`, `tcflag_t`), symbolic constants for flags and control characters, and function declarations.

**Conditions:** This header must be included by any code that uses the termios API.

**Shell relevance:** meiksh must `#include <termios.h>` in all source files that perform terminal I/O control. This single header provides everything needed for terminal manipulation: the termios structure, all flag constants (ICANON, ECHO, ISIG, etc.), control character subscripts (VINTR, VEOF, VMIN, etc.), baud rate constants, and function prototypes (tcgetattr, tcsetattr, tcflush, tcdrain, tcflow, etc.).

**Spec reference:** POSIX.1-2024, `<termios.h>` DESCRIPTION (tag_14_77_03), `basedefs/termios.h.html` lines 38–39. See also XBD Chapter 11.
