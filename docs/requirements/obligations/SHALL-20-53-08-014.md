## Extracted Text

> This variable shall be used by the getopts utility as the index of the next argument to be processed.

## Context

**Behavior:** Duplicate extraction of SHALL-20-53-08-013. `OPTIND` is the 1-based index used by `getopts` to track which argument to process next. The shell reads this variable at the start of each `getopts` invocation and updates it afterwards.
