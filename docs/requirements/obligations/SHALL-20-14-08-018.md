## Extracted Text

> The following environment variables shall affect the execution of cd:: PWD

## Context

**Behavior:** The `PWD` environment variable must affect `cd` execution. PWD holds the current working directory pathname and is both read and written by cd. The shell must recognize PWD as a variable that influences and is modified by cd.

**Applies to:** Every `cd` invocation. PWD is read during logical path resolution (step 7 — prepended to relative curpath when -P is not in effect). PWD is written on successful directory change (step 10).

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 203.
