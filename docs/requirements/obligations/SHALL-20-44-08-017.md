## Extracted Text

> The following environment variables shall affect the execution of fc:: [XSI] Determine the location of messages objects and message catalogs.

## Context

On XSI-conformant systems, the `NLSPATH` environment variable must be used by the `fc` builtin to locate message objects and message catalogs for internationalized diagnostic output. Since meiksh targets XSI conformance, `NLSPATH` should be respected when looking up locale-specific message catalogs for `fc` error messages. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
