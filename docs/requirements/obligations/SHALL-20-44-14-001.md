> The following exit values shall be returned:

## Context

The `fc` builtin must return specific exit values depending on the mode of operation. In list mode (`-l`): exit 0 on success, >0 on error. In edit/re-execute mode (without `-l`): the exit status is that of the re-executed command. See companion obligations SHALL-20-44-14-002 through SHALL-20-44-14-006 for specific values and the re-execution rule. Reference: POSIX.1-2024 §fc, EXIT STATUS (tag_20_44_14).
