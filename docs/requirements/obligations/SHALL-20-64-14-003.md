## Extracted Text

> The following exit values shall be returned:: The -l option was specified and the output specified in STDOUT was successfully written to standard output; or, the -l option was not specified, at least one matching process was found for each pid operand, and the specified signal was successfully processed for at least one matching process.

## Context

**Behavior:** Exit status 0 is returned when either:
1. **List mode (`-l`):** The `-l` option was specified and the signal name(s) were successfully written to stdout.
2. **Signal mode (no `-l`):** All of the following are true:
   - At least one matching process was found for **each** `pid` operand.
   - The specified signal was successfully processed (delivered) for at least one matching process.

**Note:** The condition requires that every pid operand resolves to at least one process, AND the signal was successfully sent to at least one process per operand. If any operand has no matching process, exit >0.
