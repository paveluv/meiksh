## Extracted Text

## Context

**Behavior:** Exit status 0 is returned when either:
1. **List mode (`-l`):** The `-l` option was specified and the signal name(s) were successfully written to stdout.
2. **Signal mode (no `-l`):** All of the following are true:
   - At least one matching process was found for **each** `pid` operand.
   - The specified signal was successfully processed (delivered) for at least one matching process.

**Note:** The condition requires that every `pid` operand resolve to at least one matching process, and that the specified signal be successfully processed for at least one matching process. If any operand has no matching process, exit status 0 is not available under this rule.
