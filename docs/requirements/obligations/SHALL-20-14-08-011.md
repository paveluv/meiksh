## Extracted Text

> The following environment variables shall affect the execution of cd:: Determine the locale for the interpretation of sequences of bytes of text data as characters (for example, single-byte as opposed to multi-byte characters in arguments).

## Context

**Behavior:** This is the description of LC_CTYPE's role for cd. LC_CTYPE determines how sequences of bytes in the `directory` argument are interpreted as characters. This is relevant when directory names contain multi-byte characters (e.g., UTF-8). The shell must use the LC_CTYPE locale setting (or its fallback via LANG/LC_ALL) when parsing directory operands.

**Applies to:** Byte-to-character interpretation in cd's directory operand.
