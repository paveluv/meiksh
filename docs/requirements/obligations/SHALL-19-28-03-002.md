## Extracted Text

> The four pairs of times shall correspond to the members of the <sys/times.h> tms structure (defined in XBD 14. Headers) as returned by times(): tms_utime, tms_stime, tms_cutime, and tms_cstime, respectively.

## Context

**Behavior:** The four pairs of times in the `times` output correspond to the members of the `<sys/times.h>` `tms` structure as returned by the `times()` system call:
1. `tms_utime` — shell user time
2. `tms_stime` — shell system time
3. `tms_cutime` — children's user time
4. `tms_cstime` — children's system time

**Implementation:** Call the `times()` C library function and format the results.

**Applies to:** All shell modes.

**Spec reference:** `times` DESCRIPTION in `utilities/V3_chap02.html`, lines 5333-5336.
