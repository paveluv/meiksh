## Extracted Text

> The following option shall be supported:: -S

## Context

**Behavior:** The `umask` utility shall accept the `-S` option. When `-S` is specified, `umask` produces symbolic output instead of the default (implementation-defined, typically octal) output format.

**Conditions:** `-S` is meaningful only when `umask` is invoked in query mode (no mask operand). When a mask operand is given, `-S` has no visible effect since no output is produced.
