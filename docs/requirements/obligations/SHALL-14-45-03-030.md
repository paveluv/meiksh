## Extracted Text

> The <signal.h> header shall define the following macros which shall expand to integer constant expressions that need not be usable in #if preprocessing directives:

## Context

The `<signal.h>` header must define macros for `sa_flags` values (such as `SA_NOCLDSTOP`, `SA_ONSTACK`, `SA_RESETHAND`, `SA_RESTART`, `SA_SIGINFO`, `SA_NOCLDWAIT`, `SA_NODEFER`) and other signal-related constants. These macros expand to integer constant expressions but need not be usable in `#if` directives (they may involve casts or other non-preprocessor-compatible constructs). For `meiksh`, the key `sa_flags` values are: `SA_RESTART` (to automatically restart interrupted system calls), `SA_SIGINFO` (to use the extended handler), `SA_NOCLDSTOP` (to suppress SIGCHLD for stopped children), and `SA_NOCLDWAIT` (to prevent zombie children). The shell must use these flags correctly when calling `sigaction()`.
