## Extracted Text

> The following exit values shall be returned:: 0

## Context

**Behavior:** The `sh` utility shall return exit status 0 under the condition described in SHALL-20-110-14-003. This is the exit value label "0" in the exit status definition list.

**Applies to:** The `sh` utility when the script contains only blank lines, comments, or both.

**Details:** Exit status 0 is returned when the shell successfully processes a script that contains no executable commands — only whitespace lines and/or comment lines. This means the shell parsed the input successfully but found nothing to execute.
