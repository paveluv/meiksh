## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcsetattr():: TCSANOW

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCSANOW` for use with `tcsetattr()`. When used, terminal attribute changes take effect immediately.

**Conditions:** Used as the `optional_actions` parameter to `tcsetattr()`.

**Shell relevance:** meiksh uses TCSANOW when it needs immediate terminal mode changes, such as when switching to non-canonical mode at the start of line editing. Changes take effect before the next `read()` call.

**Spec reference:** POSIX.1-2024, `<termios.h>` Attribute Selection (tag_14_77_03_08), `basedefs/termios.h.html` line 460.
