## Extracted Text

> The following options shall be supported:: Suppress command numbers when listing with -l.

## Context

This is the description for the `-n` option of `fc`. When `-n` and `-l` are both specified, the command numbers that normally precede each listed history entry must be suppressed. The output format becomes `"\t%s\n", <command>` instead of `"%d\t%s\n", <line number>, <command>`. The leading tab character is still present. This option has no effect when used without `-l`. Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
