## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `return` special builtin (section 2.25 in utilities/V3_chap02.html#tag_19_25) must use standard error **only** for diagnostic messages. It must not write any non-diagnostic output to stderr. The `return` builtin produces no stdout output.

Diagnostic messages that may be written to stderr include:
- Invalid return value argument (not a valid number)
- `return` used outside of a function or dot-script context

See 2.25 return in utilities/V3_chap02.html#tag_19_25.
