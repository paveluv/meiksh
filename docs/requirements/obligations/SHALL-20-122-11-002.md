## Extracted Text

> If -p is specified, the following format shall be used for the timing statistics in the POSIX locale:

## Context

**Behavior:** When `-p` is specified, the timing statistics shall be written to stderr in the POSIX locale using the format:
```
"real %f\nuser %f\nsys %f\n", <real seconds>, <user seconds>, <system seconds>
```

**Output:** Three lines: `real`, `user`, and `sys`, each followed by a floating-point number in seconds.
