## Extracted Text

> The return utility shall cause the shell to stop executing the current function or dot script. If the shell is not currently executing a function or dot script, the results are unspecified.

## Context

**Behavior:** The `return` special built-in causes the shell to stop executing the current function or dot (`.`) script. If the shell is not currently executing a function or dot script, the results are **unspecified**.

**Exit status:** If `n` is specified, the function/dot script returns with exit status `n`. If `n` is not specified, the return value is the exit status of the last command executed.

**Applies to:** All shell modes. Valid only within functions and dot scripts.
