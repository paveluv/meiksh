## Extracted Text

> The following environment variables shall affect the execution of jobs:: LC_MESSAGES

## Context

**Behavior:** `LC_MESSAGES` determines the locale for diagnostic messages (stderr) and informative messages (stdout) from `jobs`. This affects the format and language of error messages and the state strings in job status output.

**Spec reference:** POSIX.1-2024, `jobs` ENVIRONMENT VARIABLES, line 96 of `utilities/jobs.html`.
