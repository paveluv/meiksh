## Extracted Text

> When the -l option is used to list commands, the format of each command in the list shall be as follows:

## Context

When `fc -l` lists commands, each entry must use the format `"%d\t%s\n", <line number>, <command>`. The line number is the command's history number, followed by a tab character, followed by the command text, followed by a newline. This is the format when `-n` is NOT specified. See SHALL-20-44-10-002 for the format with `-n`, and SHALL-20-44-10-003 for multi-line command continuation format. Reference: POSIX.1-2024 §fc, STDOUT (tag_20_44_10).
