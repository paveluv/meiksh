## Extracted Text

> The following operand shall be supported:: alias-name

## Context

**Behavior:** The `unalias` utility shall accept one or more `alias-name` operands. Each operand is the name of an alias to be removed from the current shell execution environment.

**Conditions:** The synopsis shows `unalias alias-name...` requiring at least one operand when `-a` is not used. Each alias-name should match an existing alias definition. If a name does not match a defined alias, an error is reported for that name.
