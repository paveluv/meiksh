## Extracted Text

> The following commands shall be recognized in command mode:: Delete the character at the current cursor position and place the deleted characters in the save buffer. If the cursor was positioned on the last character of the line, the character shall be deleted and the cursor position shall be moved to the previous character (the new last character). If the count is larger than the number of characters after the cursor, this shall not be considered an error; all the characters from the cursor to the end of the line shall be deleted.

## Context

**Behavior:** In vi command mode, `x` deletes the character at the current cursor position and places the deleted characters in the save buffer (for later use by `p`/`P`).

**Edge cases:**
- If cursor is on the last character: delete it and move cursor to the previous character (new last character).
- If count exceeds characters after cursor: NOT an error; all characters from cursor to end of line are deleted.

**Implementation:** Delete `count` characters starting at cursor position (clamped to end of line). Store deleted text in save buffer. Adjust cursor position.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 576–579.
