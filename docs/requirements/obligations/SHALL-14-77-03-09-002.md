## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflush():: TCIFLUSH

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCIFLUSH` for use with `tcflush()`. When used, pending input data that has not been read is discarded.

**Conditions:** Used as the `queue_selector` parameter to `tcflush()`.

**Shell relevance:** meiksh uses `tcflush(fd, TCIFLUSH)` to discard unread input. This is useful after receiving SIGINT (to discard partially typed input), or when reinitializing the line editor.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 472.
