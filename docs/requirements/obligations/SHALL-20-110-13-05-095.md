## Extracted Text

> The following commands shall be recognized in command mode:: Put a copy of the current contents of the save buffer after the current cursor position. The current cursor position shall be advanced to the last character put from the save buffer. A count shall indicate how many copies of the save buffer shall be put.

## Context

**Behavior:** In vi command mode, `p` puts (pastes) the contents of the save buffer AFTER the current cursor position. The cursor is advanced to the last character put from the save buffer. With a count, put that many copies of the save buffer contents.

**Implementation:** Insert save buffer contents after cursor position, repeated `count` times. Set cursor to the last inserted character.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 612–614.
