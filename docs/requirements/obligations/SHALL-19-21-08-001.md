## Extracted Text

> The following environment variable shall affect the execution of exec:

## Context

**Behavior:** The `exec` special built-in utility is affected by the PATH environment variable. This is the introductory statement for the environment variables section of the `exec` built-in specification.

**Applies to:** All shell modes (interactive and non-interactive) when executing the `exec` built-in.

**Details:** The `exec` built-in replaces the current shell execution environment with a new process image (when a utility operand is provided) or modifies file descriptors in the current environment (when only redirections are given). The PATH variable determines where the shell searches for the utility named in the `exec` operand. Only PATH is listed as affecting `exec` — locale variables do not apply since `exec` itself produces no locale-sensitive output.

**Spec reference:** Section 2.21 "exec" ENVIRONMENT VARIABLES, line 3907 in `utilities/V3_chap02.html`.
