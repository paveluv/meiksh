## Extracted Text

> The following environment variables shall affect the execution of fc:: LC_ALL

## Context

The `fc` builtin must recognize the `LC_ALL` environment variable. When `LC_ALL` is set to a non-empty string, it overrides all other internationalization variables (`LANG`, `LC_CTYPE`, `LC_MESSAGES`, etc.) for the purpose of locale determination within `fc`. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
