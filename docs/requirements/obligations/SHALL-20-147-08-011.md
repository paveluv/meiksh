## Extracted Text

> The following environment variables shall affect the execution of wait:: [XSI] Determine the location of messages objects and message catalogs.

## Context

**Behavior:** On XSI-conformant systems, NLSPATH determines where message objects and message catalogs are located for use by `wait` when producing locale-dependent messages.

**Conditions:** Only applicable on XSI-conformant systems. Not a base POSIX requirement.
