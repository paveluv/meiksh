## Extracted Text

> The following additional options shall be supported:: -c

## Context

This is a label/header obligation for the `-c` option. The `-c` option tells the shell to read commands from the `command_string` operand rather than from a file or stdin. Detailed semantics are in SHALL-20-110-04-005 / SHALL-20-110-04-010.

See utilities/sh.html#tag_20_110_04.
