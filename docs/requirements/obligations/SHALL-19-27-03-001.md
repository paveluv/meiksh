## Extracted Text

> The positional parameters shall be shifted. Positional parameter 1 shall be assigned the value of parameter (1+n), parameter 2 shall be assigned the value of parameter (2+n), and so on. The parameters represented by the numbers "$#" down to "$#-n+1" shall be unset, and the parameter '#' is updated to reflect the new number of positional parameters.

## Context

**Behavior:** The `shift` special built-in shifts positional parameters by `n` positions:
1. `$1` gets the value of `$(1+n)`, `$2` gets `$(2+n)`, and so on.
2. The parameters `$#` down to `$(#-n+1)` are unset.
3. `$#` is updated to reflect the new count (`$# - n`).

**Example:** If `$1=a $2=b $3=c $#=3`, then `shift 1` results in `$1=b $2=c $#=2`.

**Applies to:** All shell modes. This is a special built-in.
