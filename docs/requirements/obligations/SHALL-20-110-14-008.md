## Extracted Text

> The following exit values shall be returned:: 127

## Context

**Behavior:** The `sh` utility shall return exit status 127 under the condition described in SHALL-20-110-14-009. This is the exit value label "127" in the exit status definition list.

**Applies to:** Non-interactive shell invocations where the command_file cannot be found.

**Details:** Exit status 127 indicates the specified command_file does not exist or cannot be found. This applies when `sh` is invoked with a command_file operand that is not locatable in the filesystem. See SHALL-20-110-14-009 for the specific condition.
