## Extracted Text

> When not inside a bracket expression, the following shall match a single character:: a <period>

## Context

When not inside a bracket expression, a period (`.`) must match any single character in the supported character set except NUL (`\0`). The period is a wildcard single-character-matching atom. It does not match NUL. Inside a bracket expression, the period loses its special meaning and matches only a literal period. This is the ERE equivalent of the BRE period rule (Section 9.3.4).
