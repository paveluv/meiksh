## Extracted Text

> The following shall be declared as functions and may also be defined as macros. Function prototypes shall be provided.

## Context

**Behavior:** `<termios.h>` **shall** declare the following as functions (with prototypes), and they **may** also be defined as macros:
- `speed_t cfgetispeed(const struct termios *)`
- `speed_t cfgetospeed(const struct termios *)`
- `int cfsetispeed(struct termios *, speed_t)`
- `int cfsetospeed(struct termios *, speed_t)`
- `int tcdrain(int)`
- `int tcflow(int, int)`
- `int tcflush(int, int)`
- `int tcgetattr(int, struct termios *)`
- `pid_t tcgetsid(int)`
- `int tcsendbreak(int, int)`
- `int tcsetattr(int, int, const struct termios *)`
- `pid_t tcgetpgrp(int)`
- `int tcsetpgrp(int, pid_t)`

Function prototypes must be provided so the compiler can type-check arguments.

**Conditions:** All these functions must be available after including `<termios.h>`.

**Shell relevance:** meiksh uses many of these functions directly:
- `tcgetattr`/`tcsetattr`: terminal mode manipulation
- `tcgetpgrp`/`tcsetpgrp`: job control (foreground process group)
- `tcflush`: discard pending I/O
- `tcdrain`: wait for output to complete
- `cfgetospeed`/`cfsetospeed`: baud rate handling for `stty`
- `tcgetsid`: get session leader for the terminal

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 494.
