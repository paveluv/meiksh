## Extracted Text

> The following exit values shall be returned:: 0

## Context

**Behavior required:** The `read` builtin must return exit status 0 on successful completion.

**Condition:** When `read` successfully reads a complete line (terminated by the delimiter) and assigns variables without error.

**Implementation detail:** Return 0 when:
- A line delimiter was encountered (newline or `-d delim` character)
- Variable assignment completed without errors (no readonly variable conflicts)
