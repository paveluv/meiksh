## Extracted Text

> The following options shall be supported:: -p

## Context

**Behavior:** The `command` built-in must support the `-p` option. When `-p` is specified, the command search uses a default PATH value guaranteed to find all standard utilities, rather than the current value of PATH.

**Applies to:** `command -p utility_name` invocations.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section OPTIONS (tag_20_22_04), line 62.
