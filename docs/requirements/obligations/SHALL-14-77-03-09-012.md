## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: Transmit a START character, intended to restart input data.

## Context

**Behavior:** This is the description of TCION: **transmit a START character, intended to restart input data**. When `tcflow(fd, TCION)` is called, a START character (typically ^Q) is sent to the terminal to signal that the sender should resume data transmission.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** See SHALL-14-77-03-09-011. This sends the XON character to request the terminal resume sending input.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 486.
