## Extracted Text

> If CREAD is set, the receiver shall be enabled; otherwise, no characters shall be received.

## Context

**Behavior:** When CREAD is set in `c_cflag`, the terminal receiver is enabled and characters can be received. When CREAD is not set, no characters shall be received from the terminal.

**Relevance to meiksh:** The `stty` builtin must support setting/clearing CREAD. This flag is typically always set for interactive terminals. Clearing it would prevent any input from being received.

**Spec reference:** POSIX.1-2024, §11.2.4 Control Modes (tag_11_02_04), `basedefs/V1_chap11.html` line 1070.
