## Extracted Text

> The following environment variables shall affect the execution of alias:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

If `LC_ALL` is set to a non-empty string value, it **shall** override the values of all other internationalization variables for the `alias` builtin. This is the highest-priority locale override mechanism. Reference: alias utility ENVIRONMENT VARIABLES, XBD 8.2.
