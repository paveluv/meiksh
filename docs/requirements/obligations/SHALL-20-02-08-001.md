## Extracted Text

> The following environment variables shall affect the execution of alias:

## Context

The `alias` builtin's execution **shall** be affected by the standard internationalization environment variables: `LANG`, `LC_ALL`, `LC_CTYPE`, `LC_MESSAGES`, and (XSI) `NLSPATH`. These variables affect character interpretation and diagnostic message formatting. The shell must ensure these variables are consulted when `alias` produces output or diagnostic messages. Reference: alias utility ENVIRONMENT VARIABLES.
