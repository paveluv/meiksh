## Extracted Text

> Atomic read-modify-write operations shall always read the last value (in the modification order) stored before the write associated with the read-modify-write operation.

## Context

This is a C-level memory model requirement about atomic read-modify-write operations. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.1.2 Memory Order and Consistency.
