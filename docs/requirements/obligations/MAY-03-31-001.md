## Extracted Text

> This implies that the process and the I/O operation may be running concurrently.

## Context

**Definition only — no shell relevance.** Defines "asynchronous I/O operation" as an I/O operation where the process and I/O may run concurrently. This is a POSIX AIO concept (`aio_read`, `aio_write`).

**Shell impact:** None. The shell command language does not expose asynchronous I/O operations. Shell I/O redirection uses synchronous read/write. No action required.
