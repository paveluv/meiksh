## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcsetattr():: TCSADRAIN

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCSADRAIN` for use with `tcsetattr()`. When used, terminal attribute changes take effect after all pending output has been transmitted (drained).

**Conditions:** Used as the `optional_actions` parameter to `tcsetattr()`.

**Shell relevance:** meiksh uses TCSADRAIN when changing terminal modes after producing output, to ensure the output is fully transmitted under the old mode settings before the new mode takes effect. For example, when restoring terminal modes after command execution and before displaying the next prompt.

**Spec reference:** POSIX.1-2024, `<termios.h>` Attribute Selection (tag_14_77_03_08), `basedefs/termios.h.html` line 462.
