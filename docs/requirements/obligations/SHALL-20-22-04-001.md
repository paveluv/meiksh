## Extracted Text

> The command utility shall conform to XBD 12.2 Utility Syntax Guidelines.

## Context

**Behavior:** The `command` built-in must parse its options according to the standard Utility Syntax Guidelines (XBD 12.2). This means:
- Options are single characters preceded by `-`.
- Options can be grouped (e.g., `-pv` is equivalent to `-p -v`).
- The `--` argument terminates option processing.
- Option arguments follow standard conventions.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section OPTIONS (tag_20_22_04), line 57. See XBD 12.2.
