## Extracted Text

> If a side effect X on an atomic object M happens before a value computation B of M, then the evaluation B shall take its value from X or from a side effect Y that follows X in the modification order of M. (This is known as "write-read coherence".)

## Context

This is a C-level memory model requirement about write-read coherence for atomic objects. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.1.1 Data Races.
