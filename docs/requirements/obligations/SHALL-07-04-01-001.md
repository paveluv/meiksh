## Context

The locale definition grammar shall process the following tokens (in addition to string constants shown in the grammar): LOC_NAME, CHAR, NUMBER, COLLSYMBOL, COLLELEMENT, CHARCLASS, CHARSYMBOL, OCTAL_CHAR, HEX_CHAR, DECIMAL_CHAR, ELLIPSIS, EXTENDED_REG_EXP, and EOL.

**Shell relevance:** Informational — the shell consumes compiled locale data, not raw locale definition source. See XBD 7.4.1.
