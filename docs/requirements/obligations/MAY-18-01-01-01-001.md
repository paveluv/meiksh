## Extracted Text

> A conforming implementation may include additional process attributes.

## Context

A conforming implementation may include additional process attributes beyond the minimum set listed in POSIX 1.1.1.1 Process Attributes (which includes: Current Working Directory, File Mode Creation Mask, File Descriptors, etc.). For `meiksh`, this means the shell should manage the POSIX-required process attributes and may encounter additional platform-specific attributes. The shell must correctly propagate at least the standard process attributes across `fork()`/`exec()` boundaries. See POSIX Utilities Chapter 1, Section 1.1.1.1 (utilities/V3_chap01.html line 69).
