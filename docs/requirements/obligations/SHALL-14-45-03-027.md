> The following signals shall be supported on all implementations (default actions are explained below the table):

## Context

All POSIX implementations must support the standard signal set. The mandatory signals include (with default actions): SIGABRT (A-abnormal termination), SIGALRM (T-terminate), SIGBUS (A-abort), SIGCHLD (I-ignore), SIGCONT (C-continue), SIGFPE (A-abort), SIGHUP (T-terminate), SIGILL (A-abort), SIGINT (T-terminate), SIGKILL (T-terminate, cannot be caught), SIGPIPE (T-terminate), SIGQUIT (A-abort), SIGSEGV (A-abort), SIGSTOP (S-stop, cannot be caught), SIGTERM (T-terminate), SIGTSTP (S-stop), SIGTTIN (S-stop), SIGTTOU (S-stop), SIGUSR1 (T-terminate), SIGUSR2 (T-terminate), SIGURG (I-ignore), plus XSI signals like SIGPOLL, SIGPROF, SIGSYS, SIGTRAP, SIGVTALRM, SIGXCPU, SIGXFSZ. For `meiksh`, the shell must: (1) know the default action for each signal, (2) support trapping all catchable signals, (3) correctly report signal names and numbers in `kill -l` and `trap`, (4) handle SIGCHLD for job control, (5) handle SIGINT/SIGQUIT/SIGTERM/SIGHUP for interactive and non-interactive modes.
