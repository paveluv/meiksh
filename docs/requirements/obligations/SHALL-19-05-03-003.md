## Extracted Text

> The following variables shall affect the execution of the shell:: ENV

## Context

This is the heading obligation for the `ENV` shell variable. It introduces `ENV` as one of the variables that shall affect execution. The detailed semantics are in SHALL-19-05-03-004 and the duplicate SHALL-19-05-03-035.

**Required behavior:**
- The shell must recognize `ENV` as a shell variable with special meaning.
- The detailed behavior (reading the ENV file for interactive shells) is covered in SHALL-19-05-03-004.

See 2.5.3 "Shell Variables" (`ENV`) in `utilities/V3_chap02.html`.
