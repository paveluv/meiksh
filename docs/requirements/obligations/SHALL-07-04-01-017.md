## Extracted Text

> The following tokens shall be processed (in addition to those string constants shown in the grammar):: One or more octal representations of the encoding of each byte in a single character. The octal representation consists of an escape character (normally a <backslash>) followed by two or more octal digits.

## Context

OCTAL_CHAR is defined as one or more octal representations of the encoding of each byte in a single character. The format is an escape character (normally backslash) followed by two or more octal digits.

**Shell relevance:** Informational for locale infrastructure. See XBD 7.4.1.
