## Extracted Text

> The following environment variables shall affect the execution of read:: NLSPATH

## Context

**Behavior required:** The `NLSPATH` environment variable must affect the execution of the `read` builtin. It determines the location of message catalogs (XSI extension).

**Condition:** When XSI conformance is required and message catalogs are used.

**Implementation detail:** NLSPATH specifies the search path for message catalogs used by `catopen()`. For `read`, this affects localized diagnostic messages. A POSIX/C locale-only implementation can note this as not applicable.

**Spec reference:** Section ENVIRONMENT VARIABLES of `utilities/read.html`, line 165.
