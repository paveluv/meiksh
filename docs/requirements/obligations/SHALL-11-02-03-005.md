## Extracted Text

> If a <form-feed> or <vertical-tab> delay is specified, it shall last for about 2 seconds.

## Context

**Behavior:** If a form-feed or vertical-tab output delay is specified (via FFDLY or VTDLY flags in `c_oflag`), the delay shall last approximately 2 seconds.

**Relevance to meiksh:** The `stty` builtin must support setting FFDLY and VTDLY delay values. The actual delay is enforced by the OS terminal driver.

**Spec reference:** POSIX.1-2024, §11.2.3 Output Modes (tag_11_02_03), `basedefs/V1_chap11.html` line 798.
