## Extracted Text

> The operand is a string containing the symbol that shall be used as a separator for groups of digits to the left of the decimal delimiter in formatted monetary quantities.

## Context

The `mon_thousands_sep` value is the string used to separate digit groups to the left of the decimal point in monetary quantities (e.g., "," in "$1,000.00"). The shell does not directly format monetary values. The shell must correctly propagate the LC_MONETARY environment variable so programs can access this through `localeconv()`.

See: `basedefs/V1_chap07.html` §7.3.3 LC_MONETARY.
