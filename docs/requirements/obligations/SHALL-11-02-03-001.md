## Context

**Behavior:** The `c_oflag` field in the `termios` structure specifies how the terminal interface treats output. It is composed of the bitwise-inclusive OR of output mode masks, which shall be bitwise-distinct (no two mask values share bits). The mask name symbols are defined in `<termios.h>`.

**Relevance to meiksh:** The `stty` builtin must correctly read and set `c_oflag` bits. The shell must recognize all standard output mode flag names (OPOST, ONLCR, OCRNL, ONOCR, ONLRET, OFILL, OFDEL, and delay masks) when processing `stty` operands.

**Spec reference:** POSIX.1-2024, §11.2.3 Output Modes (tag_11_02_03), `basedefs/V1_chap11.html` lines 538–540.
