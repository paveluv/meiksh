## Extracted Text

> The following environment variables shall affect the execution of fg:: LC_CTYPE

## Context

The `fg` builtin must recognize the `LC_CTYPE` environment variable. `LC_CTYPE` determines how byte sequences are interpreted as characters in `fg` arguments (e.g., the `job_id` operand). The standard `LC_ALL` > `LC_CTYPE` > `LANG` precedence applies. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
