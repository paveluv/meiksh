## Extracted Text

> A character that in the output stream should cause a terminal to alert its user via a visual or audible notification. It is the character designated by '\a' in the C language. It is unspecified whether this character is the exact sequence transmitted to an output device by the system to accomplish the alert function.

## Context

**Definition — moderate shell relevance.** Defines the alert character `<alert>` as the character designated by `\a` in C (BEL, 0x07). It is unspecified whether this byte is the exact sequence sent to the terminal to produce the alert.

**Shell impact:** The shell must recognize `\a` as the alert character in escape-sequence contexts: `printf '\a'`, `echo` with XSI interpretation, and `$'\a'` quoting. The shell writes byte 0x07; the terminal handles the rest. No implementation-specific alert mechanism is needed.
