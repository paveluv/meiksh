## Extracted Text

> The <termios.h> header shall define the winsize structure, which shall include at least the following members:

## Context

**Behavior:** `<termios.h>` **shall** define `struct winsize` with at least these members:
- `unsigned short ws_col` — number of columns
- `unsigned short ws_row` — number of rows
- (Additional members may exist, such as `ws_xpixel` and `ws_ypixel`)

**Conditions:** This structure is used with `TIOCGWINSZ` and `TIOCSWINSZ` ioctls to get/set terminal window size.

**Shell relevance:** meiksh needs the window size for:
- Setting the `COLUMNS` and `LINES` environment variables
- Line editor display: knowing the terminal width for line wrapping, cursor positioning, and multi-line editing
- Handling `SIGWINCH` signal (window size change) to update the stored dimensions
- The `stty` builtin must support displaying `rows` and `columns` using this structure

**Spec reference:** POSIX.1-2024, `<termios.h>` The winsize Structure (tag_14_77_03_07), `basedefs/termios.h.html` line 449.
