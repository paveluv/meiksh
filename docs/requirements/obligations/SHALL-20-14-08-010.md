## Extracted Text

> The following environment variables shall affect the execution of cd:: LC_CTYPE

## Context

**Behavior:** The `LC_CTYPE` environment variable must affect `cd` execution. LC_CTYPE is recognized as a locale variable that influences character interpretation for the cd built-in.

**Applies to:** Character interpretation in cd arguments.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 190.
