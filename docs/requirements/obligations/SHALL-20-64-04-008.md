## Extracted Text

> The following options shall be supported:: [XSI] Equivalent to -s signal_name.

## Context

**Behavior:** The `-signal_name` option form (XSI) is exactly equivalent to `-s signal_name`. The implementation must treat `kill -TERM pid` identically to `kill -s TERM pid`.

**Conditions:** This equivalence holds for all signal names recognized by the `-s` option, including the special name `0` and case-independent matching.
