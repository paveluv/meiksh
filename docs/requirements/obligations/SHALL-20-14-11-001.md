## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

**Behavior:** The cd built-in must use stderr exclusively for diagnostic (error) messages. No informational or status output may be written to stderr. Only error conditions (e.g., directory not found, permission denied, empty string operand) should produce stderr output.

**Forbidden:** Writing any non-diagnostic content to stderr during cd execution.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section STDERR (tag_20_14_11), line 227.
