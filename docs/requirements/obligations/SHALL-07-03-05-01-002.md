## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: copy

## Context

The `copy` keyword shall be recognized in an LC_TIME locale definition. When present, it names an existing locale whose LC_TIME definition is reused verbatim for this locale.

**Shell relevance:** The shell does not parse locale definition files directly, but must respect the resulting LC_TIME data provided by the system's locale infrastructure. See XBD 7.3.5.1.
