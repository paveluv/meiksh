## Extracted Text

> When job control is enabled, the shell shall create one or more jobs when it executes a list (see 2.9.3 Lists) that has one of the following forms:: A single asynchronous AND-OR list

## Context

**Job creation form 1:** A list consisting of a single asynchronous AND-OR list (e.g., `cmd1 && cmd2 &`) creates a single job. This is one of the two forms that trigger job creation under job control. The resulting job is a background job (see SHALL-19-11-011 for how background jobs are set up with process groups).
