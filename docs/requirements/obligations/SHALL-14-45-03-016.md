## Extracted Text

> The <signal.h> header shall define the following symbolic constants for the values of sigev_notify:: SIGEV_SIGNAL

## Context

`SIGEV_SIGNAL` must be defined in `<signal.h>`. When set as `sigev_notify`, a queued signal with an application-defined value is generated when the event occurs. The signal number is specified by `sigev_signo` and the value by `sigev_value`. For `meiksh`, this is the most common notification type if the shell uses POSIX timers (e.g., for the `TMOUT` variable implementation). This is a system header requirement.
