## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: MINSIGSTKSZ

## Context

The `<signal.h>` header shall define the symbolic constant `MINSIGSTKSZ`. This is an [XSI] constant representing the minimum stack size in bytes for an alternate signal handler stack. Any alternate stack allocated via `sigaltstack()` must be at least `MINSIGSTKSZ` bytes. For `meiksh`, this is not directly relevant since the shell does not use alternate signal stacks. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html line 549).
