## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use as flags in the c_lflag field. The c_lflag field of the argument structure is used to control various terminal functions.

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for the `c_lflag` field (local modes). These include at minimum: ECHO, ECHOE, ECHOK, ECHONL, ICANON, IEXTEN, ISIG, NOFLSH, TOSTOP.

**Conditions:** Must be defined as bitmask constants suitable for bitwise operations.

**Shell relevance:** Local mode flags are the most critical for the shell's terminal management:
- **ICANON**: canonical (line) mode vs. non-canonical (raw) mode — the shell clears this for line editing
- **ECHO**: echo input characters — the shell may clear this for password input or line editor control
- **ISIG**: enable signal generation (SIGINT, SIGQUIT, SIGTSTP) — typically kept set
- **IEXTEN**: implementation-defined extended input processing
- **ECHOE/ECHOK**: erase/kill echo behavior
- **NOFLSH**: suppress queue flush on INTR/QUIT/SUSP
- **TOSTOP**: stop background processes on write
The `stty` builtin must support all these flag names.

**Spec reference:** POSIX.1-2024, `<termios.h>` Local Modes (tag_14_77_03_06), `basedefs/termios.h.html` lines 425–426.
