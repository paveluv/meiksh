## Extracted Text

> The cd utility shall change the working directory of the current shell execution environment (see 2.13 Shell Execution Environment) by executing the following steps in sequence. (In the following steps, the symbol curpath represents an intermediate value used to simplify the description of the algorithm used by cd. There is no requirement that curpath be made visible to the application.)

## Context

The `cd` builtin **shall** change the working directory of the current shell execution environment (XCU 2.13) by executing a specific multi-step algorithm. The algorithm uses an intermediate variable `curpath` (not visible to the application) and processes steps sequentially: determining the directory operand (or default), resolving `CDPATH`, canonicalizing the path, calling `chdir()`, and updating `PWD`/`OLDPWD`. The shell must implement all steps of this algorithm in order. Reference: cd utility DESCRIPTION, XCU 2.13.
