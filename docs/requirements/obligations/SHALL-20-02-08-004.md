## Extracted Text

> The following environment variables shall affect the execution of alias:: LC_ALL

## Context

`LC_ALL` **shall** affect the execution of `alias`. When set to a non-empty string, it overrides all other internationalization variables (`LANG`, `LC_CTYPE`, `LC_MESSAGES`, etc.), forcing all locale categories to the value of `LC_ALL`. Reference: alias utility ENVIRONMENT VARIABLES, XBD 8.2.
