## Extracted Text

> Execute the current command line. If the current command line is not empty, this line shall be entered into the command history (see fc).

## Context

**Behavior:** On `<newline>` in vi command mode, execute the current command line. If the current command line is not empty, add it to the command history (see `utilities/fc.html`).

**Details:** Same normative content as SHALL-20-110-13-05-006. This obligation captures the standalone "shall" from the `<newline>` behavior description.

**Implementation:** On newline: (1) if non-empty, append to history, (2) submit for shell execution, (3) reset editing state.
