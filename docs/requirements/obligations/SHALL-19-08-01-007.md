## Extracted Text

> Interactive Shell: Other utility (not a special built-in) error -- shall not exit

## Context

An interactive shell SHALL NOT exit when a non-special-built-in utility encounters an error. See 2.8.1 Consequences of Shell Errors in utilities/V3_chap02.html.

**Behavior required:**
- When the shell is interactive and a non-special built-in or external utility encounters an error, the shell SHALL NOT exit.
- The shell continues to the next prompt.
- A shell diagnostic message is NOT required — the utility writes its own diagnostic.

**This is symmetric** with SHALL-19-08-01-006: both interactive and non-interactive shells survive errors from non-special utilities.
