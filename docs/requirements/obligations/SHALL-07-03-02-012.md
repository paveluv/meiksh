## Extracted Text

> Define collation rules. This statement shall be followed by one or more collation order statements, assigning character collation values and collation weights to collating elements.

## Context

**Behavior:** The `order_start` keyword begins the collation order definition. It must be followed by one or more collation order statements that assign collation values and weights to collating elements. The section ends with `order_end`.

**Shell relevance:** The shell does not parse locale definition files. This is `localedef` input syntax defining the collation order. The shell uses the compiled result via `strcoll()` and pattern matching bracket expressions.
