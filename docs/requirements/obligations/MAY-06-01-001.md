## Extracted Text

> Portable Character Set defines the characters in the portable character set and the corresponding symbolic character names used to identify each character in a character set description file. Characters defined in Non-Portable Control Characters may also be used in character set description files.

## Context

This is an informational note about character set description files (charmap files). The shell does not directly process charmap files — locale support is provided by the OS. However, the shell must be aware that the portable character set defines the minimum set of characters available in all locales. Non-portable control characters (like BEL, BS, FF, etc.) may be present but are not guaranteed across all locales. No direct shell action is required.

See: `basedefs/V1_chap06.html` §6.1 Portable Character Set.
