## Extracted Text

> When not inside a bracket expression, the following shall match a single character:: a BRE ordinary character

## Context

**Behavior:** A BRE ordinary character (not special) outside a bracket expression shall match itself — the single character it represents.

**Shell relevance:** In shell pattern matching, an ordinary character (not `?`, `*`, `[`, or `\` in pattern context) matches itself. This is the most basic matching rule: the literal character `a` in a pattern matches the character `a` in the string.
