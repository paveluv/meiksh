## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use as subscripts for the array c_cc:

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for indexing the `c_cc` array. These include at minimum:
- Canonical mode: VEOF, VEOL, VERASE, VKILL
- Non-canonical mode: VMIN, VTIME
- Both modes: VINTR, VQUIT, VSTART, VSTOP, VSUSP

These constants are integer values suitable for array subscripting: `c_cc[VINTR]`, `c_cc[VEOF]`, etc.

**Conditions:** Must be defined as compile-time constants suitable for `#if` preprocessing directives (see SHALL-14-77-03-01-015).

**Shell relevance:** meiksh uses these subscripts extensively:
- `c_cc[VMIN] = 1; c_cc[VTIME] = 0;` for non-canonical single-char reads
- `c_cc[VINTR]` to read/set the interrupt character
- `c_cc[VEOF]` for the EOF character
- `c_cc[VSUSP]` for the suspend character
The `stty` builtin must map between user-visible names (intr, quit, erase, etc.) and these subscript constants.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` lines 71–72.
