## Extracted Text

> If -S is specified, the message shall be in the following format:

## Context

**Behavior:** When `umask` is invoked with `-S` and no mask operand, the output shall be in the symbolic format: `"u=%s,g=%s,o=%s\n"` where each `%s` is a combination of letters from the set {r, w, x}.

**Output format:** The format string is `u=%s,g=%s,o=%s\n` with:
- `%s` for owner permissions: combination of r, w, x
- `%s` for group permissions: combination of r, w, x
- `%s` for other permissions: combination of r, w, x

**Example:** For mask 022, output is `u=rwx,g=rx,o=rx\n`.
