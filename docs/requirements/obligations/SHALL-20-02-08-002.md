## Extracted Text

> The following environment variables shall affect the execution of alias:: LANG

## Context

`LANG` **shall** affect the execution of `alias`. It provides a default value for internationalization variables (`LC_*`) that are unset or null. If `LC_ALL` is not set and a specific `LC_*` variable is not set, the `LANG` value is used as the default locale for that category. Reference: alias utility ENVIRONMENT VARIABLES, XBD 8.2.
