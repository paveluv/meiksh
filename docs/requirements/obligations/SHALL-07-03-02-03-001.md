> The order_start keyword shall precede collation order entries and also define the number of weights for this collation sequence definition and other collation rules. The syntax is as follows:

## Context

**Behavior:** The `order_start` keyword marks the beginning of collation order entries. It also specifies:
1. The number of collation weights (determined by the number of operands).
2. Collation directives for each weight level (forward, backward, position).

Syntax: `order_start forward;backward` (example with two weight levels).

**Shell relevance:** The shell does not parse locale definition files. This is `localedef` input syntax. The number of weights and their directives affect the compiled collation, which the shell uses via `strcoll()`.
