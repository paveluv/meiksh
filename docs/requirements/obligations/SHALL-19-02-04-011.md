## Extracted Text

> A sequence of characters starting with a <dollar-sign> immediately followed by a single-quote ($') shall preserve the literal value of all characters up to an unescaped terminating single-quote ('), with the exception of certain <backslash>-escape sequences, as follows:: \t yields a <tab> character.

## Context

**Behavior:** Inside `$'...'`, the escape sequence `\t` yields a tab character.

**Applies to:** All shell modes during `$'...'` escape processing.

**Implementation detail:** Emit byte 0x09 (HT/tab). This is the same character that C's `'\t'` produces.

**Spec reference:** Section 2.2.4, lines 196-197 in `utilities/V3_chap02.html`.
