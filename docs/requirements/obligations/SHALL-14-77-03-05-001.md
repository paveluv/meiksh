## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use as flags in the c_cflag field. The c_cflag field describes the hardware control of the terminal; not all values specified are required to be supported by the underlying hardware.

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for the `c_cflag` field (control modes). These include at minimum: CLOCAL, CREAD, CS5, CS6, CS7, CS8, CSIZE, CSTOPB, HUPCL, PARENB, PARODD. Not all values need to be supported by the underlying hardware.

**Conditions:** Must be defined as bitmask constants. CSIZE is a multi-bit mask for character size (CS5–CS8).

**Shell relevance:** meiksh uses control mode flags:
- CLOCAL: ignore modem status lines (affects SIGHUP behavior, see SHALL-11-01-10-001)
- HUPCL: hang up on last close (see SHALL-11-01-11-001)
- CREAD: enable receiver
- CS7/CS8: character size (typically CS8 for 8-bit data)
The `stty` builtin must support all these flag names.

**Spec reference:** POSIX.1-2024, `<termios.h>` Control Modes (tag_14_77_03_05), `basedefs/termios.h.html` lines 390–391.
