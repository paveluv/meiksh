## Extracted Text

> Undo all changes made to the edit line. This operation shall not undo the copy of any command line to the edit line.

## Context

**Behavior:** The `U` command shall undo ALL changes made to the edit line. This shall NOT undo the copy of a command line to the edit line (history promotion).
