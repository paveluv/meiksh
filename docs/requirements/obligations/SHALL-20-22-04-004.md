## Extracted Text

> The following options shall be supported:: Perform the command search using a default value for PATH that is guaranteed to find all of the standard utilities.

## Context

**Behavior:** This is the description of the `-p` option. When `command -p` is used:
1. The command search must use a default system PATH (not the user's current PATH).
2. This default PATH must be guaranteed to include directories containing all standard POSIX utilities.
3. The default PATH can be obtained via `getconf PATH`, but the shell should have a compiled-in default.
4. `-p` can be combined with `-v` or `-V` to show what would be found using the default PATH.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section OPTIONS (tag_20_22_04), line 63.
