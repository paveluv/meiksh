## Context

The `<signal.h>` header shall define the `ucontext_t` type as a structure ([CX] extension) with at least the following members: `ucontext_t *uc_link` (pointer to context resumed when this one returns), `sigset_t uc_sigmask` (blocked signal set when this context is active), `stack_t uc_stack` (the stack used by this context), and `mcontext_t uc_mcontext` (machine-specific saved context). This is the third argument passed to `SA_SIGINFO` signal handlers (cast from `void*`). For `meiksh`, this type is provided by the platform; the shell does not directly manipulate `ucontext_t`. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 559–570).
