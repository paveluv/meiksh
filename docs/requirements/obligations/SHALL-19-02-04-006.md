## Extracted Text

> A sequence of characters starting with a <dollar-sign> immediately followed by a single-quote ($') shall preserve the literal value of all characters up to an unescaped terminating single-quote ('), with the exception of certain <backslash>-escape sequences, as follows:: \b yields a <backspace> character.

## Context

**Behavior:** Inside `$'...'`, the escape sequence `\b` yields the backspace character.

**Applies to:** All shell modes during `$'...'` escape processing.

**Implementation detail:** Emit byte 0x08 (BS/backspace). This is the same character that C's `'\b'` produces.

**Spec reference:** Section 2.2.4, lines 181-182 in `utilities/V3_chap02.html`.
