## Extracted Text

> The following environment variables shall affect the execution of sh:: LC_COLLATE

## Context

**Behavior:** The LC_COLLATE environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-017).

**Applies to:** All shell modes (interactive and non-interactive) during pattern matching operations.

**Details:** LC_COLLATE determines the collation behavior for range expressions, equivalence classes, and multi-character collating elements within pattern matching (e.g., `[a-z]`, `[[=a=]]`). See SHALL-20-110-08-017 for the full description.
