## Extracted Text

> There shall be a single total order S on all memory_order_seq_cst operations, consistent with the "happens before" order and modification orders for all affected locations, such that each memory_order_seq_cst operation B that loads a value from an atomic object M observes one of the following values:: if A exists, the result of some modification of M that is not memory_order_seq_cst and that does not happen before A, or

## Context

This is a C-level memory model requirement about sequential consistency load values (non-seq_cst interleaving case). It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.1.2 Memory Order and Consistency.
