## Extracted Text

> The following options shall be supported by the implementation:: Handle the operand dot-dot physically; symbolic link components shall be resolved before dot-dot components are processed (see step 7. in the DESCRIPTION).

## Context

With `-P` (physical mode), the `cd` builtin **shall** handle dot-dot physically: symbolic link components are resolved before dot-dot components are processed (step 7 of the DESCRIPTION). This means `cd ..` from a symlinked directory goes to the physical parent of the actual target directory, not the logical parent where the symlink resides. The shell resolves symlinks in the path first, then processes `..` against the resolved physical path. Reference: cd utility OPTIONS, cd utility DESCRIPTION step 7.
