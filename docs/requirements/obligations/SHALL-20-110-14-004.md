## Extracted Text

> The following exit values shall be returned:: 1-125

## Context

**Behavior:** The `sh` utility shall return an exit status in the range 1-125 under the condition described in SHALL-20-110-14-005. This is the exit value label "1-125" in the exit status definition list.

**Applies to:** Non-interactive shell invocations that detect errors.

**Details:** This range is used for errors detected by a non-interactive shell, excluding command_file not found (127), command_file not executable (126), and unrecoverable read errors (128). See SHALL-20-110-14-005 for the specific conditions.
