## Extracted Text

> The implementation shall support any regular expression that does not exceed 256 bytes in length.

## Context

**Behavior:** The implementation shall support any regular expression that does not exceed 256 bytes in length.

**Shell relevance:** Shell patterns (glob patterns, `case` patterns) that contain bracket expressions must support patterns up to at least 256 bytes. The shell must not impose a pattern length limit smaller than 256 bytes. In practice, shell patterns are typically much shorter, but the implementation must handle the full 256-byte minimum.
