> The sigval union shall be defined as:

## Context

The `sigval` union must be defined with two members: `int sival_int` (integer signal value) and `void *sival_ptr` (pointer signal value). This union is used in signal queuing and event notification to pass application-specific data. For `meiksh`, `sigval` is relevant when using `sigqueue()` to send signals with data or when receiving signals via `sigwaitinfo()`/`sigtimedwait()` that carry a payload. The shell relies on the system header defining this union correctly.
