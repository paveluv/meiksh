## Extracted Text

> The following environment variables shall affect the execution of command:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error and informative messages written to standard output.

## Context

**Behavior:** LC_MESSAGES description for `command`. Determines the locale used for:
1. Diagnostic messages written to stderr (e.g., "command: not found").
2. Informative messages written to stdout (e.g., the `-V` option output which describes commands in human-readable form).

Note: Unlike most utilities where LC_MESSAGES only affects stderr, for `command` it also affects stdout (the `-V` output).

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section ENVIRONMENT VARIABLES (tag_20_22_08), line 153.
