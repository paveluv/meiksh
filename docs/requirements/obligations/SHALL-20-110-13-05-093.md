## Extracted Text

> The following commands shall be recognized in command mode:: Yank the characters from the current cursor position to the end of the line into the save buffer. The current character position shall be unchanged.

## Context

**Behavior:** In vi command mode, `Y` yanks characters from the current cursor position to the end of the line into the save buffer. Cursor position is unchanged. Equivalent to `y$`.
