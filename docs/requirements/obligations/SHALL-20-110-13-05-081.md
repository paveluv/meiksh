## Extracted Text

> The following commands shall be recognized in command mode:: Append a <space> after the current character position and then append the last bigword in the previous input line after the <space>. Then enter insert mode after the last character just appended. With a number count, append the countth bigword in the previous line.

## Context

**Behavior:** In vi command mode, `_` appends a space after the current cursor position, then appends the last bigword from the previous input line, and enters insert mode after the last appended character. With a count, append the count-th bigword (1-indexed) from the previous input line instead of the last.

**Implementation:** Parse the previous command line in history into bigwords. Select the last (or count-th) bigword. Insert a space + that bigword after the current cursor. Enter insert mode after the inserted text.
