## Extracted Text

> The following commands shall be recognized in command mode:: [count]fc

## Context

**Behavior:** The `[count]fc` (find character) command shall be recognized in vi command mode. See SHALL-20-110-13-05-051 and SHALL-20-110-13-05-136 for the detailed behavior. Note: `f` here is the vi find-character-forward command, not the `fc` builtin.

**Command key:** `f` followed by a target character `c`, optionally preceded by a count. This is a motion command.
