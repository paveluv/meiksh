## Extracted Text

> The <termios.h> header shall define the following symbolic constant:: NCCS

## Context

**Behavior:** `<termios.h>` **shall** define the symbolic constant `NCCS`. This constant specifies the size of the `c_cc` array in `struct termios`.

**Conditions:** Must be defined as a compile-time constant suitable for array sizing.

**Shell relevance:** The `c_cc` array is declared as `cc_t c_cc[NCCS]`. NCCS must be large enough to hold all standard subscript indices (VINTR, VQUIT, VERASE, VKILL, VEOF, VEOL, VMIN, VTIME, VSTART, VSTOP, VSUSP, and potentially others). meiksh can use NCCS when iterating over or copying control character arrays.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 68.
