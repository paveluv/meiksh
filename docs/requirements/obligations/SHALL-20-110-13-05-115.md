## Extracted Text

> The following commands shall be recognized in command mode:: Repeat the most recent / or ? command. If there is no previous / or ?, the terminal shall be alerted and the current command line shall remain unchanged.

## Context

**Behavior:** In vi command mode, `n` repeats the most recent `/` or `?` search command in the same direction. If there is no previous `/` or `?` command, the terminal shall be alerted and the command line remains unchanged.

**Implementation:** Store the last search direction and pattern. On `n`, re-execute the search in the same direction from the current history position.
