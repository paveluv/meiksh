## Context

**Behavior:** Omitting the `in word...` clause from a `for` loop:

```
for name do compound-list ; done
```

must be equivalent to:

```
for name in "$@" ; do compound-list ; done
```

The loop iterates over the current positional parameters.

**Applies to:** All shell modes.
