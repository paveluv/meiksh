## Extracted Text

> If CLOCAL is set, a connection shall not depend on the state of the modem status lines. If CLOCAL is clear, the modem status lines shall be monitored.

## Context

**Behavior:** When CLOCAL is set in `c_cflag`, a connection does not depend on the state of the modem status lines (DCD, etc.). When CLOCAL is clear, the modem status lines shall be monitored and the connection depends on their state.

**Relevance to meiksh:** The `stty` builtin must support setting/clearing CLOCAL (via `clocal`/`-clocal` operands). For pseudo-terminals, CLOCAL is typically set since there is no modem.

**Spec reference:** POSIX.1-2024, §11.2.4 Control Modes (tag_11_02_04), `basedefs/V1_chap11.html` line 1075.
