## Extracted Text

> If no operands are given, all alias definitions shall be written to standard output.

## Context

When `alias` is invoked with no operands, all alias definitions **shall** be written to stdout. Each alias is output in the format `alias-name=value` (one per line) with the value appropriately quoted so it is suitable for reinput to the shell. The order of output is unspecified. Reference: alias utility OPERANDS.
