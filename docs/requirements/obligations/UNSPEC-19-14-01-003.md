## Context

**Trailing unescaped backslash in a pattern: behavior is unspecified.**

If a pattern ends with an unescaped `\` (e.g., the pattern `abc\`), the behavior is **unspecified**. The implementation may:
- Treat the trailing `\` as a literal backslash
- Report an error
- Produce undefined matching results

Portable scripts must not produce patterns ending with a bare backslash. The implementation should handle this gracefully (e.g., treat as literal or ignore) but no specific behavior is mandated.
