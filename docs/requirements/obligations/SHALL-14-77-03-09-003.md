## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflush():: Flush pending input.

## Context

**Behavior:** This is the description of TCIFLUSH: **flush pending input**. When `tcflush(fd, TCIFLUSH)` is called, all data received but not yet read from the terminal is discarded.

**Conditions:** Used as the `queue_selector` parameter to `tcflush()`.

**Shell relevance:** See SHALL-14-77-03-09-002. Discards typeahead or stale input from the terminal input buffer.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 473.
