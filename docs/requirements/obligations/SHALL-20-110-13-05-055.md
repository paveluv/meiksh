## Extracted Text

> The following commands shall be recognized in command mode:: Move to the character before the first occurrence of the character 'c' that occurs after the current cursor position. If the cursor was positioned on the last character of the line, the terminal shall be alerted and the cursor shall not be advanced. If the character 'c' does not occur in the line after the current cursor position, the terminal shall be alerted and the cursor shall not be moved.

## Context

**Behavior:** In vi command mode, `tc` moves the cursor to the character BEFORE the first occurrence of character `c` that occurs after the current cursor position (i.e., one position before where `fc` would land).

**Edge cases:**
- If cursor is on the last character: alert the terminal (bell), do NOT advance.
- If character `c` does not occur after cursor: alert the terminal, do NOT move.

**Implementation:** Same as `f` but position cursor one character before the match. This is a motion command.
