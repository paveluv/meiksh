## Extracted Text

> The <termios.h> header shall define the termios structure, which shall include at least the following members:

## Context

**Behavior:** `<termios.h>` **shall** define `struct termios` with at least these members:
- `tcflag_t c_iflag` — input modes
- `tcflag_t c_oflag` — output modes
- `tcflag_t c_cflag` — control modes
- `tcflag_t c_lflag` — local modes
- `cc_t c_cc[NCCS]` — control characters array

The structure may include additional implementation-specific members.

**Conditions:** All five members listed above must be present. Additional members are permitted.

**Shell relevance:** meiksh accesses all five standard members: `c_iflag` for input processing (ICRNL, IXON, etc.), `c_oflag` for output processing (OPOST, ONLCR, etc.), `c_cflag` for hardware control (CLOCAL, HUPCL, etc.), `c_lflag` for local modes (ICANON, ECHO, ISIG, etc.), and `c_cc` for special characters and MIN/TIME. The shell must not depend on the layout or size of the structure beyond these guaranteed members.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 56.
