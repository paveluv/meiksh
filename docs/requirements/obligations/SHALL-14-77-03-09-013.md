## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: TCOOFF

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCOOFF` for use with `tcflow()`. When used, output is suspended — the terminal driver stops sending output to the terminal device.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** TCOOFF programmatically suspends output, equivalent to the terminal receiving a STOP character. The shell could use this to temporarily halt output (e.g., during a redraw operation), though in practice output suspension is usually user-triggered via ^S.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 487.
