## Extracted Text

> If the file is a directory, it shall be an empty directory; otherwise, the file shall have length zero.

## Context

Duplicate extraction of SHALL-18-01-01-04-007: a newly created directory shall be empty; a newly created non-directory file shall have length zero. For `meiksh`, files created via output redirection start at zero length (the `O_TRUNC` flag handles this for existing files; new files start at zero). See POSIX Utilities Chapter 1, Section 1.1.1.4 item 5 (utilities/V3_chap01.html line 102).
