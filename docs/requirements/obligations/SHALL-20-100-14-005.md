## Extracted Text

> The following exit values shall be returned:: End-of-file was detected.

## Context

**Behavior required:** Exit status 1 indicates that end-of-file was detected before a terminating delimiter. This is the descriptive text for exit code 1.

**Condition:** EOF on stdin without a preceding delimiter.

**Implementation detail:** Combined with SHALL-20-100-14-004: return 1 on EOF. This is NOT an error — variables are still assigned. The distinction is important: scripts use `read`'s return value to detect EOF in loops.
