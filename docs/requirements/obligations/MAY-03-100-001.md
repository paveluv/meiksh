## Extracted Text

> The following standard utilities are declaration utilities: export, readonly, and, under certain conditions, command. An implementation may provide other declaration utilities.

## Context

**Definition — direct shell relevance.** Defines "declaration utility" as a special category of shell builtins. The standard declaration utilities are: `export`, `readonly`, and `command` (under certain conditions, specifically `command -v` and `command -V`). An implementation MAY provide additional declaration utilities.

**Shell impact:** Declaration utilities have special parsing behavior in the shell: variable assignments preceding a declaration utility remain in effect after the utility completes (unlike regular builtins/commands where they are temporary). This affects how the shell parser handles lines like `export FOO=bar` — the assignment persists. The shell must identify declaration utilities during command parsing to apply this special treatment. If `meiksh` adds extension declaration utilities (e.g., `local`, `typeset`, `declare`), they should follow the same rules.
