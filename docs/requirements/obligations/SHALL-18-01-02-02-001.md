## Extracted Text

> shall be implemented to return the results equivalent to those returned from a call to the corresponding function described in the ISO C standard.

## Context

Built-in mathematical functions used by utilities (primarily `awk` and `bc`) **shall** return results equivalent to calling the corresponding ISO C standard math library functions. For the shell itself (`meiksh`), this is relevant when implementing or invoking utilities that provide mathematical functions (e.g., `awk`'s `sin()`, `cos()`, `exp()`, `log()`, `sqrt()`, etc.). The results must match what the C `<math.h>` equivalents would produce, including edge cases like domain errors and range errors. Reference: XCU 1.1.2.2.
