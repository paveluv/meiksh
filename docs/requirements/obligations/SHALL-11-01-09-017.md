## Extracted Text

> Special character on both input and output, which is recognized if the IXON (output control) or IXOFF (input control) flag is set. Can be used to resume output that has been suspended by a STOP character. If IXON is set, the START character shall be discarded when processed.

## Context

**Behavior:** The START special character (typically ^Q) — duplicate/alternate extraction of the same spec text as SHALL-11-01-09-008:
1. Recognized when IXON or IXOFF is set in `c_iflag`.
2. Resumes output that was suspended by STOP (^S).
3. If IXON is set, the START character shall be **discarded** when processed.

**Conditions:** IXON and/or IXOFF must be set in `c_iflag`.

**Shell relevance:** See SHALL-11-01-09-008 for full details. May interfere with shell line editor keybindings.

**Spec reference:** POSIX.1-2024, §11.1.9 Special Characters (tag_11_01_09), `basedefs/V1_chap11.html` line 254.
