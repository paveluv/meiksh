## Extracted Text

> The following environment variables shall affect the execution of getopts:: NLSPATH

## Context

**Behavior:** `NLSPATH` shall affect `getopts` execution. This is an XSI extension that determines the location of message objects and message catalogs for internationalized messages.

**Applicability:** Only required for XSI-conforming implementations. Non-XSI implementations may ignore this variable.

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, line 151 of `utilities/getopts.html`.
