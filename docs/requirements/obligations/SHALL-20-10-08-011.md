## Extracted Text

> The following environment variables shall affect the execution of bg:: [XSI] Determine the location of messages objects and message catalogs.

## Context

(XSI) `NLSPATH` **shall** determine the location of message objects and message catalogs for `bg`. When XSI is supported, the shell uses `NLSPATH` to locate translated message catalogs. If not supporting XSI, this obligation may be deferred. Reference: bg utility ENVIRONMENT VARIABLES.
