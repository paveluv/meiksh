## Extracted Text

> The following environment variables shall affect the execution of wait:: LANG

## Context

**Behavior:** LANG shall provide a default value for the internationalization variables that are unset or null when executing the `wait` utility.

**Conditions:** LANG is the fallback locale, used only when the corresponding LC_* variable and LC_ALL are both unset or null.

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_147_08) in `utilities/wait.html`, line 93. See also XBD 8.2.
