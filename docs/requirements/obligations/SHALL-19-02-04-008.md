## Extracted Text

> A sequence of characters starting with a <dollar-sign> immediately followed by a single-quote ($') shall preserve the literal value of all characters up to an unescaped terminating single-quote ('), with the exception of certain <backslash>-escape sequences, as follows:: \f yields a <form-feed> character.

## Context

**Behavior:** Inside `$'...'`, the escape sequence `\f` yields the form-feed character.

**Applies to:** All shell modes during `$'...'` escape processing.

**Implementation detail:** Emit byte 0x0C (FF/form-feed). This is the same character that C's `'\f'` produces.

**Spec reference:** Section 2.2.4, lines 187-188 in `utilities/V3_chap02.html`.
