## Extracted Text

> The following environment variables shall affect the execution of command:: NLSPATH

## Context

**Behavior:** The `NLSPATH` environment variable must affect `command` execution. This is an XSI extension. NLSPATH determines the location of message catalogs for internationalized diagnostic and informative messages.

**Applies to:** XSI-conformant systems only.
