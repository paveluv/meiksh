## Extracted Text

> The following environment variables shall affect the execution of fg:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

This is the description of `LC_ALL`'s effect on `fg`. When `LC_ALL` is set to a non-empty string, it overrides `LC_CTYPE`, `LC_MESSAGES`, `LANG`, and all other internationalization variables. If empty or unset, individual variables apply per standard precedence. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
