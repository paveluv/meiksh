## Extracted Text

> The following environment variables shall affect the execution of unalias:: LANG

## Context

**Behavior:** LANG shall provide a default value for the internationalization variables (LC_*) that are unset or null when executing the `unalias` utility.

**Conditions:** LANG is the fallback locale. It is only used when the corresponding LC_* variable and LC_ALL are both unset or null.

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_133_08) in `utilities/unalias.html`, line 76. See also XBD 8.2.
