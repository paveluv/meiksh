## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `times` special builtin (section 2.28 in utilities/V3_chap02.html#tag_19_28) must use standard error **only** for diagnostic messages. The `times` builtin writes accumulated user and system times to **stdout**, not stderr. Stderr is reserved for errors only.

In practice, `times` rarely produces diagnostics since it takes no operands and simply reports process times.

See 2.28 times in utilities/V3_chap02.html#tag_19_28.
