## Extracted Text

> This variable shall determine the locale category for date and time formatting information. It affects the behavior of the time functions in strftime(). Additional semantics of this variable, if any, are implementation-defined.

## Context

**Behavior:** The environment variable `LC_TIME` shall determine the locale category for date and time formatting information, affecting functions like `strftime()`.

**Shell relevance:** The shell does not directly format dates, but built-in utilities and the `PS1`/`PS4` prompt expansion may indirectly depend on time formatting. The `time` reserved word output format is affected. The shell must propagate `LC_TIME` to child processes. Any additional semantics are implementation-defined.
