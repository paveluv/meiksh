## Extracted Text

> The following environment variables shall affect the execution of cd:: This variable shall be set as specified in the DESCRIPTION. If an application sets or unsets the value of PWD , the behavior of cd is unspecified.

## Context

**Behavior:** PWD semantics for cd:

1. **Shall be set:** cd must set PWD as described in the DESCRIPTION section (step 10). Specifically:
   - Without `-P`: PWD is set to the value of curpath as it was on entry to step 9 (the logical, canonicalized path).
   - With `-P`: PWD is set to the string that `pwd -P` would output (the physical, symlink-resolved path).
2. **Application modification:** If an application (user script) explicitly sets or unsets PWD, the behavior of cd becomes unspecified. The shell implementation may assume PWD accurately reflects the current directory and is not obligated to handle externally modified PWD values.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), lines 204-205.
