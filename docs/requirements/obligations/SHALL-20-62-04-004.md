## Extracted Text

> The following options shall be supported:: (The letter ell.) Provide more information about each job listed. See STDOUT for details.

## Context

**Behavior:** The `-l` option causes `jobs` to display extended information for each job:
- For job-control background jobs: adds the process group ID before the state field, and may show additional process lines with PID and command.
- For non-job-control background jobs (if supported): adds the process ID before the state field.

The full output format with `-l` is specified in the STDOUT section (SHALL-20-62-10-014 through SHALL-20-62-10-017).

**Spec reference:** POSIX.1-2024, `jobs` OPTIONS, line 55 of `utilities/jobs.html`.
