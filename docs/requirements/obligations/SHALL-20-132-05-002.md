## Extracted Text

> The following operand shall be supported:: mask

## Context

**Behavior:** The `umask` utility shall accept an optional `mask` operand specifying the new file mode creation mask. The operand can be specified in two forms:
1. **Octal integer form:** A string of octal digits (e.g., `022`, `0077`) specifying the bits to set in the mask.
2. **Symbolic mode form:** A symbolic mode string (same syntax as `chmod`'s mode operand) specifying permission changes (e.g., `a=rx,ug+w`, `g-w`, `-w`).

**Conditions:** When the mask operand is present, `umask` sets the mask and produces no stdout output. When absent, `umask` displays the current mask.
