## Extracted Text

> The following environment variables shall affect the execution of fg:: LC_ALL

## Context

The `fg` builtin must recognize the `LC_ALL` environment variable. When set to a non-empty string, `LC_ALL` overrides all other internationalization variables for locale determination within `fg`. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
