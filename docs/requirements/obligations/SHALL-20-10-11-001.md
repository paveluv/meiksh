## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `bg` builtin **shall** use stderr only for diagnostic messages. Normal output (the `[job-number] command` line) goes to stdout. Error messages (e.g., "no such job", "job control not enabled") go to stderr. Reference: bg utility STDERR.
