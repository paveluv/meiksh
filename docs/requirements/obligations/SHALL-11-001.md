## Extracted Text

> This chapter describes a general terminal interface that shall be provided. It shall be supported on any asynchronous communications ports if the implementation provides them. It is implementation-defined whether it supports network connections or synchronous ports, or both.

## Context

**Behavior:** The implementation shall provide a general terminal interface conforming to POSIX Chapter 11. This interface must be supported on asynchronous serial ports (if the implementation has them). Support for network connections and synchronous ports is implementation-defined.

**Conditions:** This is a top-level requirement that applies to the entire terminal subsystem. The shell relies on this interface for interactive terminal I/O, job control, and line editing.

**Shell relevance:** The shell depends on the terminal interface for reading input, job control (SIGTTIN/SIGTTOU/SIGTSTP), and the `stty` builtin. The shell must interact correctly with the termios API.
