## Extracted Text

> If PARENB is set, parity generation and detection shall be enabled and a parity bit is added to each byte. If parity is enabled, PARODD shall specify odd parity if set; otherwise, even parity shall be used.

## Context

**Behavior:** When PARENB is set in `c_cflag`, parity generation and detection shall be enabled and a parity bit is added to each byte. When parity is enabled, if PARODD is set, odd parity shall be used; otherwise, even parity shall be used.

**Relevance to meiksh:** The `stty` builtin must support setting/clearing PARENB and PARODD (via `parenb`, `-parenb`, `parodd`, `-parodd` operands). See also INPCK (§11.2.2) for the input parity checking counterpart.

**Spec reference:** POSIX.1-2024, §11.2.4 Control Modes (tag_11_02_04), `basedefs/V1_chap11.html` line 1071.
