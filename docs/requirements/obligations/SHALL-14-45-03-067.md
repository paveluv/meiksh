## Extracted Text

> The following shall be declared as functions and may also be defined as macros. Function prototypes shall be provided.

## Context

The `<signal.h>` header shall declare the following as functions (and may also define them as macros), with function prototypes provided: `kill()` [CX], `killpg()` [XSI], `psiginfo()` [CX], `psignal()` [CX], `pthread_kill()` [CX], `pthread_sigmask()` [CX], `raise()`, `sig2str()` [CX], `sigaction()` [CX], `sigaddset()` [CX], `sigaltstack()` [XSI], `sigdelset()` [CX], `sigemptyset()` [CX], `sigfillset()` [CX], `sigismember()` [CX], `signal()`, `sigpending()` [CX], `sigprocmask()` [CX], `sigqueue()` [CX], `sigsuspend()` [CX], `sigtimedwait()` [CX], `sigwait()` [CX], `sigwaitinfo()` [CX], and `str2sig()` [CX]. For `meiksh`, the shell directly uses `kill()`, `sigaction()`, `sigaddset()`, `sigdelset()`, `sigemptyset()`, `sigfillset()`, `sigismember()`, `sigprocmask()`, and `signal()` for signal management, trap handling, and job control. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 1123–1167).
