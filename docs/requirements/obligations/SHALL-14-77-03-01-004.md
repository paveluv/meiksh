## Extracted Text

> The <termios.h> header shall define the following data types through typedef:: speed_t

## Context

**Behavior:** `<termios.h>` **shall** define the type `speed_t` via typedef. This type is used for terminal baud rate values (see SHALL-14-77-03-01-005).

**Conditions:** Must be available after including `<termios.h>`. Shall be an unsigned integer type (see SHALL-14-77-03-01-008).

**Shell relevance:** Used with `cfgetispeed()`, `cfgetospeed()`, `cfsetispeed()`, `cfsetospeed()` functions. The `stty` builtin must be able to display and set baud rates using `speed_t` values and the Bxxx constants.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 46.
