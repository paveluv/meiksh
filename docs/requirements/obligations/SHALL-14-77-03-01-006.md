## Extracted Text

> The <termios.h> header shall define the following data types through typedef:: tcflag_t

## Context

**Behavior:** `<termios.h>` **shall** define the type `tcflag_t` via typedef. This type is used for terminal mode flags (`c_iflag`, `c_oflag`, `c_cflag`, `c_lflag` fields of the termios structure).

**Conditions:** Must be available after including `<termios.h>`. Shall be an unsigned integer type (see SHALL-14-77-03-01-008).

**Shell relevance:** meiksh uses `tcflag_t` for all terminal mode flag manipulation. The flag constants (ICANON, ECHO, ISIG, ICRNL, OPOST, etc.) are all of this type. Bitwise operations (|, &, ~) are used to set and clear flags.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 48.
