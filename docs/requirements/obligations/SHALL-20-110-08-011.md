## Extracted Text

> Determine the pathname of the user's home directory. The contents of HOME are used in tilde expansion as described in 2.6.1 Tilde Expansion.

## Context

**Behavior:** HOME determines the pathname of the user's home directory. Its contents are used in tilde expansion:
1. An unquoted `~` at the beginning of a word expands to the value of HOME.
2. `~username` expands to the home directory of the specified user (from the user database, not HOME).

**Applies to:** All shell modes (interactive and non-interactive).

**Details:** The shell uses `HOME` for:
- Tilde expansion (`~` → `$HOME`), see 2.6.1 Tilde Expansion in `utilities/V3_chap02.html#tag_19_06_01`.
- Other consequences may be specified elsewhere, but this row itself only states the home-directory and tilde-expansion behavior.

**Error handling:** If `HOME` is unset, bare `~` expansion is governed by the separate tilde-expansion rules rather than by any additional requirement in this row.
