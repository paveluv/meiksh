> The following environment variables shall affect the execution of sh:: MAILPATH

## Context

**Behavior:** The MAILPATH environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-027).

**Applies to:** Interactive shells (User Portability [UP] option).

**Details:** MAILPATH provides a colon-separated list of pathnames (with optional custom messages) for monitoring multiple mailbox files. When set, MAILPATH takes precedence over MAIL. See SHALL-20-110-08-027 for the full behavior specification.
