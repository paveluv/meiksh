## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

**Behavior:** The `unalias` utility shall write to stderr only for diagnostic messages. No regular output goes to stderr. The `unalias` utility produces no stdout output at all.

**Conditions:** Diagnostic messages include errors when a specified alias-name does not match any defined alias, or when an invalid option is given.
