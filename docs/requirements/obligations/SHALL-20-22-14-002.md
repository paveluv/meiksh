## Extracted Text

> When the -v or -V options are specified, the following exit values shall be returned:: 0

## Context

**Behavior:** Exit status 0 must be returned by `command -v` or `command -V` when the command_name was successfully found and identified. This is the exit value label "0" for the `-v`/`-V` success case.
