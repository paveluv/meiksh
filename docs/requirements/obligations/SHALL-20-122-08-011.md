> The following environment variables shall affect the execution of time:: Determine the locale for numeric formatting.

## Context

**Behavior:** `LC_NUMERIC` determines the locale for numeric formatting in `time` output. This affects the radix character (decimal point) used in the floating-point time values in the `-p` output format.
