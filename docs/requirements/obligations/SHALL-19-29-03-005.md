## Extracted Text

> If action is neither '-' nor the empty string, then each time a matching condition arises, the action shall be executed in a manner equivalent to:

## Context

**Behavior:** When a trap action is neither `-` nor empty string, each time the matching condition arises, the action must be executed in a manner equivalent to:

```
eval action
```

This means the action string is parsed and executed as shell commands in the current environment each time the signal is received.

**Applies to:** All shell modes.
