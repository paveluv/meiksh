## Extracted Text

> The <signal.h> header shall define the following macros which shall expand to integer constant expressions that need not be usable in #if preprocessing directives:: SIG_SETMASK

## Context

The `<signal.h>` header shall define the macro `SIG_SETMASK` as an integer constant expression. This macro is used as the `how` argument to `sigprocmask()` (and `pthread_sigmask()`). When `SIG_SETMASK` is specified, the resulting process signal mask is replaced entirely by the signal set pointed to by the `set` argument. The constant need not be usable in `#if` preprocessing directives. For `meiksh`, this is used when the shell needs to atomically set a complete signal mask, such as restoring a previously saved mask after `fork()` or when entering/leaving a subshell. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 508–511).
