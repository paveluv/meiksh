## Extracted Text

> The following environment variables shall affect the execution of umask:: Determine the locale for the interpretation of sequences of bytes of text data as characters (for example, single-byte as opposed to multi-byte characters in arguments).

## Context

**Behavior:** This is the description for LC_CTYPE's effect on `umask`. LC_CTYPE determines how byte sequences in `umask`'s arguments are interpreted as characters. For example, it controls whether bytes are treated as single-byte or multi-byte character encodings.

**Conditions:** Relevant when parsing the mask operand string, particularly for symbolic mode strings that contain alphabetic characters (r, w, x, u, g, o, a, s, t).
