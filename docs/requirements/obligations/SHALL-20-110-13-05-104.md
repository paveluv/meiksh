## Extracted Text

> The following commands shall be recognized in command mode:: Set the current command line to be the countth previous command line in the shell command history. If count is not specified, it shall default to 1. The cursor shall be positioned on the first character of the new command. If a k or - command would retreat past the maximum number of commands in effect for this shell (affected by the HISTSIZE environment variable), the terminal shall be alerted, and the command shall have no effect.

## Context

**Behavior:** In vi command mode, `k` (or `-`) sets the current command line to the count-th previous command in the shell command history (default count = 1). The cursor shall be positioned on the first character of the new command.

**Edge cases:** If `k`/`-` would retreat past the oldest command in history (limited by HISTSIZE), the terminal shall be alerted and the command has no effect.

**Implementation:** Navigate backward in the history list by `count` entries. If the target entry exists, display it as the current line with cursor at position 0. If it would go before the oldest entry, ring bell, no change.
