> The following environment variables shall affect the execution of command:: LANG

## Context

**Behavior:** The `LANG` environment variable must affect `command` execution as a fallback for internationalization variables that are unset or null. For `command`, this primarily affects diagnostic messages and the output format of `-v`/`-V`.
