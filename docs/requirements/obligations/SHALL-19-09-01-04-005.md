## Extracted Text

> If the command name contains at least one <slash>, the shell shall execute a non-built-in utility as described in 2.9.1.6 Non-built-in Utility Execution.

## Context

**Behavior:** Standalone restatement of SHALL-19-09-01-04-003. Commands with at least one `/` in the name MUST be executed as non-built-in utilities per 2.9.1.6, bypassing all builtin/function/PATH lookup.

**Applies to:** All commands with `/` in the name.

**Implementation detail:** See SHALL-19-09-01-04-003.

**Spec reference:** Section 2.9.1.4, lines 1932-1933 in `utilities/V3_chap02.html`. Cross-ref: 2.9.1.6.
