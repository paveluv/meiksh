## Extracted Text

> The command set -o vi shall enable vi-mode editing and place sh into vi insert mode (see Command Line Editing (vi-mode)). This command also shall disable any other editing mode that the implementation may provide. The command set +o vi disables vi-mode editing.

## Context

**Behavior:** The command `set -o vi` must:
1. Enable vi-mode editing.
2. Place the shell into vi **insert mode** (the initial editing state).
3. **Disable** any other editing mode that the implementation may provide (e.g., an emacs mode extension).

The command `set +o vi` disables vi-mode editing.

**Implementation:** When `set -o vi` is invoked, the shell must switch its line editing subsystem to vi-mode. If another editing mode was active, it must be deactivated. The initial state upon enabling is insert mode (characters typed are inserted into the edit buffer).

**Applies to:** Interactive shell.

**Spec reference:** Command Line Editing section in `utilities/sh.html`, lines 316-321. Cross-ref: `set` built-in, vi Line Editing section.
