## Extracted Text

> The following environment variables shall affect the execution of sh:: [XSI] Determine the location of messages objects and message catalogs.

## Context

**Behavior:** NLSPATH determines the location of message objects and message catalogs used by the shell for internationalized diagnostic messages. This is part of the XSI message catalog system (catopen/catgets).

**Applies to:** All shell modes (XSI option).

**Details:** NLSPATH is used by the native language support (NLS) system to locate message catalogs. The shell uses it when looking up locale-specific error and diagnostic messages. The format of NLSPATH is a colon-separated list of pathname templates with substitution fields (%L, %l, %t, %c, %N) as defined in the catopen() specification.

**Implementation note:** For a shell that uses hardcoded English messages (common for initial implementation), NLSPATH support can be deferred. Full POSIX compliance with XSI requires message catalog support.

**Feature gate:** This is an [XSI] (X/Open System Interfaces) option feature.
