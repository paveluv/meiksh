## Extracted Text

> If there is enough data available to satisfy the entire request, the read() shall complete successfully and shall return the number of bytes read.

## Context

**Behavior:** Duplicate of SHALL-11-01-05-002 (standalone extraction). When sufficient data is available for a non-blocking terminal read, read() succeeds and returns the byte count.

**Conditions:** O_NONBLOCK set, available data >= requested count.

**Shell relevance:** The shell handles this as a successful read operation.

**Spec reference:** XBD 11.1.5 "Input Processing and Reading Data" (tag_11_01_05), `basedefs/V1_chap11.html`, lines 138–139.
