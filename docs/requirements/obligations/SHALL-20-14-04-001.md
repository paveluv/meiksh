## Extracted Text

> The cd utility shall conform to XBD 12.2 Utility Syntax Guidelines.

## Context

The `cd` builtin **shall** conform to XBD 12.2 Utility Syntax Guidelines. This means: options must precede operands; options are single letters preceded by `-`; `--` terminates option processing; option-arguments follow their options. The shell's `cd` implementation must parse options according to these guidelines. Reference: cd utility OPTIONS, XBD 12.2.
