## Extracted Text

> The following codes are used in Actions when Creating a File that Already Exists:: The named file shall be opened with the consequences defined for that file type.

## Context

Action code "OF" (Open File): the named file shall be opened with the consequences defined for that file type. This applies when the existing file type is compatible and should simply be opened (e.g., opening a device file). For `meiksh`, when redirecting to a device file (like `/dev/null`, `/dev/tty`), the shell opens the existing file with the behavior defined for that file type rather than creating a new file. See POSIX Utilities Chapter 1, Section 1.1.1.4 action code OF (utilities/V3_chap01.html line 670).
