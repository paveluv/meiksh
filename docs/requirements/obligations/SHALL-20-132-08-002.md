## Extracted Text

> The following environment variables shall affect the execution of umask:: LANG

## Context

**Behavior:** The LANG environment variable shall provide a default value for the internationalization variables (LC_*) that are unset or null when executing the `umask` utility. LANG is the fallback locale setting.

**Conditions:** LANG is only used for locale categories whose corresponding LC_* variable is unset or null, and LC_ALL is also unset or null.

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_132_08) in `utilities/umask.html`, line 97. See also XBD 8.2.
