## Extracted Text

> The following environment variables shall affect the execution of fg:: Determine the locale for the interpretation of sequences of bytes of text data as characters (for example, single-byte as opposed to multi-byte characters in arguments).

## Context

This is the description of `LC_CTYPE`'s effect on `fg`. The locale determined by `LC_CTYPE` controls byte-to-character interpretation in `fg` arguments. This matters for `job_id` operands that might contain multi-byte characters (e.g., in `%?string` job search patterns). The implementation must handle character boundaries correctly according to the active locale. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
