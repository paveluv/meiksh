## Extracted Text

> If the file is a symbolic link, the effect shall be undefined unless the {POSIX2_SYMLINKS} variable is in effect for the directory in which the symbolic link would be created.

## Context

Duplicate extraction of SHALL-18-01-01-04-008: creating a symbolic link has undefined effect unless `{POSIX2_SYMLINKS}` is in effect. For `meiksh`, the shell does not directly create symlinks during normal operation. See POSIX Utilities Chapter 1, Section 1.1.1.4 item 6 (utilities/V3_chap01.html line 103).
