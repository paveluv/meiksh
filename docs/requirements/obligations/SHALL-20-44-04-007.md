## Extracted Text

> The following options shall be supported:: -n

## Context

The `fc` builtin must support the `-n` option, which suppresses command numbers when listing commands with `-l`. When `-n` is combined with `-l`, the output format changes from `"%d\t%s\n"` to `"\t%s\n"` — the command number is omitted but the leading tab is preserved. The `-n` option only has an effect when used with `-l`; it is meaningless without `-l`. Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
