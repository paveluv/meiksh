## Extracted Text

> The following operands shall be supported:: alias-name=string

## Context

The `alias` builtin **shall** support the `alias-name=string` operand form. When an operand contains `=`, the portion before `=` is the alias name and the portion after is the replacement string. This form creates a new alias or redefines an existing one. The alias name must be a valid alias name (typically a name that would be valid as a command word, not containing `/`). Reference: alias utility OPERANDS.
