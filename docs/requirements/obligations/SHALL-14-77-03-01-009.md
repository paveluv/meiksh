## Extracted Text

> The implementation shall support one or more programming environments in which the widths of cc_t, speed_t, and tcflag_t are no greater than the width of type long. The names of these programming environments can be obtained using the confstr() function or the getconf utility.

## Context

**Behavior:** The implementation **shall** support at least one programming environment where `cc_t`, `speed_t`, and `tcflag_t` are no wider than `long`. The names of qualifying programming environments can be discovered via `confstr()` or the `getconf` utility.

**Conditions:** At least one such programming environment must exist.

**Shell relevance:** meiksh compiles in a standard programming environment where these types fit within `long`. This constraint means the shell can safely store these values in `long` variables if needed (e.g., for generic value handling in the `stty` builtin). The `getconf` utility (if implemented as a builtin or external command) can report available programming environments.
