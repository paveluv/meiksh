## Extracted Text

> This variable shall represent the terminal type for which output is to be prepared. This information is used by utilities and application programs wishing to exploit special capabilities specific to a terminal. The format and allowable values of this environment variable are unspecified.

## Context

**Behavior:** The `TERM` environment variable shall represent the terminal type for which output is to be prepared. Utilities and applications use it to exploit terminal-specific capabilities. The format and allowable values are unspecified.

**Shell relevance:** The shell uses `TERM` for interactive line editing — the terminal capabilities (via terminfo/termcap) control cursor movement, screen clearing, and other escape sequences used by the line editor. The shell must propagate `TERM` to child processes. When `COLUMNS` or `LINES` is set, any width/height implied by `TERM` is overridden.
