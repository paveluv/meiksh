## Extracted Text

> The concatenation of one or more patterns matching a single character with one or more <asterisk> characters is a valid pattern. In such patterns, each <asterisk> shall match a string of zero or more characters, matching the greatest possible number of characters that still allows the remainder of the pattern to match the string.

## Context

**Duplicate of SHALL-19-14-02-003: `*` uses greedy matching for greatest possible number of characters.**

Same obligation as SHALL-19-14-02-003. When `*` is concatenated with single-character patterns, each `*` **shall** match the greatest possible number of characters while still allowing the rest of the pattern to match. See SHALL-19-14-02-003 for full details.
