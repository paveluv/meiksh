## Extracted Text

> When job control is enabled, the shell shall create one or more jobs when it executes a list (see 2.9.3 Lists) that has one of the following forms:: One or more sequentially executed AND-OR lists followed by at most one asynchronous AND-OR list

## Context

**Job creation form 2:** A list consisting of one or more sequentially executed AND-OR lists (`;`-separated) optionally followed by one asynchronous AND-OR list (`&`-terminated). For example: `cmd1; cmd2; cmd3 &`. The sequential parts form a foreground job; the trailing async part (if any) forms a background job. See SHALL-19-11-012 for the detailed semantics of this form.
