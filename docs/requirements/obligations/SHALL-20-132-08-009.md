## Extracted Text

> The following environment variables shall affect the execution of umask:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

**Behavior:** This is the description for LC_MESSAGES' effect on `umask`. LC_MESSAGES determines the locale for formatting diagnostic messages written to stderr (e.g., error messages for invalid mask operands, unrecognized options).

**Conditions:** Applicable whenever `umask` needs to write a diagnostic message to stderr.
