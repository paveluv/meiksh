## Extracted Text

> The following commands shall be recognized in command mode:: Perform pathname expansion on the current bigword and insert all expansions into the command to replace the current bigword, with each expansion separated by a single <space>. If at the end of the line, the current cursor position shall be moved to the first column position following the expansions and sh shall be placed in insert mode. Otherwise, the current cursor position shall be the last column position of the first character after the expansions and sh shall be placed in insert mode. If the current bigword contains none of the characters '?', '*', or '[', before the operation, an <asterisk> ('*') shall be implicitly assumed at the end.

## Context

**Behavior:** In vi command mode, `*` shall perform pathname expansion on the current bigword and insert ALL expansions into the command line to replace the bigword, separated by single spaces.

**Details:**
- If the bigword contains none of `?`, `*`, or `[`, an `*` is implicitly appended before expansion.
- If at the end of the line: cursor moves to the first column position following the expansions, shell enters insert mode.
- If not at end of line: cursor is placed on the last character of the first character after the expansions, shell enters insert mode.

**Implementation:** Extract bigword, perform glob, join all results with spaces, replace bigword in buffer, position cursor appropriately, switch to insert mode.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 429–434.
