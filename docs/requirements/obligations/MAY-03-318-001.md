## Extracted Text

> A thread currently executing on a processor. On multi-processor systems there may be more than one such thread in a system at a time.

## Context

**Definition only — no shell relevance.** Defines "running process (or thread)" as one currently executing on a processor. Multi-processor systems may have multiple running processes simultaneously.

**Shell impact:** None directly. The shell manages processes via fork/exec/wait but does not concern itself with CPU scheduling states. No action required.
