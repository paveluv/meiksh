## Extracted Text

> Move to the end of the current word. If at the end of a word, move to the end of the next word. If the cursor was positioned on the last character of the line, the terminal shall be alerted and the cursor shall not be advanced. If the count is larger than the number of words after the cursor, this shall not be considered an error; the cursor shall advance to the last character on the line.

## Context

**Behavior:** The `e` command shall move to end of current word. If at end of word, move to end of next word. At last char: alert. Count exceeding remaining: not an error, go to last char.

**Details:** Same normative content as SHALL-20-110-13-05-035. This is a motion command. See `utilities/vi.html` for word definition.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 478–481.
