## Extracted Text

> If the S_ISVTX bit is set on a non-directory file, the behavior is unspecified.

## Context

**Definition — low shell relevance.** If S_ISVTX (sticky bit) is set on a non-directory file, behavior is unspecified. Historically this was used for executable caching; modern systems ignore it on regular files.

**Shell impact:** None. The shell does not need to check or handle the sticky bit on non-directory files. No action required.
