## Extracted Text

> AND-OR lists that are separated by a <semicolon> (';') shall be executed sequentially. The format for executing AND-OR lists sequentially shall be:

## Context

**Behavior:** AND-OR lists separated by `;` MUST be executed sequentially. The format is: `sequential_sep : ';' linebreak | newline_list`. This means the shell waits for the first AND-OR list to complete before starting the next.

**Applies to:** All `;`-separated command lists. Example: `cmd1; cmd2; cmd3` runs each in order, waiting for completion.

**Implementation detail:** After executing each AND-OR list, wait for it to complete (collect exit status), then proceed to the next. This is the default behavior for most command lists in scripts.
