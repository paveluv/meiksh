## Extracted Text

> Move to the start of the next bigword. If the cursor was positioned on the last character of the line, the terminal shall be alerted and the cursor shall not be advanced. If the count is larger than the number of bigwords after the cursor, this shall not be considered an error; the cursor shall advance to the last character on the line.

## Context

**Behavior:** In vi command mode, `W` moves the cursor to the start of the next bigword. A "bigword" is any sequence of non-blank characters, separated by blanks (simpler than vi words).

**Edge cases:**
- If cursor is on the last character: alert the terminal (bell), do NOT advance.
- If count exceeds the number of bigwords after cursor: NOT an error; cursor advances to the last character on the line.

**Implementation:** Move forward `count` bigwords (skip non-blanks, skip blanks, land on first non-blank). Clamp at line end. This is a motion command.
