## Extracted Text

> If its process group is the foreground process group associated with the terminal, the shell shall set its process group ID to its process ID (if they are not already equal) and set the foreground process group ID associated with the terminal to its process group ID.

## Context

**Job control initialization — set process group and claim terminal (detail):** This is the specific step when the shell's process group IS the foreground group: the shell shall call `setpgid(0, getpid())` to make its process group ID equal to its PID (if not already equal), then call `tcsetpgrp()` to set the terminal's foreground process group to its process group.

This is a detail obligation extracted from the same source line as SHALL-19-11-002. The `setpgid` + `tcsetpgrp` sequence ensures the shell is both a process group leader and the terminal's foreground group.
