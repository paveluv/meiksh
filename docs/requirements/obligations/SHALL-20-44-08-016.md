## Extracted Text

> The following environment variables shall affect the execution of fc:: NLSPATH

## Context

The `fc` builtin must recognize the `NLSPATH` environment variable on XSI-conformant systems. `NLSPATH` determines the location of message objects and message catalogs used for internationalized diagnostic messages. See SHALL-20-44-08-017 for the full description. This is an [XSI] extension. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
