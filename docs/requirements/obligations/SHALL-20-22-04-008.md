## Extracted Text

> The following options shall be supported:: Write a string to standard output that indicates how the name given in the command_name operand will be interpreted by the shell, in the current shell execution environment (see 2.13 Shell Execution Environment), but do not invoke command_name. Although the format of this string is unspecified, it shall indicate in which of the following categories command_name falls and shall include the information stated: Executable utilities, regular built-in utilities, and any implementation-provided functions that are found using the PATH variable (as described in 2.9.1.4 Command Search and Execution), shall be identified as such and include the absolute pathname in the string. Other shell functions shall be identified as functions. Aliases shall be identified as aliases and their definitions included in the string. Special built-in utilities shall be identified as special built-in utilities. Regular built-in utilities not associated with a PATH search shall be identified as regular built-in utilities. (The term "regular" need not be used.) Shell reserved words shall be identified as reserved words.

## Context

**Behavior:** Full specification of the `-V` option output. `command -V name` must write a human-readable description to stdout. The exact format is unspecified, but it must:

1. **Identify the category** of the command_name.
2. **Include specific information** per category:
   - **Executable utilities / regular built-ins found via PATH / implementation functions via PATH:** Identify as such AND include the absolute pathname. Example: `ls is /usr/bin/ls`
   - **Shell functions:** Identify as a function. Example: `myfunc is a function`
   - **Aliases:** Identify as an alias AND include the alias definition. Example: `ll is aliased to 'ls -la'`
   - **Special built-in utilities:** Identify as special built-in. Example: `export is a special shell builtin`
   - **Regular built-in utilities (not PATH-associated):** Identify as a built-in (the term "regular" need not appear). Example: `echo is a shell builtin`
   - **Shell reserved words:** Identify as reserved words. Example: `if is a reserved word`

**Output format:** `"%s\n", <unspecified>` — the format is implementation-defined but must convey the category and required info.
