## Extracted Text

> The following environment variables shall affect the execution of cd:: [XSI] Determine the location of messages objects and message catalogs.

## Context

**Behavior:** This is the description of NLSPATH's role for cd (XSI extension). NLSPATH determines the location of message objects and message catalogs used for internationalized diagnostic messages. On XSI-conformant systems, cd must use NLSPATH to locate translated message catalogs.

**Applies to:** XSI-conformant systems. Message catalog lookup during cd diagnostic generation.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), lines 197-198.
