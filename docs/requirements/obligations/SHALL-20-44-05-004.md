## Extracted Text

> The following operands shall be supported:: old=new

## Context

The `fc` builtin must support the `old=new` operand when used with the `-s` option. This operand specifies a string substitution to perform on the command before re-execution: the first occurrence of string `old` in the selected command text is replaced with string `new`. The substitution is a simple string replacement (not a regex). If `old=new` is not specified with `-s`, the command is re-executed as-is. The `old=new` operand is only valid with `-s`. Reference: POSIX.1-2024 §fc, OPERANDS (tag_20_44_05).
