## Extracted Text

> The name of the coded character set for which the character set description file is defined. The characters of the name shall be taken from the set of characters with visible glyphs defined in Portable Character Set.

## Context

**Behavior:** The `<code_set_name>` declaration in a charmap file specifies the name of the coded character set. The name must consist only of characters with visible glyphs from the Portable Character Set (ASCII letters, digits, and printable special characters).

**Shell relevance:** The shell does not parse charmap files. This constrains the code set name to portable characters, which the shell may encounter via `locale` output or `LANG`/`LC_*` environment variable values.
