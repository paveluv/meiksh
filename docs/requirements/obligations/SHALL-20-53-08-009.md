## Extracted Text

> The following environment variables shall affect the execution of getopts:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

**Behavior:** Describes the effect of `LC_MESSAGES`: determines the locale for diagnostic messages written to stderr by `getopts`. These diagnostics are produced when an unknown option or missing argument is detected in verbose mode (no leading `:` in `optstring`).

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, line 149 of `utilities/getopts.html`.
