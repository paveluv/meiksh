## Extracted Text

> If the utility utility is invoked, the exit status of time shall be the exit status of utility; otherwise, the time utility shall exit with one of the following values:: 1-125

## Context

**Behavior:** When the utility is not successfully invoked, `time` shall exit with status 1-125 if the error is in the `time` utility itself (e.g., invalid options). Exit 126 if the named utility was found but could not be invoked (e.g., permission denied). Exit 127 if the named utility could not be found in `PATH`.

**Implementation:** Use exit 127 when all `exec` attempts fail with `ENOENT`. Use exit 126 when any `exec` attempt fails for other reasons (e.g., `EACCES`, `ENOEXEC`).
