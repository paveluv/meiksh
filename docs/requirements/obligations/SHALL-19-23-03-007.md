## Extracted Text

> The shell shall format the output, including the proper use of quoting, so that it is suitable for reinput to the shell as commands that achieve the same exporting results, except:: Variables that were unset at the time they were output need not be reset to the unset state if a value is assigned to the variable between the time the state was saved and the time at which the saved output is reinput to the shell.

## Context

**Behavior:** Exception to the reinput guarantee for `export -p`: variables that were unset at the time they were output need not be reset to the unset state if a value is assigned to the variable between the time the state was saved and the time the saved output is reinput.

**Applies to:** All shell modes.
