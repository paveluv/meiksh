## Extracted Text

> The following environment variables shall affect the execution of cd:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

**Behavior:** This is the description of LC_MESSAGES' role for cd. LC_MESSAGES determines the locale used for formatting diagnostic messages that cd writes to standard error. This includes error messages like "cd: /nonexistent: No such file or directory". The shell must use the LC_MESSAGES locale (or its fallback) when producing these messages.

**Applies to:** All diagnostic messages from the cd built-in written to stderr.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 194.
