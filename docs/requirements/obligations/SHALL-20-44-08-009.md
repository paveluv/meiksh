## Extracted Text

> The following environment variables shall affect the execution of fc:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

This is the description of `LANG`'s effect on `fc`. When any of the `LC_*` internationalization variables are unset or null, `LANG` provides the default locale value. The precedence chain is: `LC_ALL` (overrides everything) > specific `LC_*` variable > `LANG` > implementation default (POSIX locale). This affects character interpretation (`LC_CTYPE`) and diagnostic message formatting (`LC_MESSAGES`) within the `fc` builtin. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08); XBD 8.2.
