## Extracted Text

> The following environment variables shall affect the execution of kill:: LC_CTYPE

## Context

**Behavior:** The `LC_CTYPE` environment variable must be recognized by the `kill` built-in. It determines the locale for interpretation of byte sequences as characters in arguments (e.g., single-byte vs. multi-byte). See SHALL-20-64-08-007.
