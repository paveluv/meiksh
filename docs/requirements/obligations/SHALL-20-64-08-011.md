## Extracted Text

> The following environment variables shall affect the execution of kill:: [XSI] Determine the location of messages objects and message catalogs.

## Context

**Behavior:** `NLSPATH` (XSI extension) determines the location of message objects and message catalogs for the `kill` built-in. This is used in conjunction with `LC_MESSAGES` to find locale-specific diagnostic message files.

**Conditions:** Only applicable on systems supporting the XSI option. On non-XSI systems, this variable may be ignored.

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_64_08) in `utilities/kill.html`, lines 177–178. XSI extension.
