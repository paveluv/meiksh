## Extracted Text

> Implementations may provide additional or alternate file access control mechanisms, or both. An additional access control mechanism shall only further restrict the access permissions defined by the file permission bits. An alternate file access control mechanism shall:: Be enabled only by explicit user action, on a per-file basis by the file owner or a user with appropriate privileges.
