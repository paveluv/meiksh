## Extracted Text

> The following commands shall be recognized in command mode:: Enter insert mode at the beginning of the current command line.

## Context

**Behavior:** In vi command mode, `I` enters insert mode at the beginning of the current command line. Equivalent to `0i`.

**Implementation:** Move cursor to position 0, then enter insert mode.
