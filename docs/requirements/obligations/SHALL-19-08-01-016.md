## Extracted Text

> Non-Interactive Shell: Variable assignment error -- shall exit

## Context

A non-interactive shell SHALL exit when a variable assignment error occurs. See 2.8.1 Consequences of Shell Errors in utilities/V3_chap02.html.

**Behavior required:**
- When the shell is non-interactive and a variable assignment error occurs, the shell SHALL write a diagnostic to stderr and exit.
- A shell diagnostic message IS required.

**What constitutes a variable assignment error:** Attempting to assign to a readonly variable (e.g., `readonly x=1; x=2`), or an assignment that fails due to an expansion error in the value.

**Exit status:** Non-zero (typically 1 or 2).
