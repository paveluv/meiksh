## Extracted Text

> where the optional n represents the file descriptor number. If the number is omitted, the redirection shall refer to standard input (file descriptor 0).

## Context

Specifies the default file descriptor for input redirection when `n` is omitted. See 2.7.1 Redirecting Input in utilities/V3_chap02.html.

**Behavior required:**
- If the fd number `n` is omitted from `[n]<word`, the redirection defaults to standard input (file descriptor 0).
- Example: `<file` is equivalent to `0<file`.

This is the counterpart to output redirection defaulting to fd 1 (see SHALL-19-07-02-001).
