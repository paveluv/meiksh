## Extracted Text

> When the mask operand is not specified, the umask utility shall write a message to standard output that can later be used as a umask mask operand.

## Context

**Behavior:** When `umask` is invoked in query mode (no mask operand), it shall write a message to stdout that can later be used as a `umask` `mask` operand.

**Output format:** The format depends on whether `-S` is specified. Without `-S`, the format is implementation-defined (typically octal, e.g., `0022`). With `-S`, symbolic format is used.

**Round-trip requirement:** The key requirement is that the output can be fed back to `umask` as an operand. This means the output must be a syntactically valid mask string.
