## Extracted Text

> The following environment variables shall affect the execution of fc:: Determine the locale for the interpretation of sequences of bytes of text data as characters (for example, single-byte as opposed to multi-byte characters in arguments and input files).

## Context

This is the description of `LC_CTYPE`'s effect on `fc`. The locale determined by `LC_CTYPE` controls how the `fc` builtin interprets byte sequences as characters — this is relevant when processing command text that may contain multi-byte characters (e.g., UTF-8), both in arguments passed to `fc` and in the temporary file used for editing. The implementation must handle multi-byte character boundaries correctly when performing string operations (e.g., `old=new` substitution in `-s` mode). Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
