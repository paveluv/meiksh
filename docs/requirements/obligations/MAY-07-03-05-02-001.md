> The alternative symbols for digits, corresponding to the %O conversion specification modifier. The value consists of <semicolon>-separated symbols. ## Context

The ALT_DIGITS `nl_langinfo()` constant provides alternative symbols for digits used with the `%O` conversion specification modifier. The value consists of semicolon-separated symbols. The first symbol corresponds to zero, the second to one, and so on. Up to 100 alternative symbols may be specified. This is an optional capability.

**Shell relevance:** If the shell supports `%O` modified time conversions, it should use these alternative digit symbols. See XBD 7.3.5.2.
