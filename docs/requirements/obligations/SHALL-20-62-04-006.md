## Extracted Text

> The following options shall be supported:: Display only the process IDs for the process group leaders of job-control background jobs and the process IDs associated with non-job-control background jobs (if supported).

## Context

**Behavior:** The `-p` option causes `jobs` to output only process IDs:
- For job-control background jobs: display the process group leader's PID.
- For non-job-control background jobs (if the shell supports them without job control): display the associated PID.

**Output format:** One PID per line: `"%d\n", <process ID>` (see SHALL-20-62-10-001).

**Use case:** `$(jobs -p)` is the portable way to get the process group ID of background jobs.

**Spec reference:** POSIX.1-2024, `jobs` OPTIONS, line 57 of `utilities/jobs.html`.
