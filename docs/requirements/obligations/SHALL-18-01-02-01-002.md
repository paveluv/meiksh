## Extracted Text

> Arithmetic operators and control flow keywords shall be implemented as equivalent to those in the cited ISO C standard section, as listed in Selected ISO C Standard Operators and Control Flow Keywords.

## Context

Arithmetic operators and control flow keywords in shell arithmetic contexts **shall** behave equivalently to their ISO C counterparts as listed in XCU Table "Selected ISO C Standard Operators and Control Flow Keywords." For the shell's `$(( ))` arithmetic expansion, this includes: unary `+`, `-`, `~`, `!`; binary `*`, `/`, `%`, `+`, `-`, `<<`, `>>`, `<`, `<=`, `>`, `>=`, `==`, `!=`, `&`, `^`, `|`, `&&`, `||`; ternary `?:`; and assignment operators `=`, `*=`, `/=`, `%=`, `+=`, `-=`, `<<=`, `>>=`, `&=`, `^=`, `|=`. The comma operator is also supported. Operator precedence and associativity must match C. Reference: XCU 1.1.2.1.
