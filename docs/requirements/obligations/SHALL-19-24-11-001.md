## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `readonly` special builtin (section 2.24 in utilities/V3_chap02.html#tag_19_24) must use standard error **only** for diagnostic messages. Note that `readonly` does write to stdout when invoked without arguments or with `-p` (listing readonly variables), but stderr must be reserved for diagnostics.

Diagnostic messages that may be written to stderr include:
- Invalid variable name
- Attempting to set a variable that is already readonly

See 2.24 readonly in utilities/V3_chap02.html#tag_19_24.
