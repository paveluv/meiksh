## Extracted Text

> In the context of providing natural language messages to the user, a per-process unique value used to identify an open message catalog. A message catalog descriptor may be implemented using a file descriptor.

## Context

**Definition only — no shell relevance.** Defines "message catalog descriptor" as a per-process unique value identifying an open message catalog. It may be implemented using a file descriptor.

**Shell impact:** None. Message catalogs are used by internationalized programs via `catopen()`/`catgets()`. The shell does not directly manage message catalog descriptors. If `meiksh` uses `gettext()` for its own i18n, that is an internal detail. No action required.
