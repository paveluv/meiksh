## Extracted Text

> The following environment variables shall affect the execution of fc:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

This is the description of `LC_ALL`'s effect on `fc`. When `LC_ALL` is set to a non-empty string, it takes precedence over all other internationalization variables for locale category determination. This means `LC_ALL` overrides `LC_CTYPE`, `LC_MESSAGES`, `LANG`, etc. If `LC_ALL` is empty or unset, the individual `LC_*` variables and `LANG` determine locale categories according to the standard precedence (XBD 8.2). Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
