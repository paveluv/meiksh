## Extracted Text

> The following options shall be supported:: -l

## Context

The `fc` builtin must support the `-l` option (the letter ell). When `-l` is specified, `fc` operates in list mode: it lists commands from the history rather than opening them in an editor. Commands are displayed in the sequence indicated by the `first` and `last` operands, as affected by the `-r` option, with each command preceded by its command number (unless `-n` is also specified). The output format is `"%d\t%s\n"` with line number and command text. Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
