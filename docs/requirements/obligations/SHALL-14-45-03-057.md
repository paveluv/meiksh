## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: [XSI] Minimum stack size for a signal handler.

## Context

This is the [XSI] extension defining the semantics of `MINSIGSTKSZ`: it specifies the minimum stack size in bytes for a signal handler using an alternate signal stack. Any `stack_t.ss_size` passed to `sigaltstack()` must be at least this value. For `meiksh`, this is not directly needed. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 550–551).
