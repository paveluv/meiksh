## Extracted Text

> If an operation A that modifies an atomic object M happens before an operation B that modifies M, then A shall be earlier than B in the modification order of M. (This is known as "write-write coherence".)

## Context

This is a C-level memory model requirement about write-write coherence for atomic objects. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.1.1 Data Races.
