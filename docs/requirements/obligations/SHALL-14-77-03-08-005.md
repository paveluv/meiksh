## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcsetattr():: Change attributes when output has drained.

## Context

**Behavior:** This is the description of TCSADRAIN: **change attributes when output has drained**. When `tcsetattr(fd, TCSADRAIN, &termios)` is called, the function waits until all output written to `fd` has been transmitted before applying the attribute changes. Pending input is not affected.

**Conditions:** Used as the `optional_actions` parameter to `tcsetattr()`.

**Shell relevance:** See SHALL-14-77-03-08-004. This ensures that output like prompts or command results is fully displayed before terminal mode changes take effect.

**Spec reference:** POSIX.1-2024, `<termios.h>` Attribute Selection (tag_14_77_03_08), `basedefs/termios.h.html` line 463.
