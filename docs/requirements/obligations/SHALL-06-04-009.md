## Extracted Text

> The character that, when placed in column 1 of a charmap line, is used to indicate that the line shall be ignored. The default character shall be the <number-sign> ('#').

## Context

**Behavior:** The `<comment_char>` declaration in a charmap file defines the comment character. It defaults to `#`. Lines beginning with this character in column 1 are ignored.

**Shell relevance:** The shell does not parse charmap files. This is a charmap file format detail only. No shell implementation action needed.

**Spec reference:** Section 6.4, lines 1903-1904 in `basedefs/V1_chap06.html`.
