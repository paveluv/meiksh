## Context

**Behavior:** BRE operator precedence (highest to lowest): (1) Collating-related bracket symbols `[==]`, `[::]`, `[..]`; (2) Escaped characters `\<special>`; (3) Bracket expression `[]`; (4) Subexpressions/back-references `\(\)`, `\n`; (5) Single-character BRE duplication `*`, `\{m,n\}`; (6) Concatenation; (7) Anchoring `^`, `$`.

**Shell relevance:** Shell pattern matching has a simpler precedence since it lacks subexpressions, back-references, and repetition quantifiers. Bracket expression components (collating symbols, equivalence classes, character classes) have the highest precedence within bracket expressions, which is directly relevant to shell pattern matching.
