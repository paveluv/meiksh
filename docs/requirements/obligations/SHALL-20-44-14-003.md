## Extracted Text

> The following exit values shall be returned:: Successful completion of the listing.

## Context

This is the description for exit status 0 of `fc`: "Successful completion of the listing." This applies specifically to `fc -l` mode — when the listing of history commands completes without error, `fc` must exit with status 0. Note that this description explicitly says "of the listing," confirming that exit 0 is for the `-l` form; the edit/re-execute forms have different exit status semantics (see SHALL-20-44-14-006). Reference: POSIX.1-2024 §fc, EXIT STATUS (tag_20_44_14).
