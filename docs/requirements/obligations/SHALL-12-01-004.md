## Extracted Text

> Arguments or option-arguments enclosed in the '[' and ']' notation are optional and can be omitted. Conforming applications shall not include the '[' and ']' symbols in data submitted to the utility.

## Context

This is a documentation convention: square brackets in SYNOPSIS notation indicate optional arguments. Users must not include literal `[` and `]` as part of the argument syntax. This is a constraint on conforming applications rather than on the shell implementation. The shell does not need to strip brackets from arguments—they are a documentation convention only.
