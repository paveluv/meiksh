## Extracted Text

> Otherwise, the following exit values shall be returned:

## Context

**Behavior:** When `command` is used in execution mode (without `-v` or `-V`), a different set of exit values applies. The defined exit values for execution mode are: 126 (found but not invokable), 127 (not found or error in command utility), or the exit status of the executed simple command.

**Applies to:** `command [-p] command_name [argument ...]` invocations without `-v`/`-V`.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section EXIT STATUS (tag_20_22_14), line 202.
