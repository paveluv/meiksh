## Extracted Text

> Define the full weekday names, corresponding to the %A conversion specification. The operand shall consist of seven <semicolon>-separated strings, each surrounded by double-quotes. The first string is the full name of the day corresponding to Sunday, the second the full name of the day corresponding to Monday, and so on.

## Context

The `day` operand for full weekday names shall consist of exactly seven semicolon-separated, double-quoted strings. The first string is the full name for Sunday, the second for Monday, through Saturday. These correspond to the `%A` conversion specification.

**Shell relevance:** Used by `date +%A`. See XBD 7.3.5.1.
