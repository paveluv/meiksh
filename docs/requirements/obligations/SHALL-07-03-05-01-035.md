## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: Define alternative symbols for digits, corresponding to the %O modified conversion specification. The operand shall consist of <semicolon>-separated strings, each surrounded by double-quotes. The first string shall be the alternative symbol corresponding with zero, the second string the symbol corresponding with one, and so on. Up to 100 alternative symbol strings can be specified. The %O modifier shall indicate that the string corresponding to the value specified via the conversion specification shall be used instead of the value.

## Context

The `alt_digits` keyword defines alternative symbols for digits corresponding to the `%O` modified conversion specification. The operand shall consist of semicolon-separated strings, each in double-quotes. The first string is the symbol for zero, the second for one, and so on. Up to 100 alternative symbol strings can be specified. When the `%O` modifier is used, the string corresponding to the numeric value shall be substituted instead of the numeric value.

**Constraints:** Up to 100 strings. Semicolon-separated, double-quoted.

**Shell relevance:** Affects `date` with `%O` modifier. See XBD 7.3.5.1.
