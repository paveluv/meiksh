## Extracted Text

> When the -v or -V options are specified, the following exit values shall be returned:: Successful completion.

## Context

**Behavior:** Exit status 0 means successful completion for `command -v`/`-V`. The command_name was found in the current shell execution environment (as an alias, function, built-in, reserved word, or executable in PATH) and the appropriate information was written to stdout.
