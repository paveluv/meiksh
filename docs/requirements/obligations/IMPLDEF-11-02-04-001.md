## Extracted Text

> The initial hardware control value after open() is implementation-defined.

## Context

**Behavior:** The initial value of `c_cflag` (hardware control) after `open()` on a terminal device is implementation-defined. POSIX does not mandate default values for CSIZE, CSTOPB, CREAD, PARENB, PARODD, HUPCL, CLOCAL, or baud rate settings.

**Relevance to meiksh:** meiksh must document its chosen initial hardware control values. The shell should explicitly configure control modes it requires when setting up an interactive terminal, rather than relying on any particular default.

**Spec reference:** POSIX.1-2024, §11.2.4 Control Modes (tag_11_02_04), `basedefs/V1_chap11.html` line 1084.
