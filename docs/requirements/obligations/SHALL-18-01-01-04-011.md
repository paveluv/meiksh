## Extracted Text

> The group ID of the file shall be set to the effective group ID of the calling process or the group ID of the directory in which the file is being created.

## Context

Duplicate extraction of SHALL-18-01-01-04-004: when a new file is created, the group ID shall be set to the effective group ID of the calling process or the group ID of the parent directory (implementation-defined). For `meiksh`, this is handled by the OS kernel. See POSIX Utilities Chapter 1, Section 1.1.1.4 item 2 (utilities/V3_chap01.html line 90).
