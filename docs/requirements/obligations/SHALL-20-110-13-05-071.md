## Extracted Text

> The following commands shall be recognized in command mode:: Enter insert mode, replacing characters from the command line beginning at the current cursor position.

## Context

**Behavior:** In vi command mode, `R` enters insert mode in replace/overwrite style, beginning at the current cursor position. Each character typed replaces the character under the cursor rather than shifting text rightward.

**Implementation:** Enter replace mode (a variant of insert mode) at the current cursor position. Characters overwrite existing content until ESC returns to command mode.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 543–544.
