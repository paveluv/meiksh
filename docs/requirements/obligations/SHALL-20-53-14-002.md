## Extracted Text

> The following exit values shall be returned:: 0

## Context

**Behavior:** `getopts` shall return exit status 0 when an option is found. This includes both valid options from `optstring` and error conditions (unknown option, missing argument) — these are application errors, not getopts errors, so exit status remains 0.

**Spec reference:** POSIX.1-2024, `getopts` EXIT STATUS, line 204 of `utilities/getopts.html`.
