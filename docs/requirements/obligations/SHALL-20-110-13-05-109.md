## Extracted Text

> The following commands shall be recognized in command mode:: Set the current command line to be the oldest command line stored in the shell command history. With a number number, set the current command line to be the command line number in the history. If command line number does not exist, the terminal shall be alerted and the command line shall not be changed.

## Context

**Behavior:** In vi command mode, `G` (without a number) sets the current command line to the oldest command in the shell command history. With a number, sets the current command line to command number `number` in the history.

**Edge cases:** If command number `number` does not exist in history, the terminal shall be alerted and the command line shall not be changed.

**Implementation:** Without number: jump to history entry 1 (oldest). With number: look up history entry by its sequential number. If found, display it. If not found, ring bell.
