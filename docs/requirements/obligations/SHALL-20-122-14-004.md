## Extracted Text

> If the utility utility is invoked, the exit status of time shall be the exit status of utility; otherwise, the time utility shall exit with one of the following values:: 126

## Context

**Behavior:** When the `time` utility cannot successfully invoke the specified utility, `time` shall exit with status 126 if the utility was found in PATH (or by the specified path) but could not be invoked (e.g., the file exists but is not executable, or exec failed for a reason other than ENOENT).

**Conditions:** The utility file was located but execution failed. This typically happens when the file lacks execute permission, is not a valid executable format, or the system otherwise refuses to execute it.

**Distinction from 127:** Exit code 126 means "found but not invocable" while 127 means "not found at all." This follows KornShell convention where ENOENT produces 127 and all other exec failures produce 126.
