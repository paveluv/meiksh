## Extracted Text

> The following environment variables shall affect the execution of cd:: HOME

## Context

**Behavior:** The `HOME` environment variable must affect `cd` execution. When `cd` is invoked without a `directory` operand, `HOME` provides the default target directory. The variable name `HOME` must be recognized by the cd implementation.

**Applies to:** `cd` invocations with no operand.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 182.
