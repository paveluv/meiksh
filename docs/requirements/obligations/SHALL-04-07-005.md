## Context

**Requirement — moderate shell relevance.** Preamble for the file access permission algorithm: when a process requests read, write, or execute/search access and no additional mechanism denies it, access SHALL be determined by the rules in -006 through -009.

**Shell impact:** This is the access permission algorithm the shell relies on when:
- Searching `$PATH` for executable commands (execute permission check)
- Opening files for redirection (read/write permission check)
- `test -r`, `test -w`, `test -x` builtins
- `cd` to directories (search permission check)

The shell delegates these checks to the OS but must understand the algorithm to provide correct error messages and `test` results.

**Spec reference:** basedefs/V1_chap04.html, section 4.7.
