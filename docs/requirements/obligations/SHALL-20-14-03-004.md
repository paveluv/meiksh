## Context

The `curpath` value **shall** be canonicalized as follows, processing components from beginning to end: (1) Dot components (`.`) and their surrounding slashes are deleted. (2) For each dot-dot (`..`) component, if preceded by a non-root, non-dot-dot component: first verify the preceding component refers to a directory (if not, display an error and stop); then delete the preceding component, the `..`, and separating slashes. (3) The implementation **may** further simplify by: removing trailing slashes (that aren't also leading), collapsing multiple consecutive non-leading slashes to one, and collapsing three or more leading slashes to one. (4) If canonicalization produces a null `curpath`, no further steps are taken. This is the logical path canonicalization used by `cd -L` (the default). Reference: cd utility DESCRIPTION step 8.
