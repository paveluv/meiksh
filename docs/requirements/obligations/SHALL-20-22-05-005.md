## Extracted Text

> The following operands shall be supported:: The name of a utility or a special built-in utility.

## Context

**Behavior:** The `command_name` operand must be the name of a utility or special built-in utility. This is the command to be executed (or looked up with -v/-V). The name may refer to any executable utility, regular built-in, special built-in, shell function (though functions are skipped by command), alias, or reserved word. For -v/-V, any command name is valid for lookup. For execution mode, the name should resolve to an executable command.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section OPERANDS (tag_20_22_05), line 126.
