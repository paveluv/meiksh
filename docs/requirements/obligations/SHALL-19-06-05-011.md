## Extracted Text

> Each field containing the results from an expansion shall be processed in order, intermixed with fields not containing the results of expansions, processed as described above, as if by using the following algorithm, examining bytes in the input field, from beginning to end:: Begin with an empty candidate field and the input as specified above.

## Context

**Behavior:** This is step 1 of the field splitting algorithm: **initialization**. Before processing any bytes in an input field, the algorithm must:
1. Start with an **empty candidate field** (the accumulator buffer is cleared)
2. Set the input to the full content of the current input field

The candidate field will accumulate non-delimiter bytes as the algorithm processes the input. When a delimiter boundary is detected, the candidate becomes an output field and is reset to empty.

**Spec reference:** See 2.6.5 Field Splitting in utilities/V3_chap02.html, line 1254.
