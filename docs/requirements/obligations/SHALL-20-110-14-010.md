## Extracted Text

> The following exit values shall be returned:: 128

## Context

**Behavior:** The `sh` utility shall return exit status 128 under the condition described in SHALL-20-110-14-011. This is the exit value label "128" in the exit status definition list.

**Applies to:** The `sh` utility when an unrecoverable read error occurs while reading commands.

**Details:** Exit status 128 indicates a fatal I/O error occurred while reading shell input (from the command_file or stdin), making it impossible to continue processing commands. See SHALL-20-110-14-011 for the specific condition and the dot-builtin exception.

**Spec reference:** Section "sh" EXIT STATUS, line 691 in `utilities/sh.html`.
