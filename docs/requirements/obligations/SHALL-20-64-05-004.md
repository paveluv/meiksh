## Extracted Text

> The following operands shall be supported:: exit_status

## Context

**Behavior:** The `kill` built-in must accept the `exit_status` operand when the `-l` option is specified. This operand is a decimal integer representing either a signal number or the exit status of a process terminated by a signal. See SHALL-20-64-05-005 for the detailed semantics.

**Conditions:** The `exit_status` operand is only meaningful with `-l`. Without `-l`, it is not used.
