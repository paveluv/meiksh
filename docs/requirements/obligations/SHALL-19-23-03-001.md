## Extracted Text

> The shell shall give the export attribute to the variables corresponding to the specified names, which shall cause them to be in the environment of subsequently executed commands. If the name of a variable is followed by =word, then the value of that variable shall be set to word.

## Context

**Behavior:** The `export` special built-in gives the **export attribute** to specified variables, causing them to appear in the environment of subsequently executed commands. If `name=word` form is used, the variable's value is also set.

**Details:**
1. `export VAR` — marks VAR for export without changing its value.
2. `export VAR=value` — sets VAR to value and marks it for export.
3. Exported variables are included in the environment passed to child processes (via `execve()`).

**Applies to:** All shell modes. This is a special built-in.
