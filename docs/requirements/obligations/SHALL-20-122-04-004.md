## Extracted Text

> The following option shall be supported:: Write the timing output to standard error in the format shown in the STDERR section.

## Context

**Behavior:** When `-p` is specified, the timing output written to stderr shall use the POSIX portable format defined in the STDERR section: `"real %f\nuser %f\nsys %f\n"`. Without `-p`, the format is implementation-defined.
