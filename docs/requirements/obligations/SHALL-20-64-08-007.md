## Extracted Text

> The following environment variables shall affect the execution of kill:: Determine the locale for the interpretation of sequences of bytes of text data as characters (for example, single-byte as opposed to multi-byte characters in arguments).

## Context

**Behavior:** `LC_CTYPE` determines how the `kill` built-in interprets byte sequences in its arguments as characters. This affects parsing of signal names and pid operands in multi-byte locales. In practice, signal names are ASCII, so the primary impact is on error message formatting and argument validation.
