## Extracted Text

> Interactive Shell: Redirection error with other utilities (not special built-ins) -- shall not exit

## Context

An interactive shell SHALL NOT exit when a redirection error occurs with a non-special-built-in utility. See 2.8.1 Consequences of Shell Errors in utilities/V3_chap02.html.

**Behavior required:**
- When the shell is interactive and a redirection attached to a regular built-in or external command fails, the shell SHALL write a diagnostic to stderr but SHALL NOT exit.
- A shell diagnostic message IS required.

**This is symmetric** with SHALL-19-08-01-014: both interactive and non-interactive shells survive redirection errors on non-special utilities.
