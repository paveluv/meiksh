## Extracted Text

> The if command shall execute a compound-list and use its exit status to determine whether to execute another compound-list.

## Context

**Behavior:** The `if` command executes a compound-list (the condition) and uses its exit status to determine whether to execute another compound-list (the then-body). The syntax is:

```
if compound-list ; then compound-list
[ elif compound-list ; then compound-list ] ...
[ else compound-list ]
fi
```

**Decision logic:** The condition compound-list's exit status (zero = true, non-zero = false) controls branching.

**Applies to:** All shell modes.
