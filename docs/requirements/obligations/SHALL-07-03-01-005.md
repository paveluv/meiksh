## Extracted Text

> Define characters to be classified as lowercase letters. In the POSIX locale, only: a b c d e f g h i j k l m n o p q r s t u v w x y z shall be included. In a locale definition file, no character specified for the keywords cntrl, digit, punct, or space shall be specified. The lowercase letters <a> to <z> of the portable character set are automatically included in this class.

## Context

**Behavior:** The `lower` keyword defines the set of lowercase letter characters:
- In the POSIX locale: exactly a-z (26 ASCII lowercase letters).
- In other locales: a-z are automatically included plus locale-specific lowercase letters.
- Constraint: No character in `lower` may also be in `cntrl`, `digit`, `punct`, or `space`.

**Shell relevance:** The shell uses the `lower` class for:
- `[[:lower:]]` bracket expression in pattern matching.
- Case conversion: `${var,,}` (to lowercase) — needs to know what is lowercase.
- In the POSIX locale, `[[:lower:]]` matches exactly a-z.
