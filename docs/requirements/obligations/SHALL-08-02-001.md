## Extracted Text

> This variable shall determine the locale category for native language, local customs, and coded character set in the absence of the LC_ALL and other LC_* (LC_COLLATE , LC_CTYPE , LC_MESSAGES , LC_MONETARY , LC_NUMERIC , LC_TIME ) environment variables. This can be used by applications to determine the language to use for error messages and instructions, collating sequences, date formats, and so on.

## Context

The LANG variable shall determine the locale category for native language, local customs, and coded character set in the absence of LC_ALL and other LC_* variables. It serves as the fallback locale setting.

**Shell relevance:** The shell must consult LANG as the fallback when specific LC_* variables and LC_ALL are not set. The precedence order is: (1) LC_ALL, (2) specific LC_* variable, (3) LANG, (4) implementation default. See XBD 8.2.
