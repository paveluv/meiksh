## Extracted Text

> The <signal.h> header shall define the following symbolic constant. The value shall be suitable for use in #if preprocessing directives:: Maximum size of a signal name returned by sig2str(), including the terminating null byte.

## Context

This describes the semantics of `SIG2STR_MAX`: it is the maximum size in bytes of a signal name string returned by `sig2str()`, including the null terminator. Signal names like "SIGRTMIN+3" or implementation-defined names must all fit within `SIG2STR_MAX` bytes. For `meiksh`, this establishes the upper bound for signal name buffer sizes throughout the codebase.
