## Extracted Text

> The <signal.h> header shall define the sigevent structure, which shall include at least the following members:

## Context

The `<signal.h>` header must define the `sigevent` structure with at least these members: `int sigev_notify` (notification type), `int sigev_signo` (signal number), `union sigval sigev_value` (signal value), `void (*sigev_notify_function)(union sigval)` (notification function), `pthread_attr_t *sigev_notify_attributes` (notification attributes). For `meiksh`, the `sigevent` structure is relevant if the shell uses timer-based or asynchronous I/O notifications. The shell itself typically does not create `sigevent` structures but depends on the system header defining them correctly.
