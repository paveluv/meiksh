## Extracted Text

> Special character on input, which is recognized if the ISIG flag is set. Generates a SIGQUIT signal which is sent to all processes in the foreground process group for which the terminal is the controlling terminal. If ISIG is set, the QUIT character shall be discarded when processed.

## Context

**Behavior:** The QUIT special character (typically ^\) — duplicate/alternate extraction of the same spec text as SHALL-11-01-09-002:
1. Recognized on input only when ISIG is set in `c_lflag`.
2. Generates SIGQUIT sent to all processes in the foreground process group of the controlling terminal.
3. The QUIT character shall be **discarded** when processed.

**Conditions:** ISIG must be set in `c_lflag`. The terminal must be the controlling terminal.

**Shell relevance:** See SHALL-11-01-09-002 for full details. The interactive shell ignores SIGQUIT. Foreground children receive it and typically core dump. The shell must report this in job status.

**Spec reference:** POSIX.1-2024, §11.1.9 Special Characters (tag_11_01_09), `basedefs/V1_chap11.html` line 224.
