## Extracted Text

> The shell shall format the output, including the proper use of quoting, so that it is suitable for reinput to the shell as commands that achieve the same exporting results, except:

## Context

**Behavior:** The output of `export -p` must be formatted with proper quoting so that it is suitable for **reinput** to the shell as commands that achieve the same exporting results. The output must be valid shell commands that, when executed, reproduce the same set of exported variables with the same values. Exceptions are listed in SHALL-19-23-03-006 and -007.

**Implementation:** Values containing special characters must be properly quoted (e.g., using single quotes or escaped double quotes).

**Applies to:** All shell modes.
