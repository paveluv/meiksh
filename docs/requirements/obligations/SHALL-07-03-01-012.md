## Extracted Text

> Define characters to be classified as printable characters, not including the <space>. In the POSIX locale, all characters in classes alpha, digit, and punct shall be included; no characters in class cntrl shall be included. In a locale definition file, characters specified for the keywords upper, lower, alpha, digit, xdigit, and punct are automatically included in this class. No character specified for the keyword cntrl shall be specified.

## Context

**Behavior:** The `graph` keyword defines printable characters excluding space:
- In the POSIX locale: union of `alpha`, `digit`, and `punct`. No `cntrl` characters.
- Automatically includes all `upper`, `lower`, `alpha`, `digit`, `xdigit`, and `punct` characters.
- Constraint: No `cntrl` character may be in `graph`.

**Shell relevance:** The shell uses the `graph` class for:
- `[[:graph:]]` bracket expression in pattern matching.
- `graph` defines "visible" characters (printable, non-space), useful for output formatting.
