## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

**Behavior:** The `kill` built-in must write only diagnostic (error) messages to stderr. No normal output should go to stderr. Normal output (signal listings) goes to stdout only when `-l` is specified.

**Conditions:** Diagnostic messages include errors like invalid signal names, invalid PIDs, permission denied, and no such process.
