## Extracted Text

> The following variables shall affect the execution of the shell:: [XSI] Determine the location of message catalogs for the processing of LC_MESSAGES .

## Context

This defines the semantics of the `NLSPATH` variable. Tagged `[XSI]` (X/Open System Interfaces Extension).

**Required behavior:**
- `NLSPATH` determines the location of message catalogs for the processing of `LC_MESSAGES`.
- This is an XSI extension. For a minimal POSIX shell, this variable must be recognized and passed to child processes in the environment, but the shell itself may not use message catalogs unless it implements XSI message catalog support.

**Conditions:** Relevant primarily for internationalized message support. The shell must at minimum preserve `NLSPATH` in the environment for child processes.

**Implementation note:** For meiksh, the practical impact is limited unless message catalogs (catopen/catgets) are used for shell diagnostic messages. The variable must still be recognized as a shell-affecting variable.

See 2.5.3 "Shell Variables" (`NLSPATH`) in `utilities/V3_chap02.html`.
