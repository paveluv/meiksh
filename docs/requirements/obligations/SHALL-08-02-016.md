## Extracted Text

> The values of locale categories shall be determined by a precedence order; the first condition met below determines the value:: If the LANG environment variable is defined and is not null, the value of the LANG environment variable shall be used.

## Context

**Behavior:** Locale precedence rule 3: If neither `LC_ALL` nor the specific `LC_*` variable for a category is defined/non-null, and the `LANG` environment variable is defined and is not null, the value of `LANG` shall be used for that locale category.

**Shell relevance:** `LANG` serves as the default locale for all categories when no more-specific variable is set. The shell must fall back to `LANG` when determining locale for pattern matching, character classification, message formatting, etc., if `LC_ALL` and the corresponding `LC_*` variable are both unset or null.
