## Extracted Text

> The following variables shall affect the execution of the shell:: The pathname of the user's home directory. The contents of HOME are used in tilde expansion (see 2.6.1 Tilde Expansion).

## Context

This defines the semantics of the `HOME` variable.

**Required behavior:**
- `HOME` contains the pathname of the user's home directory.
- The value of `HOME` is used in tilde expansion (see 2.6.1 "Tilde Expansion"): an unquoted `~` at the beginning of a word expands to the value of `$HOME`.
- `HOME` is typically set by the login program before the shell starts, but the shell must use whatever value is present.

**Conditions:** Applies whenever tilde expansion is performed (word expansion step 1) and whenever `$HOME` is referenced explicitly by scripts or builtins (e.g., `cd` with no arguments uses `$HOME`).

See 2.5.3 "Shell Variables" (`HOME`) and 2.6.1 "Tilde Expansion" in `utilities/V3_chap02.html`.
