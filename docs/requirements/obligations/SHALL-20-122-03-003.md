## Extracted Text

> The time utility shall invoke the utility named by the utility operand with arguments supplied as the argument operands and write a message to standard error that lists timing statistics for the utility. The message shall include the following information:: The User CPU time, equivalent to the sum of the tms_utime and tms_cutime fields returned by the times() function defined in the System Interfaces volume of POSIX.1-2024 for the process in which utility is executed.

## Context

**Behavior:** The timing message shall include User CPU time, equivalent to the sum of `tms_utime` and `tms_cutime` fields returned by the `times()` function for the process in which the utility is executed.

**Meaning:** `tms_utime` is user CPU time of the process itself; `tms_cutime` is the cumulative user CPU time of all waited-for child processes. The sum gives total user CPU time consumed by the utility and all its children.

**Implementation:** Call `times()` before and after executing the utility. The user CPU time is `(after.tms_utime + after.tms_cutime) - (before.tms_utime + before.tms_cutime)`, divided by `sysconf(_SC_CLK_TCK)` to convert to seconds.
