## Extracted Text

> The following exit values shall be returned:: 0

## Context

The `fc` builtin must return exit status 0 when successfully completing a listing operation (`fc -l`). This means the history commands were successfully retrieved and written to stdout without error. This exit status specifically applies to the listing mode; in edit/re-execute mode, the exit status is instead determined by the re-executed command (see SHALL-20-44-14-006). Reference: POSIX.1-2024 §fc, EXIT STATUS (tag_20_44_14).
