## Extracted Text

> The following exit values shall be returned:: An error occurred.

## Context

**Behavior:** Exit status >0 is returned when an error occurred. Errors include:
- Invalid signal name or number
- No matching process found for a pid operand
- Permission denied when sending the signal
- Invalid pid operand syntax
- Write failure during `-l` output
- Any other error condition

The specific value >0 is not further defined by the spec (typically 1 is used for general errors, but any non-zero value satisfies the requirement).
