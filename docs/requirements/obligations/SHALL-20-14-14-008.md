## Extracted Text

> The following exit values shall be returned:: 1

## Context

**Behavior:** The cd built-in must return exit status exactly 1 in the case described in SHALL-20-14-14-009. This is a specific exit code (not just ">0") for the case where both -e and -P are in effect, chdir succeeded, but PWD could not be determined.

**Applies to:** `cd -eP` invocations where chdir succeeds but the physical path cannot be resolved.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 250.
