## Context

When a new file is created, the following default attributes shall apply (unless the utility description overrides): (1) user ID set to effective user ID, (2) group ID set to effective group ID or parent directory group ID, (3) permission bits based on file type with umask applied, (4) timestamps updated, (5) directories created empty / regular files created with length zero, (6) symlink creation undefined without `POSIX2_SYMLINKS`, (7) default type is regular file. For `meiksh`, these rules govern file creation for output redirection (`>`, `>>`), here-documents (temp files), and any files created by shell operations. See POSIX Utilities Chapter 1, Section 1.1.1.4 (utilities/V3_chap01.html lines 86–105).
