## Extracted Text

> The following environment variables shall affect the execution of unalias:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

**Behavior:** LANG provides default values for any i18n variables that are unset or null during `unalias` execution. The precedence is: LC_ALL overrides all, then specific LC_* variables, then LANG as fallback.

**Conditions:** For `unalias`, this primarily affects diagnostic message formatting (LC_MESSAGES) and character interpretation in alias-name operands (LC_CTYPE).

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_133_08) in `utilities/unalias.html`, lines 77-78. See XBD 8.2.
