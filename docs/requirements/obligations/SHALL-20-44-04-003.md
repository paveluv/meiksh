## Extracted Text

> The following options shall be supported:: -e editor

## Context

The `fc` builtin must support the `-e editor` option, which specifies the editor to use for editing commands from the history. The `editor` argument is a utility name that is resolved via `PATH` lookup. If `-e` is not specified, the value of `FCEDIT` is used as the default editor. If `FCEDIT` is null or unset, `ed` must be used as the editor. The `-e` option is used with the edit-and-re-execute form of `fc` (without `-l` or `-s`). Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
