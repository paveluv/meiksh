## Extracted Text

> If no directory operand is given and the HOME environment variable is empty or undefined, the default behavior is implementation-defined and no further steps shall be taken.

## Context

When `cd` is invoked with no directory operand and the `HOME` environment variable is empty or undefined, the behavior is **implementation-defined** and no further steps **shall** be taken. The shell must detect this condition (no operand + empty/unset `HOME`) and handle it according to its implementation-defined policy (e.g., print an error, do nothing, or change to some default). The implementation must document its chosen behavior. Reference: cd utility DESCRIPTION step 1.
