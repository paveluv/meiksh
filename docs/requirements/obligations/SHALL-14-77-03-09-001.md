## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflush():

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for use as the `queue_selector` argument to `tcflush()`. These constants specify which queue(s) to flush: TCIFLUSH, TCIOFLUSH, TCOFLUSH.

**Conditions:** These constants must be defined and usable with `tcflush()`.

**Shell relevance:** meiksh uses `tcflush()` to discard pending input or output:
- **TCIFLUSH**: flush input queue (discard unread typeahead)
- **TCOFLUSH**: flush output queue (discard unsent output)
- **TCIOFLUSH**: flush both queues
Useful when the shell needs to discard stale input (e.g., after a signal) or cancel pending output.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` lines 468–469.
