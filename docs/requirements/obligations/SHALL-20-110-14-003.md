## Extracted Text

> The following exit values shall be returned:: The script to be executed consisted solely of zero or more blank lines or comments, or both.

## Context

**Behavior:** When the script (command_file or standard input) consists solely of zero or more blank lines or comments (or a mix of both), the `sh` utility shall return exit status 0.

**Applies to:** Non-interactive shell invocations processing a script file or stdin.

**Details:** This covers:
- An empty script (zero bytes or only newlines).
- A script with only comment lines (lines starting with `#`).
- A script with a mix of blank lines and comments.
In all these cases, the shell parsed the input, found no commands to execute, and exits with status 0.
