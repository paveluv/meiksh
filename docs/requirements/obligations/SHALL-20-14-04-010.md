## Extracted Text

> Handle the operand dot-dot physically; symbolic link components shall be resolved before dot-dot components are processed (see step 7. in the DESCRIPTION).

## Context

This reiterates the `-P` (physical) mode behavior for `cd`: symbolic link components **shall** be resolved before dot-dot components are processed. The shell first resolves all symlinks in the path, then handles `..` against the physical (resolved) directory structure. This corresponds to step 7 of the cd DESCRIPTION algorithm. Reference: cd utility OPTIONS, cd utility DESCRIPTION step 7.
