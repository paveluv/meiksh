## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: t_fmt_ampm

## Context

The `t_fmt_ampm` keyword shall be recognized in an LC_TIME locale definition. It introduces the definition of the 12-hour clock time format string.

**Shell relevance:** This format is used by `%r` in `date`. In the POSIX locale: "%I:%M:%S %p". See XBD 7.3.5.1.
