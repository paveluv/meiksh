## Context

The LC_MESSAGES category shall define the format and values for affirmative and negative responses used by various utilities. This information is available through `nl_langinfo()`. In the POSIX locale, `yesexpr` is "^[yY]" and `noexpr` is "^[nN]".

**Shell relevance:** The shell and utilities that prompt for yes/no confirmation (e.g., `rm -i`) use LC_MESSAGES to determine which responses are affirmative or negative. The shell must honor the active locale's yesexpr/noexpr patterns. See XBD 7.3.6.
