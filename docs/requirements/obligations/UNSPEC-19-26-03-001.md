## Extracted Text

> Write the current settings of the options to standard output in an unspecified format.

## Context

**Behavior (unspecified):** The `set -o` command (with no option-argument) writes the current settings of the options to standard output in an **unspecified format**. This is the display mode for `-o`. Common implementations show a list like `option on/off`.

**Note:** `set +o` (with `+` prefix) is required to output in a format suitable for reinput, but `set -o` with no argument has unspecified format.

**Applies to:** All shell modes.
