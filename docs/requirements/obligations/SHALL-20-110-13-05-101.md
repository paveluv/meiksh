## Extracted Text

> The following commands shall be recognized in command mode:: Undo all changes made to the edit line. This operation shall not undo the copy of any command line to the edit line.

## Context

**Behavior:** In vi command mode, `U` undoes ALL changes made to the edit line, restoring it to its original state when editing began.

**Restriction:** This operation shall NOT undo the copy of a command line to the edit line (same exception as `u`). The "original" state is updated when a history line is promoted to become the new edit line.

**Implementation:** Store the original edit line content when editing begins (or when a history line is promoted to edit line). On `U`, restore the edit line to that stored original content.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 622–623.
