## Extracted Text

> The following commands shall be recognized in command mode:: Repeat the most recent f, F, t, or T command. Any number argument on that previous command shall be ignored. Errors are those described for the repeated command.

## Context

**Behavior:** In vi command mode, `;` repeats the most recent `f`, `F`, `t`, or `T` command in the same direction. Any count from the original command is ignored; the new count (if any) applies. Errors follow the same rules as the repeated command (e.g., alert if char not found).

**Implementation:** Store the last f/F/t/T command type and its target character. On `;`, replay the same search in the same direction with the current count (default 1). This is a motion command.
