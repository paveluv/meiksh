## Extracted Text

> The for loop shall execute a sequence of commands for each member in a list of items. The for loop requires that the reserved words do and done be used to delimit the sequence of commands.

## Context

**Behavior:** The `for` loop executes a sequence of commands once for each member in a list of items. The syntax requires the reserved words `do` and `done` to delimit the command body:

```
for name [ in word... ] ; do compound-list ; done
```

**Parsing:** The parser must recognize `for` as a reserved word, followed by a variable name, optionally followed by `in` and a word list, then `;` (or newline), `do`, compound-list, `done`. The `do` and `done` tokens are required reserved words.

**Applies to:** All shell modes.
