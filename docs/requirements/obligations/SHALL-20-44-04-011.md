## Extracted Text

> The following options shall be supported:: -s

## Context

The `fc` builtin must support the `-s` option, which provides quick command re-execution without invoking an editor. The synopsis form is `fc -s [old=new] [first]`. When `-s` is specified, `fc` re-executes the selected command immediately. If the `old=new` operand is given, the first occurrence of string `old` in the command is replaced with `new` before re-execution. If `first` is omitted, the previous command is used. The `-s` option is mutually exclusive with `-l` and `-e`. Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
