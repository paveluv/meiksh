> Otherwise, the implementation-defined default locale shall be used.

## Context

**Behavior:** Duplicate of the core rule in SHALL-08-02-017: When all locale variables (`LC_ALL`, specific `LC_*`, and `LANG`) are unset or null, the implementation-defined default locale shall be used.

**Shell relevance:** Same as SHALL-08-02-017. The `meiksh` implementation should default to the POSIX ("C") locale as its implementation-defined default.
