## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcsetattr():: Change attributes immediately.

## Context

**Behavior:** This is the description of TCSANOW: **change attributes immediately**. When `tcsetattr(fd, TCSANOW, &termios)` is called, the terminal attributes are changed without waiting for pending output to be transmitted or pending input to be received.

**Conditions:** Used as the `optional_actions` parameter to `tcsetattr()`.

**Shell relevance:** See SHALL-14-77-03-08-002. TCSANOW is the simplest and most commonly used option. The shell uses it when it doesn't need to wait for output to drain (e.g., at prompt display time when output is already flushed).

**Spec reference:** POSIX.1-2024, `<termios.h>` Attribute Selection (tag_14_77_03_08), `basedefs/termios.h.html` line 461.
