## Extracted Text

> When the -v option is specified, standard output shall be formatted as:

## Context

**Behavior:** When `command -v` is used, stdout must be formatted as:
```
"%s\n", <pathname or command>
```
where `<pathname or command>` is determined by the category of the command_name (see SHALL-20-22-04-006):
- Absolute pathname for executables/PATH-found builtins.
- Just the name for functions, special builtins, non-PATH builtins, reserved words.
- Alias definition for aliases.
- No output if not found.

Each result is a single line terminated by a newline.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section STDOUT (tag_20_22_10), line 172.
