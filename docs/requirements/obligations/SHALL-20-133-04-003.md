## Extracted Text

> The following option shall be supported:: -a

## Context

**Behavior:** The `unalias` utility shall accept the `-a` option. When `-a` is specified, all alias definitions are removed from the current shell execution environment. This is equivalent to removing every currently defined alias.

**Conditions:** When `-a` is used, no alias-name operands are expected (the synopsis shows `unalias -a` as a separate form). Using `-a` removes all aliases regardless of how they were defined (including implementation-defined predefined aliases and aliases set via ENV).
