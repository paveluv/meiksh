## Extracted Text

> The following operands shall be supported:: optstring

## Context

**Behavior:** The `getopts` utility must accept an `optstring` operand as its first argument. This is a string containing the recognized option characters. See SHALL-20-53-05-003/006 for the detailed semantics of `optstring`.
