## Extracted Text

> The following exit values shall be returned:: The end of options was encountered.

## Context

**Behavior:** Describes exit status 1: returned when getopts has reached the end of options. At this point, `OPTIND` is set to the index of the first non-option operand (or param count + 1 if none), and `name` is set to `?`.

**Spec reference:** POSIX.1-2024, `getopts` EXIT STATUS, line 207 of `utilities/getopts.html`.
