## Extracted Text

> The following environment variables shall affect the execution of getopts:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

**Behavior:** Describes the effect of `LC_ALL`: when set to a non-empty string, it overrides all other internationalization variables for `getopts`. This is the standard semantics of `LC_ALL` per XBD 8.2.

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, line 144 of `utilities/getopts.html`.
