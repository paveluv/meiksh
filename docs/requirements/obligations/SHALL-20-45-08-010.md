## Extracted Text

> The following environment variables shall affect the execution of fg:: NLSPATH

## Context

The `fg` builtin must recognize the `NLSPATH` environment variable on XSI-conformant systems. `NLSPATH` determines the location of message objects and message catalogs for internationalized diagnostic messages. See SHALL-20-45-08-011 for the full description. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
