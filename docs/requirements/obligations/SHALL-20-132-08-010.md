## Extracted Text

> The following environment variables shall affect the execution of umask:: NLSPATH

## Context

**Behavior:** NLSPATH shall determine the location of message objects and message catalogs for the `umask` utility. This is an XSI extension.

**Conditions:** Only relevant on systems supporting the XSI option. NLSPATH is used to locate message catalogs for locale-dependent messages.
