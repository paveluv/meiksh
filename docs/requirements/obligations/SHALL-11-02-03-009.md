## Extracted Text

> Backspace delay shall last about 0.05 seconds. If OFILL is set, one fill character shall be transmitted.

## Context

**Behavior:** Backspace delay (BSDLY mask in `c_oflag`) shall last about 0.05 seconds. If OFILL is set, one fill character shall be transmitted instead.

**Relevance to meiksh:** The `stty` builtin must support setting BS0/BS1 via the BSDLY mask. The actual delay is handled by the OS terminal driver.

**Spec reference:** POSIX.1-2024, §11.2.3 Output Modes (tag_11_02_03), `basedefs/V1_chap11.html` line 807.
