## Extracted Text

> The unalias utility shall conform to XBD 12.2 Utility Syntax Guidelines.

## Context

**Behavior:** The `unalias` utility's option parsing shall conform to the POSIX Utility Syntax Guidelines (XBD 12.2). This means:
- Options are single characters preceded by `-`.
- `--` terminates option processing.
- The only defined option is `-a`.

**Error handling:** Unrecognized options shall produce a diagnostic message on stderr and a non-zero exit status.

**Spec reference:** Section OPTIONS (tag_20_133_04) in `utilities/unalias.html`, line 45. See XBD 12.2.
