## Extracted Text

> The <signal.h> header shall define the following data types:: sigset_t

## Context

The `<signal.h>` header must define `sigset_t`, used to represent sets of signals. For `meiksh`, `sigset_t` is essential for signal mask operations: `sigprocmask()` to block/unblock signals, `sigpending()` to check pending signals, and `sigsuspend()` to atomically set a mask and wait. The shell uses signal masks extensively—blocking SIGCHLD during critical sections, managing signal delivery for job control, etc.
