## Extracted Text

> A sequence of all the directory entries in a particular directory. An open directory stream may be implemented using a file descriptor.

## Context

**Definition only — no shell relevance.** Defines "directory stream" as a sequence of all directory entries in a directory. Notes it may be implemented using a file descriptor.

**Shell impact:** None directly. The shell uses directory operations for filename expansion (globbing) and the `cd` builtin, but interacts via higher-level APIs (`opendir`/`readdir`), not directly with directory stream semantics. No action required.
