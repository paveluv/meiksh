## Extracted Text

> The following environment variables shall affect the execution of alias:: LC_CTYPE

## Context

`LC_CTYPE` **shall** affect the execution of `alias`. It determines the locale for character classification — how sequences of bytes are interpreted as characters (single-byte vs. multi-byte). This affects how the shell interprets alias names and values that may contain multi-byte characters. Reference: alias utility ENVIRONMENT VARIABLES.
