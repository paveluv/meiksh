## Extracted Text

> The following environment variables shall affect the execution of alias:: LC_MESSAGES

## Context

`LC_MESSAGES` **shall** affect the execution of `alias`. It determines the locale used for diagnostic messages written to stderr (e.g., "alias not found" type messages). The language and format of these messages should follow the `LC_MESSAGES` locale. Reference: alias utility ENVIRONMENT VARIABLES.
