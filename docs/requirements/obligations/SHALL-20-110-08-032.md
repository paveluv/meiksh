> The following environment variables shall affect the execution of sh:: PWD

## Context

**Behavior:** The PWD environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-033).

**Applies to:** All shell modes (interactive and non-interactive).

**Details:** PWD represents the absolute pathname of the current working directory. The shell updates PWD when the current directory changes (e.g., via `cd`). See SHALL-20-110-08-033 for the full behavior specification.
