## Extracted Text

> If both -L and -P options are specified, the last of these options shall be used and all others ignored. If neither -L nor -P is specified, the operand shall be handled dot-dot logically; see the DESCRIPTION.

## Context

If both `-L` and `-P` options are specified to `cd`, the last one given **shall** be used and all others ignored. This means `cd -L -P dir` uses physical mode, while `cd -P -L dir` uses logical mode. If neither `-L` nor `-P` is specified, the default behavior **shall** be logical mode (`-L`) — dot-dot is handled logically as described in the DESCRIPTION. The shell must track which option was specified last during option parsing. Reference: cd utility OPTIONS.
