## Extracted Text

> The following environment variables shall affect the execution of kill:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

**Behavior:** `LANG` provides default values for any LC_* variable that is unset or null. The precedence order (per XBD 8.2) is: LC_ALL overrides everything; then specific LC_* variables; then LANG as fallback. The `kill` built-in must follow this standard precedence when determining locale for character interpretation and diagnostic messages.

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_64_08) in `utilities/kill.html`, lines 165–166. See also XBD 8.2 Internationalization Variables.
