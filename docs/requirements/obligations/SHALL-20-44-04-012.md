## Extracted Text

> The following options shall be supported:: Re-execute the command without invoking an editor.

## Context

This is the description for the `-s` option of `fc`. When `-s` is specified, `fc` re-executes the selected command directly without opening an editor. The command is first optionally modified by the `old=new` substitution (replacing the first occurrence of `old` with `new`), then appended to the history list, and then re-executed. If no `first` operand is given, the most recently entered command is used. This provides a quick "redo" capability. Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
