## Extracted Text

> If a value computation A of an atomic object M happens before an operation B on M, then A shall take its value from a side effect X on M, where X precedes B in the modification order of M. (This is known as "read-write coherence".)

## Context

This is a C-level memory model requirement about read-write coherence for atomic objects. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.1.1 Data Races.
