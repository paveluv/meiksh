## Extracted Text

> The following environment variables shall affect the execution of sh:: ENV

## Context

**Behavior:** The ENV environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-003).

**Applies to:** Interactive shells only (the [UP] User Portability option).

**Details:** ENV names the initialization file for interactive shells. When an interactive shell starts, it parameter-expands the value of ENV and sources the resulting pathname. See SHALL-20-110-08-003 for the full behavior specification including security restrictions.
