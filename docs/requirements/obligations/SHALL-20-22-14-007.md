## Extracted Text

> Otherwise, the following exit values shall be returned:: 126

## Context

**Behavior:** Exit status 126 must be returned by `command` (execution mode) when the utility specified by command_name was found but could not be invoked. This typically means the file exists but lacks execute permission, or is not a valid executable format.

**Applies to:** `command command_name` where command_name resolves to a file that cannot be executed.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section EXIT STATUS (tag_20_22_14), line 205.
