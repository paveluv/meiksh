## Extracted Text

> The following exit values shall be returned:: The current working directory was successfully changed, both the -e and the -P options are in effect, and the correct value of the PWD environment variable could not be determined.

## Context

**Behavior:** Exit status 1 (exactly) must be returned when ALL of these conditions hold:
1. The working directory was successfully changed (chdir succeeded).
2. Both `-e` AND `-P` options are in effect.
3. The correct value of PWD could not be determined (e.g., insufficient permissions to resolve the physical path).

This distinguishes from the case without -e (which returns 0) — the -e flag makes PWD-determination failure a non-zero exit. The directory HAS been changed, but the exit status signals that PWD may be wrong.
