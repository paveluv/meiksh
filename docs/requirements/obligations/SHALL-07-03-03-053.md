## Extracted Text

> A string that shall be used to indicate a non-negative-valued formatted monetary quantity.

## Context

The `positive_sign` value is the string displayed to indicate a non-negative monetary value (e.g., "+" or empty string). The shell does not directly format monetary values. The shell must correctly propagate the LC_MONETARY environment variable so programs can access this through `localeconv()`.

See: `basedefs/V1_chap07.html` §7.3.3 LC_MONETARY.
