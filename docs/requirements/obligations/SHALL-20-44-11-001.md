## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `fc` builtin must never write normal output to stderr. Standard error must be used exclusively for diagnostic messages such as invalid option errors, history access failures, command not found in history, or editor invocation failures. All normal output (command listings, etc.) goes to stdout. Reference: POSIX.1-2024 §fc, STDERR (tag_20_44_11).
