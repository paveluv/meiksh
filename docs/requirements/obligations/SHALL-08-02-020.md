## Extracted Text

> If the LANG environment variable is defined and is not null, the value of the LANG environment variable shall be used.

## Context

**Behavior:** Duplicate of the core rule in SHALL-08-02-016: If `LANG` is defined and non-null, it shall be used as the default locale value for any category not overridden by `LC_ALL` or a specific `LC_*` variable.

**Shell relevance:** Same as SHALL-08-02-016. `LANG` is the fallback locale for all categories.

**Spec reference:** Section 8.2 "Internationalization Variables", line 612 in `basedefs/V1_chap08.html`.
