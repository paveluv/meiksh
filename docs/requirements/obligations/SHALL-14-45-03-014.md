## Extracted Text

> The <signal.h> header shall define the following symbolic constants for the values of sigev_notify:: SIGEV_NONE

## Context

`SIGEV_NONE` must be defined in `<signal.h>`. When set as `sigev_notify`, no asynchronous notification is delivered when the event occurs. For `meiksh`, this is relevant if the shell creates timers or asynchronous I/O requests that should not generate notifications. This is a system header requirement.
