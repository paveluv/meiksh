## Extracted Text

> The jobs utility shall conform to XBD 12.2 Utility Syntax Guidelines.

## Context

**Behavior:** The `jobs` utility must accept options and operands according to the standard Utility Syntax Guidelines (XBD 12.2). This means:
- Options are single characters preceded by `-`.
- Options `-l` and `-p` are mutually exclusive (specified with `|` in synopsis).
- Options must precede operands.
- `--` can be used to terminate option processing.

**Synopsis:** `jobs [-l|-p] [job_id...]`

**Spec reference:** POSIX.1-2024, `jobs` OPTIONS, line 49 of `utilities/jobs.html`; XBD 12.2.
