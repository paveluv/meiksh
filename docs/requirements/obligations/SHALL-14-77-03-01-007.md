## Extracted Text

> The <termios.h> header shall define the following data types through typedef:: Used for terminal modes.

## Context

**Behavior:** The `tcflag_t` type is described as being **used for terminal modes**. This is the descriptive purpose of the `tcflag_t` typedef — it holds the bitwise-OR combinations of terminal mode flags in the four mode fields of the termios structure.

**Conditions:** Part of the `tcflag_t` type definition in `<termios.h>`.

**Shell relevance:** All terminal mode flag variables in meiksh should be declared as `tcflag_t`. This includes saved mode values when switching between canonical and non-canonical modes, and intermediate values during flag manipulation.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 49.
