## Extracted Text

> The following exit values shall be returned:: The file mode creation mask was successfully changed, or no mask operand was supplied.

## Context

**Behavior:** This is the description for exit status 0 of `umask`. Exit status 0 shall be returned when:
1. A mask operand was provided and the file mode creation mask was successfully changed to the specified value, OR
2. No mask operand was provided (query mode) and the current mask value was successfully written to stdout.
