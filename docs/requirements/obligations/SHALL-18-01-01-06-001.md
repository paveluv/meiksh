## Extracted Text

> All files shall have the three time values described by XBD 4.12 File Times Update.

## Context

All files shall have the three time values described by XBD 4.12 File Times Update: (1) last data access time (`st_atim`), updated when file data is read; (2) last data modification time (`st_mtim`), updated when file data is written; (3) last file status change time (`st_ctim`), updated when file metadata (permissions, ownership, link count, etc.) changes. For `meiksh`, these timestamps affect the behavior of test operators (`-nt`, `-ot` for file comparison by modification time), the `ls -l` output format, and any conditional logic based on file age. The shell's redirection operations trigger timestamp updates on the affected files. See POSIX Utilities Chapter 1, Section 1.1.1.6 (utilities/V3_chap01.html line 728) and XBD 4.12.
