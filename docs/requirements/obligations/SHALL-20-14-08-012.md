## Extracted Text

> The following environment variables shall affect the execution of cd:: LC_MESSAGES

## Context

**Behavior:** The `LC_MESSAGES` environment variable must affect `cd` execution. LC_MESSAGES is recognized as a locale variable that influences diagnostic message formatting for the cd built-in.

**Applies to:** Diagnostic messages written to stderr by cd (e.g., "cd: no such file or directory").

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 193.
