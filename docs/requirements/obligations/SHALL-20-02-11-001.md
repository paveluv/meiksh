## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `alias` builtin **shall** use stderr only for diagnostic messages. Normal output (alias definitions) goes to stdout. Error messages (e.g., "alias not found") go to stderr. No other output shall be written to stderr. Reference: alias utility STDERR.
