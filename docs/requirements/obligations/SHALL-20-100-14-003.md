## Extracted Text

> The following exit values shall be returned:: Successful completion.

## Context

**Behavior required:** Exit status 0 indicates successful completion of the `read` builtin. This is the descriptive text for exit code 0.

**Condition:** When `read` successfully reads a delimiter-terminated line and assigns all variables.

**Implementation detail:** Combined with SHALL-20-100-14-002: on success (delimiter found, no errors), return 0. This is distinct from EOF (return 1) where variables are still set but no delimiter was found.
