## Extracted Text

> The following operands shall be supported:: alias-name

## Context

The `alias` builtin **shall** support the `alias-name` operand (a bare name without `=`). When an `alias-name` operand is given without `=string`, the shell looks up the alias with that name and writes its definition to stdout. If no alias with that name exists, a diagnostic is written to stderr and a non-zero exit status is returned. Reference: alias utility OPERANDS.
