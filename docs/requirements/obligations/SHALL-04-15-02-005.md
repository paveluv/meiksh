## Extracted Text

> The mtx_lock(), mtx_timedlock(), and mtx_trylock() functions shall synchronize memory on all calls that acquire the mutex. The mtx_unlock() function shall synchronize memory on all calls that release the mutex.

## Context

This is a C11 threads-level requirement about C11 mutex memory synchronization. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.2 Memory Synchronization.
