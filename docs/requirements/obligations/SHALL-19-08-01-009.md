## Extracted Text

> Interactive Shell: Redirection error with special built-in utilities -- shall not exit

## Context

An interactive shell SHALL NOT exit when a redirection error occurs with a special built-in utility. See 2.8.1 Consequences of Shell Errors in utilities/V3_chap02.html.

**Behavior required:**
- When the shell is interactive and a redirection attached to a special built-in fails, the shell SHALL write a diagnostic to stderr but SHALL NOT exit.
- A shell diagnostic message IS required.
- The shell returns to the prompt.

**Example:** In an interactive shell, `exec >/no/such/dir` fails but the shell continues.
