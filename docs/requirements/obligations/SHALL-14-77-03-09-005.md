## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflush():: Flush both pending input and untransmitted output.

## Context

**Behavior:** This is the description of TCIOFLUSH: **flush both pending input and untransmitted output**. When `tcflush(fd, TCIOFLUSH)` is called, both the input and output queues are flushed.

**Conditions:** Used as the `queue_selector` parameter to `tcflush()`.

**Shell relevance:** See SHALL-14-77-03-09-004. Combines TCIFLUSH and TCOFLUSH behavior.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 475.
