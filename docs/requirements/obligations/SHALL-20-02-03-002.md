## Extracted Text

> An alias definition shall affect the current shell execution environment and the execution environments of the subshells of the current shell. When used as specified by this volume of POSIX.1-2024, the alias definition shall not affect the parent process of the current shell nor any utility environment invoked by the shell; see 2.13 Shell Execution Environment.

## Context

An alias definition created by `alias` **shall** affect the current shell execution environment and the execution environments of subshells of the current shell. It **shall not** affect the parent process of the current shell or any utility environment invoked by the shell (i.e., external commands run in child processes do not inherit shell aliases). This means aliases are stored in the shell's internal state and are copied into subshell environments (created by `()`, command substitution, etc.) but not exported to external processes. Reference: alias utility DESCRIPTION, XCU 2.13 (Shell Execution Environment).
