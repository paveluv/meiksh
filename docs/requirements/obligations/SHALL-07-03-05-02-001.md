## Extracted Text

> The appropriate time representation in the 12-hour clock format; if the 12-hour format is not supported in the locale, this shall be either an empty string or a string specifying a 24-hour clock format.

## Context

The T_FMT_AMPM `nl_langinfo()` constant provides the 12-hour clock time representation. If the 12-hour format is not supported in the locale, this value shall be either an empty string or a string specifying a 24-hour clock format. Accessed via `nl_langinfo(T_FMT_AMPM)` from `<langinfo.h>`.

**Shell relevance:** The shell or its `date`/`printf` builtins may query this value when formatting time with `%r`. See XBD 7.3.5.2.
