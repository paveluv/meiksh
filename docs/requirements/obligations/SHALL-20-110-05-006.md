## Extracted Text

> The following operands shall be supported:: command_file

## Context

This is a label/header obligation for the `command_file` operand. The `command_file` is the pathname of a file containing shell commands, used when `sh` is invoked without `-c` or `-s` and a non-option operand is provided. Detailed semantics (path resolution, `$0` assignment) are in SHALL-20-110-05-007 / SHALL-20-110-05-014.

See utilities/sh.html#tag_20_110_05.
