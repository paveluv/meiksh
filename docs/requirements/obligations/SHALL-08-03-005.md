## Extracted Text

> [XSI] Describes which message components shall be used in writing messages by fmtmsg().

## Context

**Behavior:** The `MSGVERB` variable (XSI extension) describes which message components shall be used by `fmtmsg()` when writing messages.

**Shell relevance:** The shell does not use `fmtmsg()` directly, and this is an XSI extension. The shell must propagate `MSGVERB` to child processes but does not need to interpret it itself. No shell-side implementation is required beyond environment propagation.
