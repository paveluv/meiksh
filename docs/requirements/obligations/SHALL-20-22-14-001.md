## Extracted Text

> When the -v or -V options are specified, the following exit values shall be returned:

## Context

**Behavior:** When `command` is invoked with `-v` or `-V`, the exit status rules differ from execution mode. The defined exit values for information mode are: 0 (successful completion — command_name was found) and >0 (command_name could not be found or an error occurred).

**Applies to:** `command -v name` and `command -V name` invocations.
