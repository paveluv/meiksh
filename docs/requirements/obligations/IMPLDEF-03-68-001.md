## Extracted Text

> An interval of time; an implementation-defined number of these occur each second. Clock ticks are one of the units that may be used to express a value found in type clock_t.

## Context

**Definition — moderate shell relevance.** Defines "clock tick" as an implementation-defined interval; an implementation-defined number occur each second. Clock ticks are used in `clock_t` values, which the `times` builtin reports.

**Shell impact:** The `times` builtin (XSI) reports user and system CPU time consumed by the shell and its children, measured in clock ticks. The shell must convert clock ticks to seconds for display using `sysconf(_SC_CLK_TCK)`. The `time` reserved word also reports timing information, though typically in seconds.

**Spec reference:** basedefs/V1_chap03.html, section 3.68; see also `utilities/time.html` and `basedefs/sys_times.h.html`.
