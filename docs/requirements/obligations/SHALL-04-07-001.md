## Context

**Requirement — moderate shell relevance.** Implementations MAY provide additional or alternate file access control mechanisms. An additional mechanism SHALL only further restrict permissions. An alternate mechanism has additional requirements (see -002 through -004).

**Shell impact:** The shell's file access behavior (command execution, redirections, `test -r/-w/-x`) depends on the OS-level access control. The shell does not implement access control but must correctly interpret and report access denial errors. The `test` builtin's `-r`, `-w`, `-x` operators should reflect effective access, which may be affected by additional/alternate mechanisms.
