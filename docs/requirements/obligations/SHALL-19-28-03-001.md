## Extracted Text

> The times utility shall write the accumulated user and system times for the shell and for all of its child processes, in the following POSIX locale format:

## Context

**Behavior:** The `times` special built-in writes the accumulated user and system times for the shell and all its child processes to standard output, in the POSIX locale format:

```
%dm%fs %dm%fs
%dm%fs %dm%fs
```

First line: shell user time and system time. Second line: children's user time and system time. Times are in minutes and seconds with fractional seconds.

**Applies to:** All shell modes. This is a special built-in.
