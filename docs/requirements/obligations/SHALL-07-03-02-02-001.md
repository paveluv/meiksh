## Context

**Behavior:** The `collating-symbol` keyword defines symbolic names that can be used in collation order statements (between `order_start` and `order_end`). These symbols serve as position markers in the collation sequence — they represent collation positions without corresponding to actual characters.

Example: `collating-symbol <CAPITAL>` defines a symbol that can be placed in the ordering to create a boundary between groups of characters.

**Shell relevance:** The shell does not parse locale definition files. Collating symbols are internal to the collation definition and do not appear in the shell's pattern matching syntax. They affect the resulting collation order which the shell uses indirectly.
