## Extracted Text

> The export special built-in shall be a declaration utility. Therefore, if export is recognized as the command name of a simple command, then subsequent words of the form name=word shall be expanded in an assignment context. See 2.9.1.1 Order of Processing.

## Context

**Behavior:** `export` is a **declaration utility**. When recognized as the command name of a simple command, subsequent words of the form `name=word` are expanded in an **assignment context** (tilde expansion occurs after `=`, field splitting and pathname expansion do NOT apply to the value). See 2.9.1.1 Order of Processing for details on assignment context expansion.

**Applies to:** All shell modes.

**Spec reference:** `export` DESCRIPTION in `utilities/V3_chap02.html`, lines 4222-4224. Cross-ref: 2.9.1.1 Order of Processing.
