## Extracted Text

> Unless stated otherwise, the following shall apply to every symbolic constant:: It may be defined as another type of constant—e.g., an enumeration constant—as well as being a macro.

## Context

**Definition only — no shell relevance.** States that a symbolic constant MAY be defined as an enumeration constant as well as being a macro. This is a C API flexibility provision.

**Shell impact:** None. No action required.
