## Extracted Text

> The following environment variables shall affect the execution of sh:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

**Behavior:** LC_MESSAGES determines the locale used for formatting and content of diagnostic messages written to standard error by the shell. This includes:
- Syntax error messages
- "command not found" messages
- Permission denied messages
- Built-in error messages (e.g., from `cd`, `read`, etc.)
- Mail notification messages
- Any other shell-generated diagnostic output

**Applies to:** All shell modes (interactive and non-interactive).

**Details:** If LC_MESSAGES is unset or null, the value of LANG is used. If LC_ALL is set and non-empty, it overrides LC_MESSAGES. In the POSIX locale, messages are in English. The shell should use the locale to select the appropriate message catalog or format strings.
