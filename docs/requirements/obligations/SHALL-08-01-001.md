## Context

For a C-language program, an array of strings called the environment shall be made available when a process begins, pointed to by the external variable `environ`. Environment variable values are arbitrary byte sequences except for the null byte.

**Shell relevance:** The shell must maintain an environment (key=value pairs) and pass it to child processes via `execve()`. The shell's `export` builtin marks variables for inclusion in the environment of subsequently executed commands. See XBD 8.1.
