## Extracted Text

> The following exit values shall be returned:: An error occurred.

## Context

**Behavior:** Describes exit status >1: a getopts processing error occurred. Examples include inability to set variables (readonly), or other internal failures. This is the only exit status that indicates a problem with getopts itself, as opposed to the application's argument usage.

**Spec reference:** POSIX.1-2024, `getopts` EXIT STATUS, line 209 of `utilities/getopts.html`.
