## Extracted Text

> Unless specified otherwise (see trap), traps that are not being ignored shall be set to the default action.

## Context

**Duplicate of SHALL-19-13-006: subshell resets non-ignored traps to default.**

Same obligation as SHALL-19-13-006. When creating a subshell, traps that are not being ignored (i.e., have an active handler) **shall** be set to the default action. Traps set to ignore (`trap '' SIG`) remain ignored. See SHALL-19-13-006 for full implementation details. The `trap` built-in (see 2.29 in utilities/V3_chap02.html) may specify additional nuances.
