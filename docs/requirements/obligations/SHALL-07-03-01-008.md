## Extracted Text

> Define characters to be classified as letters and numeric digits. Only the characters specified for the alpha and digit keywords shall be specified. Characters specified for the keywords alpha and digit are automatically included in this class.

## Context

**Behavior:** The `alnum` keyword defines the set of alphanumeric characters:
- Exactly the union of `alpha` and `digit` classes. No additional characters allowed.
- Characters from `alpha` and `digit` are automatically included.

**Shell relevance:** The shell uses the `alnum` class for:
- `[[:alnum:]]` bracket expression in pattern matching.
- Variable name validation (shell variable names use `[A-Za-z_][A-Za-z0-9_]*`, which overlaps with alnum).
- In practice, the shell may use `isalnum()` for name character validation.
