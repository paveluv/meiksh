## Extracted Text

> The following commands shall be recognized in command mode:: Move the current cursor position to the first character on the input line that is not a <blank>.

## Context

**Behavior:** In vi command mode, `^` moves the cursor to the first non-blank character on the input line.

**Implementation:** Scan from position 0 forward, skipping blank characters (space, tab per current locale's `blank` character class). Position cursor on the first non-blank character found. If the line contains only blanks, position at the end of the line. This is a motion command.
