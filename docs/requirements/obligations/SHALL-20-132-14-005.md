## Extracted Text

> The following exit values shall be returned:: An error occurred.

## Context

**Behavior:** This is the description for exit status >0 of `umask`. A non-zero exit status indicates that an error occurred. Typical errors include:
- Invalid mask operand (e.g., `umask abc`, `umask 999`)
- Unrecognized option (e.g., `umask -Z`)
- Malformed symbolic mode string

**Error handling:** When an error occurs, a diagnostic message should be written to stderr and the file mode creation mask should remain unchanged.
