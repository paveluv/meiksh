## Extracted Text

> A string that shall be used to indicate a negative-valued formatted monetary quantity.

## Context

The `negative_sign` value is the string displayed to indicate a negative monetary value (e.g., "-"). The shell does not directly format monetary values. The shell must correctly propagate the LC_MONETARY environment variable so programs can access this through `localeconv()`.

See: `basedefs/V1_chap07.html` §7.3.3 LC_MONETARY.
