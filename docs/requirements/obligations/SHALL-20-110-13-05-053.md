## Extracted Text

> The following commands shall be recognized in command mode:: Move to the first occurrence of the character 'c' that occurs before the current cursor position. If the cursor was positioned on the first character of the line, the terminal shall be alerted and the cursor shall not be moved. If the character 'c' does not occur in the line before the current cursor position, the terminal shall be alerted and the cursor shall not be moved.

## Context

**Behavior:** In vi command mode, `Fc` moves the cursor to the first occurrence of character `c` that occurs BEFORE the current cursor position (searching leftward).

**Edge cases:**
- If cursor is on the first character: alert the terminal (bell), do NOT move.
- If character `c` does not occur before cursor: alert the terminal, do NOT move.
- With count: find the count-th occurrence of `c` before cursor.

**Implementation:** Search backward from cursor-1 for character `c`. On count > 1, find the count-th match. If not found, ring bell. This is a motion command.
