## Extracted Text

> A BRE or ERE that shall match either a single character or a single collating element.

## Context

**Behavior:** A "BRE (ERE) matching a single character" is defined as a BRE or ERE that shall match either a single character or a single collating element.

**Shell relevance:** This definition applies to shell pattern matching elements that match a single character: `?` (matches any single character), bracket expressions `[...]` (match a single character from the set), and literal characters. In the shell, a collating element can be a multi-character sequence if defined by the locale (e.g., `[[.ch.]]`).
