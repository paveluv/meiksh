## Extracted Text

> The standard input shall be used only if one of the following is true:: The -s option is specified.

## Context

**Behavior:** When the `-s` option is specified on the `sh` command line, the shell shall read commands from standard input.

**Conditions:** This applies regardless of whether operands are present. With `-s`, operands become positional parameters (`$1`, `$2`, ...) rather than script filenames. The shell reads its command stream from stdin until EOF.

**Applies to:** Both interactive and non-interactive invocations. If stdin is a terminal and `-s` is given (and `-i` is not explicitly disabled), the shell is interactive.
