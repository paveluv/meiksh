## Extracted Text

> The following options shall be supported:: -signal_number

## Context

**Behavior:** The `kill` built-in must support the `-signal_number` option form (XSI extension). This allows specifying the signal as a numeric option, e.g., `kill -9 pid`. See SHALL-20-64-04-010 and SHALL-20-64-04-013 for the defined signal number mappings and disambiguation rules.

**Conditions:** This is an XSI option. The signal_number is a non-negative decimal integer.
