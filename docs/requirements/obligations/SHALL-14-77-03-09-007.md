## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflush():: Flush untransmitted output.

## Context

**Behavior:** This is the description of TCOFLUSH: **flush untransmitted output**. When `tcflush(fd, TCOFLUSH)` is called, all output that has been written but not yet physically transmitted to the terminal is discarded.

**Conditions:** Used as the `queue_selector` parameter to `tcflush()`.

**Shell relevance:** See SHALL-14-77-03-09-006. Discards pending output from the terminal output buffer.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 477.
