## Extracted Text

> If IXANY is set, any input character shall restart output that has been suspended.

## Context

**Behavior:** When IXANY is set in `c_iflag`, any input character shall restart output that has been suspended (e.g., by a STOP character when IXON is set). Without IXANY, only a START character restarts output.

**Relevance to meiksh:** The `stty` builtin must correctly set/clear the IXANY flag. The actual flow control behavior is handled by the OS terminal driver.

**Spec reference:** POSIX.1-2024, §11.2.2 Input Modes (tag_11_02_02), `basedefs/V1_chap11.html` line 526.
