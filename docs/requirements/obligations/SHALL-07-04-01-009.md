## Extracted Text

> The following tokens shall be processed (in addition to those string constants shown in the grammar):: A symbolic name, enclosed between angle brackets. The string cannot duplicate any charmap symbol defined in the current charmap (if any), or a COLLELEMENT symbol.

## Context

COLLSYMBOL is defined as a symbolic name enclosed between angle brackets. The name cannot duplicate any charmap symbol defined in the current charmap or a COLLELEMENT symbol.

**Shell relevance:** Collation symbols can appear in `[.sym.]` bracket expressions in shell pattern matching. See XBD 7.4.1.
