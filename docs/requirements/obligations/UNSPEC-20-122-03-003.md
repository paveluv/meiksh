> When time is used in any of the following circumstances, via a simple command for which the word time is the command name (see 2.9.1.1 Order of Processing), and none of the characters in the word time is quoted, the results (including parsing of later words) are unspecified:: The next word that follows time would, if the word time were not present, be recognized as a reserved word (see 2.4 Reserved Words) or a control operator (see XBD 3.85 Control Operator).

## Context

**Behavior (unspecified):** When unquoted `time` is the command name and the next word would (if `time` were not present) be recognized as a reserved word (see 2.4) or a control operator (see XBD 3.85), the results are unspecified. Examples: `time while ...`, `time ( cmd )`, `time;`, `time shift`.
