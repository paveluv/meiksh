## Context

The vertical-line `|` is an ERE special character used as the alternation operator. It has special meaning everywhere except inside a bracket expression. Inside a bracket expression, it matches a literal `|`. When preceded by an unescaped backslash (`\|`), it matches the literal `|` character. See SHALL-09-04-03-009 for additional constraints on placement and undefined-result conditions.
