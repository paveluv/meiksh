## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: TCION

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCION` for use with `tcflow()`. When used, a START character is transmitted to the terminal to request that the remote end resume sending data.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** TCION reverses the effect of TCIOFF. The shell could use `tcflow(fd, TCION)` to resume input after a previous TCIOFF pause. In practice, this is rarely needed directly by shells.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 485.
