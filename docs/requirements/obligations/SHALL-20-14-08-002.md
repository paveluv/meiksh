## Extracted Text

> The following environment variables shall affect the execution of cd:: CDPATH

## Context

**Behavior:** The `CDPATH` environment variable must affect `cd` execution. CDPATH is a colon-separated list of directory pathnames used during directory resolution. When a relative directory operand is given that does not start with `.` or `..`, cd must search each CDPATH entry by concatenating it with the operand to find a valid directory (step 5 of the cd algorithm). The variable name itself (CDPATH) must be recognized by the cd implementation.

**Applies to:** `cd` invocations with relative directory operands that don't start with `.` or `..`.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 178.
