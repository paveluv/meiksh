## Extracted Text

> The following environment variables shall affect the execution of getopts:: OPTIND

## Context

**Behavior:** `OPTIND` is listed as an environment variable that affects `getopts`. It shall be used by getopts as the index of the next argument to be processed. See SHALL-20-53-08-013/014 for the detailed semantics.

**Key facts about OPTIND:**
- Initialized to 1 when the shell starts.
- Updated by `getopts` on each invocation to track parsing state.
- Setting `OPTIND=1` resets getopts parsing.
- Not exported by default (see SHALL-20-53-03-016).

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, line 155 of `utilities/getopts.html`.
