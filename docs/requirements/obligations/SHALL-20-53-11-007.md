## Extracted Text

> If an option requiring an option-argument is found, but an option-argument is not found, this error shall be identified and the invalid option character shall be identified in the message.

## Context

**Behavior:** Duplicate extraction of SHALL-20-53-11-004. When a required argument is missing (verbose mode), the stderr diagnostic must identify both the error type (missing argument) and the option character. The format is unspecified.

**Spec reference:** POSIX.1-2024, `getopts` STDERR, lines 185-186 of `utilities/getopts.html`.
