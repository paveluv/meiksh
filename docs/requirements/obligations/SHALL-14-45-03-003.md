## Extracted Text

> The <signal.h> header shall define the timespec structure as described in <time.h>.

## Context

The `<signal.h>` header must provide the `timespec` structure definition (used for timed signal operations). For `meiksh`, this is relevant if the shell uses `sigtimedwait()` or similar timed signal-waiting functions. This is a system header requirement—the shell depends on the platform providing this structure when `<signal.h>` is included.
