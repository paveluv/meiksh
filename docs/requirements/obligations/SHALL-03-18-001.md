## Extracted Text

> When the User Portability Utilities option is supported, requirements placed on applications relating to the use of standard utilities shall also apply to the actions of a user who is entering shell command language statements into an interactive shell.

## Context

**Requirement — direct shell relevance.** When the User Portability Utilities (UP) option is supported, any requirements placed on "applications" regarding use of standard utilities SHALL also apply to a user typing shell commands interactively. This means the interactive shell user is treated as an "application" for conformance purposes.

**Shell impact:** If `meiksh` supports the UP option, the interactive shell must enforce the same constraints on utility usage that apply to scripts. For example, if a utility specifies that an application "shall not" do something, the interactive user is equally bound. In practice, this means `meiksh` should not provide special relaxations for interactive use that would violate application-level requirements.
