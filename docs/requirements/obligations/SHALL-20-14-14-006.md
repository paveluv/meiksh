## Extracted Text

> The following exit values shall be returned:: >0

## Context

**Behavior:** The cd built-in must return an exit status greater than 0 in the error case described in SHALL-20-14-14-007. This is the exit value label ">0" for when either -e or -P is not in effect and an error occurred during cd.

**Applies to:** Error conditions when cd is used without both -e and -P together.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 248.
