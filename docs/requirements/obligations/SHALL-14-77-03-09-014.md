## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: Suspend output.

## Context

**Behavior:** This is the description of TCOOFF: **suspend output**. When `tcflow(fd, TCOOFF)` is called, the terminal driver suspends output to the terminal device. Output remains buffered and will be transmitted when output is resumed.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** See SHALL-14-77-03-09-013. Programmatic equivalent of the ^S flow control character for output.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 488.
