## Extracted Text

> The standard input shall be used only if one of the following is true:: The -c option is not specified and no operands are specified.

## Context

**Behavior:** When the `-c` option is NOT specified and no operands (script filenames) are given, the shell shall read commands from standard input.

**Conditions:** This is the default invocation mode (e.g., running `sh` with no arguments). If stdin is a terminal, the shell operates interactively. If stdin is a pipe or file, it reads commands from that source.

**Applies to:** The shell startup logic must detect the absence of both `-c` and filename operands and fall through to reading stdin.

**Spec reference:** STDIN section of `utilities/sh.html`, line 137–138.
