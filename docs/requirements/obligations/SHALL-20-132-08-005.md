## Extracted Text

> The following environment variables shall affect the execution of umask:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

**Behavior:** LC_ALL, when set to a non-empty string, shall override all other internationalization variables for the `umask` utility. This is the standard POSIX i18n override behavior.

**Conditions:** If LC_ALL is unset or null, individual LC_* variables and LANG take effect. If LC_ALL is non-empty, it takes precedence over everything.
