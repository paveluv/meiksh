## Extracted Text

> The following rules can be used to construct BREs matching multiple characters from BREs matching a single character:: The concatenation of BREs shall match the concatenation of the strings matched by each component of the BRE.

## Context

**Behavior:** The concatenation of BREs shall match the concatenation of the strings matched by each component of the BRE.

**Shell relevance:** In shell pattern matching, the same concatenation principle applies: the pattern `a?b` is a concatenation of three pattern elements (literal `a`, `?`, literal `b`), and it matches a string that is the concatenation of what each element matches (the character `a`, any single character, the character `b`). This is fundamental to how shell patterns work.
