## Extracted Text

> The following commands shall be recognized in command mode:: Insert the character '#' at the beginning of the current command line and treat the resulting edit line as a comment. This line shall be entered into the command history; see fc.

## Context

**Behavior:** In vi command mode, the `#` command shall insert the character `#` at the beginning of the current command line, converting it into a comment, and then execute it (which effectively makes the shell ignore it as a comment). The resulting line shall be entered into the command history.

**Implementation:** (1) Insert '#' at position 0 of the line buffer, (2) add the line (with '#') to command history, (3) submit for execution (the shell parser treats `#...` as a comment, producing no commands), (4) display next prompt. This allows users to "save" a command in history without executing it.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 404–405. See also `utilities/fc.html` for command history.
