## Context

The `alias` builtin **shall** support the operands described in its OPERANDS section: bare `alias-name` (to display a definition), `alias-name=string` (to set a definition), and no operands (to display all definitions). The shell must parse the operand list to distinguish between these forms based on the presence of `=` in each operand. Reference: alias utility OPERANDS.
