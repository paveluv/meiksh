## Extracted Text

> The following variables shall affect the execution of the shell:: Determine the language in which messages should be written.

## Context

This defines the semantics of the `LC_MESSAGES` variable.

**Required behavior:**
- `LC_MESSAGES` determines the language in which diagnostic and informational messages are written by the shell.
- This affects shell error messages (syntax errors, command not found, etc.) and any locale-aware message formatting the shell performs.

**Conditions:** Applies to all shell-generated messages. The shell should use the `LC_MESSAGES` locale category when formatting its own output messages.

See 2.5.3 "Shell Variables" (`LC_MESSAGES`) in `utilities/V3_chap02.html`.
