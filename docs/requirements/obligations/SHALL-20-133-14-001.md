## Extracted Text

> The following exit values shall be returned:

## Context

**Behavior:** The `unalias` utility shall return exit status 0 on success and >0 on error. This is an enumeration preamble for the exit status values.

**Exit codes:**
- 0: All specified aliases were successfully removed (or `-a` succeeded).
- >0: One or more alias-name operands did not represent a valid alias definition, or another error occurred.
