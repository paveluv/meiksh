## Extracted Text

> The following shall be declared as a function and may also be defined as a macro. A function prototype shall be provided.

## Context

The `<sys/times.h>` header shall declare `times()` as a function and may also define it as a macro, with a function prototype provided: `clock_t times(struct tms *)`. This function fills the `tms` structure with CPU time accounting information and returns the elapsed real time. For `meiksh`, `times()` is called to implement the `times` builtin (which prints accumulated user and system times for the shell and its children) and may also be used for the `time` reserved word timing. The shell must include `<sys/times.h>` to access this function. See POSIX `<sys/times.h>` DESCRIPTION (basedefs/sys_times.h.html lines 47–49).
