## Extracted Text

> Description: IEXTEN -- Enable extended (implementation-defined) functions.

## Context

**Behavior:** The IEXTEN flag in `c_lflag` enables implementation-defined extended terminal functions. The specific extended functions available when IEXTEN is set are implementation-defined.

**Relevance to meiksh:** meiksh must document what extended functions (if any) are recognized when IEXTEN is set. The `stty` builtin must support the `iexten`/`-iexten` operands. See SHALL-11-02-05-008 for the behavioral requirements of IEXTEN.

**Spec reference:** POSIX.1-2024, §11.2.5 Local Modes (tag_11_02_05), `basedefs/V1_chap11.html` lines 1144–1145.
