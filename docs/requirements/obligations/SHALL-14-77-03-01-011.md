## Context

**Behavior:** `<termios.h>` **shall** define the symbolic constant NCCS (see SHALL-14-77-03-01-012 and SHALL-14-77-03-01-013). This introduces the section defining NCCS.

**Conditions:** Must be defined after including `<termios.h>`.

**Shell relevance:** NCCS defines the size of the `c_cc` array in the termios structure. meiksh may use NCCS to iterate over all control characters or to validate array bounds.
