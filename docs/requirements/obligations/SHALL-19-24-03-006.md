## Extracted Text

> The shell shall format the output, including the proper use of quoting, so that it is suitable for reinput to the shell as commands that achieve the same value and readonly attribute-setting results in a shell execution environment in which:: Variables that were unset at the time they were output do not have a value at the time at which the saved output is reinput to the shell.

## Context

**Behavior:** Condition for `readonly -p` reinput guarantee: variables that were unset at the time they were output do **not** have a value at the time the saved output is reinput. This ensures the `readonly name` (without value) commands in the output correctly mark the variables as readonly-and-unset.

**Applies to:** All shell modes.
