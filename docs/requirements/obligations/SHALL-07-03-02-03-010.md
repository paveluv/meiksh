## Extracted Text

> Specifies that comparison operations for the weight level shall proceed from end of string towards the beginning of string.

## Context

In the LC_COLLATE `order_start` keyword, the `backward` directive specifies that collation comparison for a given weight level proceeds from the end of the string toward the beginning. This is used in some locales (e.g., French accented character sorting where secondary weights are compared in reverse). The shell is affected indirectly: pattern matching bracket expression ranges (`[a-z]`) and any locale-aware string comparisons depend on the collation order defined by LC_COLLATE. The shell delegates to system collation routines (`strcoll()`) which honor the backward directive when the active locale specifies it.

See: `basedefs/V1_chap07.html` §7.3.2.3 The order_start Keyword.
