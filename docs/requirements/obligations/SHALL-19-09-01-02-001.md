## Extracted Text

> Variable assignments shall be performed as follows:

## Context

**Behavior:** Introductory statement establishing that variable assignments in simple commands follow specific rules depending on the command type. The detailed rules are in SHALL-19-09-01-02-002 through SHALL-19-09-01-02-012, covering five cases: no command name, regular commands, standard utilities implemented as functions, special built-in utilities, and user-defined functions.

**Applies to:** All simple commands with variable assignments (e.g., `VAR=value cmd`, `VAR=value`, `VAR=value export X`).

**Implementation detail:** The variable assignment behavior depends on the classification of the command name (or absence thereof). The shell must dispatch to different assignment behaviors based on the command type determined during command search.

**Spec reference:** Section 2.9.1.2, line 1761 in `utilities/V3_chap02.html`.
