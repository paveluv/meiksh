## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: SS_DISABLE

## Context

The `<signal.h>` header shall define the symbolic constant `SS_DISABLE`. This is an [XSI] constant used with `sigaltstack()`. When set in `ss_flags` of a `stack_t` passed to `sigaltstack()`, it disables the alternate signal stack. When returned in the old stack info, it indicates the alternate stack was disabled. For `meiksh`, this constant is not directly needed since the shell does not use alternate signal stacks. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html line 546).
