## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: SA_RESETHAND

## Context

The `<signal.h>` header shall define the symbolic constant `SA_RESETHAND`. This flag is used in the `sa_flags` member of `struct sigaction`. When set, the signal disposition is reset to `SIG_DFL` upon entry to the signal handler (one-shot behavior). For `meiksh`, this flag may be relevant when the shell needs to handle a signal exactly once and then revert to the default action (e.g., for certain fatal signals). See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html line 524).
