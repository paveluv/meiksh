## Extracted Text

> The following operands shall be supported:: A decimal integer specifying a signal number or the exit status of a process terminated by a signal.

## Context

**Behavior:** The `exit_status` operand is a decimal integer that may represent:
1. A raw signal number (e.g., `9` for SIGKILL) — `kill -l 9` should output `KILL`.
2. The exit status of a process terminated by a signal (typically 128 + signal_number in most shells) — `kill -l 137` should output `KILL` (137 = 128 + 9).

The implementation must handle both interpretations as described in SHALL-20-64-04-004/SHALL-20-64-04-011.
