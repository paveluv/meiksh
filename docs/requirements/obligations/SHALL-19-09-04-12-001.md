## Extracted Text

> The exit status of the until loop shall be the exit status of the last compound-list-2 executed, or zero if none was executed.

## Context

The `until` loop (`until compound-list-1; do compound-list-2; done`) repeatedly executes compound-list-2 as long as compound-list-1 returns **non-zero** (the inverse of `while`). The exit status rules are:

1. **If compound-list-2 was executed at least once**: the exit status is the exit status of the **last compound-list-2** that was executed (i.e., from the final iteration of the loop body).
2. **If compound-list-2 was never executed** (compound-list-1 returned zero on the first check): the exit status is **zero**.

Examples:
- `until true; do echo hi; done` — exit status is 0 (body never ran, condition was zero immediately)
- `x=0; until [ $x -ge 3 ]; do x=$((x+1)); false; done` — exit status is 1 (last body execution returned 1)

See 2.9.4.11 The until Loop in utilities/V3_chap02.html#tag_19_09_04_11.
