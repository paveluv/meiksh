## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: TCOON

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCOON` for use with `tcflow()`. When used, suspended output is resumed — the terminal driver resumes sending output to the terminal device.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** TCOON reverses the effect of TCOOFF. The shell could use `tcflow(fd, TCOON)` to resume output that was programmatically suspended. Equivalent to the terminal receiving a START character for output.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 489.
