## Extracted Text

> The cnd_wait() and cnd_timedwait() functions shall synchronize memory on all calls that release and re-acquire the specified mutex, both when the mutex is released and when it is re-acquired.

## Context

This is a C11 threads-level requirement about C11 condition variable memory synchronization. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.2 Memory Synchronization.
