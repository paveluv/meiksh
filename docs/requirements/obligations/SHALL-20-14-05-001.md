> The following operands shall be supported:

## Context

**Behavior:** The `cd` builtin must accept the operands defined in the OPERANDS section. Specifically, `cd` supports a single optional `directory` operand. If no operand is given, `cd` uses $HOME. If the operand is `-`, `cd` uses $OLDPWD. The shell must not reject any of these forms as syntax errors.

**Applies to:** The `cd` special regular built-in utility.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section OPERANDS (tag_20_14_05), line 152.
