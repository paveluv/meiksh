## Extracted Text

> Handle the operand dot-dot logically; symbolic link components shall not be resolved before dot-dot components are processed (see steps 8. and 9. in the DESCRIPTION).

## Context

This reiterates the `-L` (logical) mode behavior for `cd`: symbolic link components **shall not** be resolved before dot-dot components are processed. The shell manipulates the logical path string when handling `..` — it removes the preceding path component from the string without checking if it was a symlink. This corresponds to steps 8 and 9 of the cd DESCRIPTION algorithm. Reference: cd utility OPTIONS, cd utility DESCRIPTION steps 8-9.
