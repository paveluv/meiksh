## Extracted Text

> If a process has appropriate privileges: If read, write, or directory search permission is requested, access shall be granted. If execute permission is requested, access shall be granted if execute permission is granted to at least one user by the file permission bits or by an alternate access control mechanism; otherwise, access shall be denied.

## Context

**Requirement — moderate shell relevance.** Restatement of the privileged-process access rule (same content as SHALL-04-07-006): read/write/search always granted for privileged processes; execute only if at least one user has execute permission.

**Shell impact:** Same as SHALL-04-07-006. This appears to be a duplicate extraction. See that obligation for details.
