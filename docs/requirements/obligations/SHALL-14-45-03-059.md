## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: [XSI] Default size in bytes for the alternate signal stack.

## Context

This is the [XSI] extension defining the semantics of `SIGSTKSZ`: it is the default size in bytes for the alternate signal stack. Applications allocating an alternate stack should use at least `SIGSTKSZ` bytes. For `meiksh`, this is not directly needed since the shell does not employ alternate signal stacks. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 553–554).
