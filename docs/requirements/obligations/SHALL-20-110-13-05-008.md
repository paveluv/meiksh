## Extracted Text

> Redraw the current command line. Position the cursor at the same location on the redrawn line.

## Context

**Behavior:** In vi command mode, `Ctrl+L` shall redraw the current command line on the terminal. The cursor shall be positioned at the same location on the redrawn line as it was before the redraw.

**Implementation:** Clear the current terminal line(s) and re-output the prompt and command line buffer. Restore the cursor to the same character position. This is useful when terminal output has been garbled by background process output.
