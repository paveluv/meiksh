## Extracted Text

> If the file is an empty directory (see XBD 3.119 Empty Directory): If no process has the directory open, the space occupied by the directory shall be freed and the directory shall no longer be accessible. If one or more processes have the directory open, the directory contents shall be preserved until all references to the file have been closed.

## Context

Duplicate extraction of SHALL-18-01-01-05-004: for removed empty directories, if no process has them open, space is freed; if processes have them open, contents are preserved until all references close. For `meiksh`, this affects the shell when its CWD is removed. See POSIX Utilities Chapter 1, Section 1.1.1.5 item 3 (utilities/V3_chap01.html lines 700–705).
