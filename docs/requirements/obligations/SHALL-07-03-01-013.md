## Extracted Text

> Define characters to be classified as printable characters, including the <space>. In the POSIX locale, all characters in class graph shall be included; no characters in class cntrl shall be included. In a locale definition file, characters specified for the keywords upper, lower, alpha, digit, xdigit, punct, graph, and the <space> are automatically included in this class. No character specified for the keyword cntrl shall be specified.

## Context

**Behavior:** The `print` keyword defines printable characters including space:
- In the POSIX locale: `graph` class plus space. No `cntrl` characters.
- Automatically includes all `upper`, `lower`, `alpha`, `digit`, `xdigit`, `punct`, `graph`, and space.
- Constraint: No `cntrl` character may be in `print`.

**Shell relevance:** The shell uses the `print` class for:
- `[[:print:]]` bracket expression in pattern matching.
- `print` defines all displayable characters (including space), useful for determining what characters can be safely displayed to a terminal.
