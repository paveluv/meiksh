## Extracted Text

> The following environment variables shall affect the execution of sh:: HISTFILE

## Context

**Behavior:** The HISTFILE environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-007).

**Applies to:** Interactive shells using command history (User Portability [UP] option).

**Details:** HISTFILE determines the pathname of the command history file. See SHALL-20-110-08-007 for the complete behavior specification including fallback to `$HOME/.sh_history`, access requirements, initialization timing, and deletion ordering.
