## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: t_fmt

## Context

The `t_fmt` keyword shall be recognized in an LC_TIME locale definition. It introduces the definition of the time format string.

**Shell relevance:** This format is used by `%X` in `date`. In the POSIX locale: "%H:%M:%S". See XBD 7.3.5.1.
