## Extracted Text

> All lines of the here-document shall be expanded, when the redirection operator is evaluated but after the trailing delimiter for the here-document has been located, for parameter expansion, command substitution, and arithmetic expansion. If the redirection operator is never evaluated (because the command it is part of is not executed), the here-document shall be read without performing any expansions.

## Context

This is a duplicate extraction of the expansion-timing rule from SHALL-19-07-04-005. See 2.7.4 Here-Document in utilities/V3_chap02.html.

**Behavior required:**
- When the delimiter is unquoted, all lines of the here-document body SHALL be expanded for parameter expansion, command substitution, and arithmetic expansion.
- Expansion occurs when the redirection is evaluated, after the trailing delimiter is located.
- If the redirection is never evaluated (command not executed), the here-document is read but not expanded.

See SHALL-19-07-04-005 for full context. This obligation and SHALL-19-07-04-005 cover the same requirement from the same source line.
