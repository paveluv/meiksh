> The following environment variables shall affect the execution of kill:: NLSPATH

## Context

**Behavior:** The `NLSPATH` environment variable must be recognized by the `kill` built-in (XSI extension). It determines the location of message objects and message catalogs used for internationalized diagnostic messages. See SHALL-20-64-08-011.
