## Extracted Text

> Move to the start of the next word. If the cursor was positioned on the last character of the line, the terminal shall be alerted and the cursor shall not be advanced. If the count is larger than the number of words after the cursor, this shall not be considered an error; the cursor shall advance to the last character on the line.

## Context

**Behavior:** The `w` command shall move to the start of the next word. At last char: alert, don't advance. Count exceeding remaining: not an error, go to last char.

**Details:** Same normative content as SHALL-20-110-13-05-031. This is a motion command. See `utilities/vi.html` for word definition.

**Spec reference:** vi Line Editing Command Mode in `utilities/sh.html`, line 470–472.
