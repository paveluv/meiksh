## Extracted Text

> The fg utility shall write the command line of the job to standard output in the following format:

## Context

When `fg` successfully brings a job to the foreground, it must write the command line of that job to stdout in the format `"%s\n", <command>`. The `<command>` is the original command line text of the job as it was entered. This output appears before the job resumes execution in the foreground. Only the command text is printed — no job number or status prefix. Reference: POSIX.1-2024 §fg, STDOUT (tag_20_45_10).
