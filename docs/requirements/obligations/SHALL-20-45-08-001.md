> The following environment variables shall affect the execution of fg:

## Context

The `fg` builtin must honor the following environment variables: `LANG`, `LC_ALL`, `LC_CTYPE`, `LC_MESSAGES`, and `NLSPATH` (XSI). These control locale-related behavior for diagnostic messages and character interpretation. See companion obligations SHALL-20-45-08-002 through SHALL-20-45-08-011 for individual descriptions. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
