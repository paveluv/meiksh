## Extracted Text

> The following operand shall be supported:: The name of an alias to be removed.

## Context

**Behavior:** This is the description for the `alias-name` operand. Each operand specifies the name of an alias that shall be removed from the current shell execution environment's alias table.

**Conditions:** The alias name must match an existing alias definition exactly. If the specified name is not a currently defined alias, the `unalias` utility shall report an error for that name (contributing to a non-zero exit status).
