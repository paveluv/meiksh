## Extracted Text

> The following environment variables shall affect the execution of fc:: LC_CTYPE

## Context

The `fc` builtin must recognize the `LC_CTYPE` environment variable, which determines the locale for interpretation of byte sequences as characters. This affects how `fc` processes multi-byte characters in command text, arguments, and input files (the temporary edit file). The standard `LC_ALL` > `LC_CTYPE` > `LANG` precedence applies. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
