## Context

The LC_TIME locale definition file shall recognize a set of mandatory keywords that define time-related locale data. These keywords include: `copy`, `abday`, `day`, `abmon`, `ab_alt_mon`, `mon`, `alt_mon`, `d_t_fmt`, `d_fmt`, `t_fmt`, `am_pm`, `t_fmt_ampm`, `era`, `era_d_fmt`, `era_t_fmt`, `era_d_t_fmt`, and `alt_digits`. A locale definition parser must accept all of these keywords in the LC_TIME category block. Unrecognized keywords or missing mandatory keywords are errors.

**Shell relevance:** The shell's `printf` builtin `%(fmt)T` and the `date` utility rely on the LC_TIME category. The shell must load or delegate to the system locale for time formatting. See XBD 7.3.5.1.
