## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: Restart output.

## Context

**Behavior:** This is the description of TCOON: **restart output**. When `tcflow(fd, TCOON)` is called, output that was previously suspended (via TCOOFF or a STOP character) is resumed.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** See SHALL-14-77-03-09-015. Programmatic equivalent of the ^Q flow control character for output resumption.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 490.
