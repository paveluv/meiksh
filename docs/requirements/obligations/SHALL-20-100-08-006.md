## Extracted Text

> The following environment variables shall affect the execution of read:: LC_ALL

## Context

**Behavior required:** The `LC_ALL` environment variable must affect the execution of the `read` builtin. When set to a non-empty string, it overrides all other internationalization variables.

**Condition:** When LC_ALL is set to a non-empty string value.

**Implementation detail:** LC_ALL overrides LC_CTYPE (character interpretation for field splitting), LC_MESSAGES (diagnostic messages), and all other locale categories for `read`.

**Spec reference:** Section ENVIRONMENT VARIABLES of `utilities/read.html`, line 157.
