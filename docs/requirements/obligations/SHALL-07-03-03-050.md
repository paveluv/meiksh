## Extracted Text

> The operand is a string containing the symbol that shall be used as the decimal delimiter (radix character) in monetary formatted quantities.

## Context

The `mon_decimal_point` value is the string used as the decimal delimiter (radix character) in monetary formatted quantities (e.g., "." or ","). The shell does not directly format monetary values. The shell must correctly propagate the LC_MONETARY environment variable so programs can access this through `localeconv()`.

See: `basedefs/V1_chap07.html` §7.3.3 LC_MONETARY.
