## Extracted Text

> When in insert mode, an entered character shall be inserted into the command line, except as noted in vi Line Editing Insert Mode. Upon entering sh and after termination of the previous command, sh shall be in insert mode.

## Context

**Behavior:** In vi-mode **insert mode**:
1. Each entered character is inserted into the command line (at the cursor position), except for special characters noted in the vi Line Editing Insert Mode section (e.g., ESC, backspace, Ctrl-W, etc.).
2. Upon entering `sh` (shell startup) and after termination of the previous command (when a new prompt is displayed), the shell must be in **insert mode** by default.

**Implementation:** The line editor must start in insert mode. After each command completes and a new prompt is shown, the editor resets to insert mode. The user switches to command mode by pressing ESC.

**Applies to:** Interactive shell in vi editing mode.

**Spec reference:** Command Line Editing (vi-mode) section in `utilities/sh.html`, lines 333-335. Cross-ref: vi Line Editing Insert Mode section.
