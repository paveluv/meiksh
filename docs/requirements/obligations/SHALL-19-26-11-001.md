## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `set` special builtin (section 2.26 in utilities/V3_chap02.html#tag_19_26) must use standard error **only** for diagnostic messages. Note that `set` writes to stdout when invoked without arguments (listing all shell variables) or with `-o`/`+o` without arguments (listing option settings), but stderr must be reserved for diagnostics.

Diagnostic messages that may be written to stderr include:
- Invalid option name (e.g., `set -o nonexistent_option`)
- Invalid arguments

See 2.26 set in utilities/V3_chap02.html#tag_19_26.
