## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `eval` special builtin (section 2.20 in utilities/V3_chap02.html#tag_19_20) must use standard error **only** for diagnostic messages. It must not write any non-diagnostic output to stderr. Output from commands executed by `eval` comes from those commands, not from `eval` itself.

Diagnostic messages that may be written to stderr include:
- Syntax errors in the evaluated string

See 2.20 eval in utilities/V3_chap02.html#tag_19_20.
