## Extracted Text

> The following environment variables shall affect the execution of jobs:: LC_CTYPE

## Context

**Behavior:** `LC_CTYPE` determines the locale for interpreting byte sequences as characters in `jobs` arguments. This affects how `job_id` operands and command names are interpreted in multi-byte locales.

**Spec reference:** POSIX.1-2024, `jobs` ENVIRONMENT VARIABLES, line 93 of `utilities/jobs.html`.
