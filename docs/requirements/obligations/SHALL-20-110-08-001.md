## Extracted Text

> The following environment variables shall affect the execution of sh:

## Context

**Behavior:** This is the introductory statement for the `sh` utility's ENVIRONMENT VARIABLES section. The shell is affected by a specific set of environment variables: ENV, FCEDIT, HISTFILE, HISTSIZE, HOME, LANG, LC_ALL, LC_COLLATE, LC_CTYPE, LC_MESSAGES, MAIL, MAILCHECK, MAILPATH, NLSPATH, PATH, and PWD.

**Applies to:** All invocations of `sh`, both interactive and non-interactive.

**Details:** The shell must recognize and honor each of these environment variables according to their individual specifications. Some are conditional on interactive mode (ENV, MAIL, MAILCHECK, MAILPATH), some are conditional on the UP (User Portability) option (ENV, FCEDIT, HISTFILE, HISTSIZE, MAIL, MAILCHECK, MAILPATH), and some on XSI (NLSPATH). The shell must read these from the inherited environment at startup and respond to changes as specified by each variable's individual rules.
