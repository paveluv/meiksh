## Extracted Text

> The following exit values shall be returned:: The current working directory was successfully changed and the value of the PWD environment variable was set correctly.

## Context

**Behavior:** Exit status 0 must be returned when the working directory was successfully changed AND the PWD environment variable was correctly set to the new directory's pathname. This is the normal success case for cd.

**Conditions:** Both conditions must be true:
1. The `chdir()` equivalent succeeded (directory was changed).
2. PWD was set to the correct value (either the logical path without -P, or the physical path with -P).

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 243.
