## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: Specify the name of an existing locale which shall be used as the definition of this category. If this keyword is specified, no other keyword shall be specified.

## Context

When the `copy` keyword is specified in an LC_TIME locale definition, the operand shall name an existing locale whose LC_TIME category definition is used. If `copy` is specified, no other keyword shall appear in the LC_TIME category block. Specifying both `copy` and another keyword is an error.

**Shell relevance:** Informational for locale infrastructure. The shell relies on system locale data and does not parse locale definition files directly. See XBD 7.3.5.1.
