## Extracted Text

> This variable shall represent a pathname of a directory made available for programs that need a place to create temporary files.

## Context

**Behavior:** The `TMPDIR` environment variable shall represent a pathname of a directory available for programs to create temporary files.

**Shell relevance:** The shell uses `TMPDIR` for creating temporary files needed by here-documents (the shell must store here-document content in temporary files). If `TMPDIR` is set, the shell should create its temporary files in that directory. If unset, the shell should fall back to `/tmp` or another system default. The shell must propagate `TMPDIR` to child processes.
