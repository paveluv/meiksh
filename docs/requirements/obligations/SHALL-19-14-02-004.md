## Extracted Text

> The <asterisk> ('*') is a pattern that shall match any string, including the null string.

## Context

**Duplicate of SHALL-19-14-02-001: `*` matches any string including the null string.**

Same obligation as SHALL-19-14-02-001. The `*` pattern **shall** match any string, including the empty string. See SHALL-19-14-02-001 for full details.
