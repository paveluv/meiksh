## Extracted Text

> If no directory operand is given and the HOME environment variable is set to a non-empty value, the cd utility shall behave as if the directory named in the HOME environment variable was specified as the directory operand.

## Context

When `cd` is invoked with no directory operand and `HOME` is set to a non-empty value, the shell **shall** behave as if the value of `HOME` was specified as the directory operand. This means `cd` with no arguments is equivalent to `cd "$HOME"`. The shell must read the `HOME` variable and use it as the target directory for the remaining steps of the `cd` algorithm. Reference: cd utility DESCRIPTION step 2.
