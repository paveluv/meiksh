## Extracted Text

> A sequence of characters starting with a <dollar-sign> immediately followed by a single-quote ($') shall preserve the literal value of all characters up to an unescaped terminating single-quote ('), with the exception of certain <backslash>-escape sequences, as follows:: \v yields a <vertical-tab> character.

## Context

**Behavior:** Inside `$'...'`, the escape sequence `\v` yields a vertical-tab character.

**Applies to:** All shell modes during `$'...'` escape processing.

**Implementation detail:** Emit byte 0x0B (VT/vertical-tab). This is the same character that C's `'\v'` produces.

**Spec reference:** Section 2.2.4, lines 199-200 in `utilities/V3_chap02.html`.
