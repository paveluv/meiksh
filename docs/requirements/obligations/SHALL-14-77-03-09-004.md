## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflush():: TCIOFLUSH

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCIOFLUSH` for use with `tcflush()`. When used, both pending input (unread data) and pending output (untransmitted data) are discarded.

**Conditions:** Used as the `queue_selector` parameter to `tcflush()`.

**Shell relevance:** meiksh uses `tcflush(fd, TCIOFLUSH)` when it needs a clean slate — discarding both stale input and unsent output. This may be useful when aborting a command and resetting the terminal state.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 474.
