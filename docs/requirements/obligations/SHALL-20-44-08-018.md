## Extracted Text

> This variable, when expanded by the shell, shall determine the default value for the -e editor option's editor option-argument. If FCEDIT is null or unset, ed shall be used as the editor.

## Context

This obligation (duplicate extraction of the `FCEDIT` description) specifies that when `fc` needs an editor and `-e` was not given, it must expand `FCEDIT` and use the result as the editor name. The expansion is standard shell expansion (parameter expansion, command substitution, etc. if applicable). If the expansion yields an empty string or `FCEDIT` is unset, the implementation must default to `ed`. The editor name is resolved via `PATH` search. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
