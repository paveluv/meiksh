## Extracted Text

> The following environment variables shall affect the execution of command:: LC_MESSAGES

## Context

**Behavior:** The `LC_MESSAGES` environment variable must affect `command` execution. LC_MESSAGES determines the locale for diagnostic messages (stderr) and informative messages (stdout) produced by `command`.

**Applies to:** Error messages from `command` and the output of `-V` (which is locale-dependent in its wording).

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section ENVIRONMENT VARIABLES (tag_20_22_08), line 152.
