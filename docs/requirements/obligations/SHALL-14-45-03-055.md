## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: [XSI] Alternate signal stack is disabled.

## Context

This is the [XSI] extension defining the semantics of `SS_DISABLE`: the alternate signal stack is disabled. When this value is in `ss_flags` of the `stack_t` returned by `sigaltstack()`, it means no alternate stack is in use. When passed to `sigaltstack()`, it disables any previously established alternate stack. For `meiksh`, this is not directly used. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 547–548).
