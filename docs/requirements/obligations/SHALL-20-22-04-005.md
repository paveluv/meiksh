## Extracted Text

> The following options shall be supported:: -v

## Context

**Behavior:** The `command` built-in must support the `-v` option. When specified, `command -v name` writes a concise string to stdout indicating how the shell would resolve `name`, without actually executing it.

**Applies to:** `command -v command_name` invocations.
