## Extracted Text

> The <signal.h> header shall also define the following symbolic constants:: SA_RESTART

## Context

The `<signal.h>` header shall define the symbolic constant `SA_RESTART`. This flag is used in the `sa_flags` member of `struct sigaction`. When set, certain functions that are interrupted by this signal are automatically restarted rather than failing with `EINTR`. For `meiksh`, this is important: the shell typically wants `SA_RESTART` set for signals like `SIGCHLD` so that system calls like `read()` and `wait()` are not interrupted during normal operation. Without `SA_RESTART`, the shell would need to manually retry every interruptible system call. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html line 528).
