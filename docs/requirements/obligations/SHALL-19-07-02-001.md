## Extracted Text

> where the optional n represents the file descriptor number. If the number is omitted, the redirection shall refer to standard output (file descriptor 1).

## Context

Specifies the default file descriptor for output redirection when `n` is omitted. See 2.7.2 Redirecting Output in utilities/V3_chap02.html.

**Behavior required:**
- If the fd number `n` is omitted from `[n]>word` or `[n]>|word`, the redirection defaults to standard output (file descriptor 1).
- Example: `>file` is equivalent to `1>file`.

This applies to both the `>` and `>|` (clobber-override) output redirection operators.
