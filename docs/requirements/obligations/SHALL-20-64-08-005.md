## Extracted Text

> The following environment variables shall affect the execution of kill:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

**Behavior:** When `LC_ALL` is set to a non-empty string, it overrides the values of all other internationalization variables (`LANG`, `LC_CTYPE`, `LC_MESSAGES`, etc.) for the `kill` built-in's locale determination.
