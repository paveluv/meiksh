## Extracted Text

> The time utility shall invoke the utility named by the utility operand with arguments supplied as the argument operands and write a message to standard error that lists timing statistics for the utility. The message shall include the following information:: The System CPU time, equivalent to the sum of the tms_stime and tms_cstime fields returned by the times() function for the process in which utility is executed.

## Context

**Behavior:** The timing message shall include System CPU time, equivalent to the sum of `tms_stime` and `tms_cstime` fields returned by the `times()` function for the process in which the utility is executed.

**Meaning:** `tms_stime` is system (kernel) CPU time of the process itself; `tms_cstime` is the cumulative system CPU time of all waited-for child processes.

**Implementation:** Call `times()` before and after executing the utility. The system CPU time is `(after.tms_stime + after.tms_cstime) - (before.tms_stime + before.tms_cstime)`, divided by `sysconf(_SC_CLK_TCK)` to convert to seconds.
