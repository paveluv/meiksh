## Extracted Text

> The following environment variable shall affect the execution of exec:: PATH

## Context

**Behavior:** The PATH environment variable is the specific variable that shall affect the `exec` built-in. When `exec utility [argument...]` is invoked, PATH is used to locate the named utility.

**Applies to:** All shell modes when `exec` is invoked with a utility operand.

**Details:** PATH provides the search path for finding the utility to exec. The exec built-in uses PATH to find the utility, following the same search rules defined in XBD 8.3 Other Environment Variables. When `exec` is used with only redirections (no utility operand), PATH is not relevant.

**Spec reference:** Section 2.21 "exec" ENVIRONMENT VARIABLES, line 3910 in `utilities/V3_chap02.html`.
