## Extracted Text

> The following exit values shall be returned:

## Context

**Behavior required:** The `read` builtin must return the exit values specified in the subsequent obligations: 0 (success), 1 (EOF), >1 (error).

**Condition:** Every invocation of `read`.

**Implementation detail:** This is a preamble obligation. The actual values are in SHALL-20-100-14-002 through SHALL-20-100-14-007. Note that `read` has a three-level exit status (0, 1, >1) unlike most utilities which only have two (0, >0).

**Spec reference:** Section EXIT STATUS of `utilities/read.html`, line 196.
