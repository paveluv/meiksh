## Extracted Text

> The meaning of "enclosing" shall be as specified in the description of the break utility.

## Context

**Behavior:** The definition of "enclosing" for `continue` is the same as for `break` — it uses lexical enclosure as defined in the `break` description (SHALL-19-16-03-002 through SHALL-19-16-03-005). A loop lexically encloses `continue` if the same three conditions are met: same execution environment, contained in the loop's compound-list, and not inside a function definition within the loop.

**Applies to:** All shell modes.

**Spec reference:** `continue` DESCRIPTION in `utilities/V3_chap02.html`, line 3462. Cross-ref: `break` description.
