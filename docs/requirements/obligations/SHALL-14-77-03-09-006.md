## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflush():: TCOFLUSH

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCOFLUSH` for use with `tcflush()`. When used, pending output data that has not been transmitted is discarded.

**Conditions:** Used as the `queue_selector` parameter to `tcflush()`.

**Shell relevance:** meiksh may use `tcflush(fd, TCOFLUSH)` to discard unsent output, e.g., when interrupting a command that was producing output and the shell wants to immediately display a new prompt without waiting for the remaining output to be transmitted.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 476.
