## Extracted Text

> A value that uniquely identifies each thread in a process during the thread's lifetime. The value shall be unique across all threads in a process, regardless of whether the thread is:

## Context

**Definition only — no shell relevance.** Defines "thread ID" as a per-process unique value for each thread during its lifetime. Thread IDs SHALL be unique across all threads in a process regardless of creation method. This is a preamble for -002 through -005.

**Shell impact:** None. Thread IDs are a C threading concept. The shell command language is process-based, not thread-based. No action required.
