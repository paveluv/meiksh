## Extracted Text

> The following environment variables shall affect the execution of getopts:: LANG

## Context

**Behavior:** The `LANG` environment variable shall affect `getopts` execution. It provides a default value for unset or null internationalization variables (LC_*). This is the standard i18n fallback variable per XBD 8.2.

**Relevance to getopts:** Affects locale determination for diagnostic messages (stderr output in verbose mode) and character interpretation.

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, line 139 of `utilities/getopts.html`.
