## Extracted Text

> In a locale definition, the following mandatory keywords shall be recognized:: Define the full weekday names, corresponding to the %A conversion specification. The operand shall consist of seven <semicolon>-separated strings, each surrounded by double-quotes. The first string is the full name of the day corresponding to Sunday, the second the full name of the day corresponding to Monday, and so on.

## Context

The `day` keyword defines full weekday names corresponding to the `%A` conversion specification. The operand shall consist of exactly seven semicolon-separated strings, each surrounded by double-quotes. The first string is the full name for Sunday, the second for Monday, and so on.

**Constraints:** Exactly 7 strings required. Sunday-first order. In the POSIX locale: "Sunday";"Monday";"Tuesday";"Wednesday";"Thursday";"Friday";"Saturday".

**Shell relevance:** Used by `date +%A`. See XBD 7.3.5.1.
