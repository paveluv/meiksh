## Extracted Text

> where the fields shall be as follows:: A number that can be used to identify the job to the wait, fg, bg, and kill utilities. Using these utilities, the job can be identified by prefixing the job number with '%'.

## Context

**Behavior:** The `<job-number>` is a positive integer assigned to each background job. It can be used with `wait`, `fg`, `bg`, and `kill` by prefixing it with `%` (e.g., `%1`, `%2`). The shell assigns job numbers sequentially and reuses numbers of completed/removed jobs.

**Cross-references:** The `%n` format is part of the Job ID syntax defined in XBD 3.182.

**Spec reference:** POSIX.1-2024, `jobs` STDOUT, lines 132-135 of `utilities/jobs.html`.
