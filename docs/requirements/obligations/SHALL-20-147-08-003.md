## Extracted Text

> The following environment variables shall affect the execution of wait:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

**Behavior:** LANG provides default values for any i18n variables that are unset or null during `wait` execution. The precedence is: LC_ALL overrides all, then specific LC_* variables, then LANG as fallback.

**Conditions:** For `wait`, this primarily affects diagnostic message formatting (LC_MESSAGES) and character interpretation in arguments (LC_CTYPE).

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_147_08) in `utilities/wait.html`, lines 94-95. See XBD 8.2.
