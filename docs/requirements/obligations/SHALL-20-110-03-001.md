## Extracted Text

> The sh utility is a command language interpreter that shall execute commands read from a command line string, the standard input, or a specified file. The application shall ensure that the commands to be executed are expressed in the language described in 2. Shell Command Language.

## Context

**Behavior:** The `sh` utility is a command language interpreter that must execute commands read from:
1. A command line string (via `-c` option).
2. Standard input.
3. A specified file (script argument).

The commands must be expressed in the Shell Command Language described in chapter 2 of the POSIX specification.

**Applies to:** All invocation modes of the shell.
