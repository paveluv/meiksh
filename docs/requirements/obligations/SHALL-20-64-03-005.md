## Extracted Text

> The value of the pid operand shall be used as the pid argument.

## Context

**Behavior:** The pid operand value from the command line **shall** be passed directly as the `pid` argument to the `kill()` system call. This is a restatement/duplicate of SHALL-20-64-03-003.

**Conditions:** The pid operand is a signed integer parsed from the command line arguments.

**Shell relevance:** See SHALL-20-64-03-003 for full details. The `kill` builtin must correctly parse and pass the pid value. Special values: 0 signals the current process group, -1 signals all permissible processes, negative values signal specific process groups.

**Spec reference:** POSIX.1-2024, `kill` DESCRIPTION (tag_20_64_03), `utilities/kill.html` lines 49–50.
