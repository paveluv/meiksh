## Extracted Text

> The following operands shall be supported:

## Context

The `fc` builtin must support the following operands: `first` and `last` (to select commands from the history by number, negative offset, or string prefix), and `old=new` (for string substitution with `-s`). All operands are optional. The specific behavior of each operand depends on the synopsis form being used (`-l`, `-s`, or edit mode). See companion obligations SHALL-20-44-05-002 through SHALL-20-44-05-006 for details. Reference: POSIX.1-2024 §fc, OPERANDS (tag_20_44_05).
