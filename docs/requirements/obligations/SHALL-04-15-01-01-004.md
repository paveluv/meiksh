## Extracted Text

> The value of a non-atomic scalar object M, as determined by evaluation B, shall be the value stored by the visible side effect A.

## Context

This is a C-level memory model requirement about visible side effects on non-atomic objects. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.1.1 Data Races.
