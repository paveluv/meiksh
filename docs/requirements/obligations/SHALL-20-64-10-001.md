## Extracted Text

> When the -l option is not specified, the standard output shall not be used.

## Context

**Behavior:** When `kill` is invoked without `-l`, it must not write anything to stdout. Successful signal-sending mode is silent on standard output.

**Conditions:** Applies whenever `-l` is absent from the command line.

**Spec reference:** Section STDOUT (tag_20_64_10) in `utilities/kill.html`, line 188.
