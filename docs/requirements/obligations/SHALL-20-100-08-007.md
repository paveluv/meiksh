## Extracted Text

> The following environment variables shall affect the execution of read:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

**Behavior required:** This is the descriptive text for LC_ALL: when set to a non-empty string, it overrides all other internationalization variables for the `read` builtin.

**Condition:** When LC_ALL is non-empty.

**Implementation detail:** With LC_ALL set, all locale-dependent behavior of `read` (character interpretation, diagnostic messages) uses the locale specified by LC_ALL, ignoring individual LC_* settings and LANG.
