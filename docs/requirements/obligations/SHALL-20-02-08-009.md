## Extracted Text

> The following environment variables shall affect the execution of alias:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

`LC_MESSAGES` **shall** determine the locale that affects the format and contents of diagnostic messages written to stderr by `alias`. When an alias name is not found or another error occurs, the error message text should be formatted according to the `LC_MESSAGES` locale. Reference: alias utility ENVIRONMENT VARIABLES.
