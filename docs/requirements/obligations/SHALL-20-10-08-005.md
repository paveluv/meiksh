## Extracted Text

> The following environment variables shall affect the execution of bg:: If set to a non-empty string value, override the values of all the other internationalization variables.

## Context

If `LC_ALL` is set to a non-empty string value, it **shall** override all other internationalization variables for the `bg` builtin. This is the highest-priority locale override. Reference: bg utility ENVIRONMENT VARIABLES, XBD 8.2.
