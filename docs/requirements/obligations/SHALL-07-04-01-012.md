## Extracted Text

> The following tokens shall be processed (in addition to those string constants shown in the grammar):: CHARCLASS

## Context

The CHARCLASS token shall be recognized in the locale definition grammar. It represents a character class name.

**Shell relevance:** Character class names (e.g., `[:alpha:]`) are used in shell pattern matching bracket expressions. See XBD 7.4.1.
