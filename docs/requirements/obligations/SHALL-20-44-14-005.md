## Extracted Text

> The following exit values shall be returned:: An error occurred.

## Context

This is the description for exit status >0 of `fc`: "An error occurred." This covers any error condition in `fc` operation, including but not limited to: invalid options, history file access errors, editor launch failures, temp file I/O errors, and write failures during listing. Reference: POSIX.1-2024 §fc, EXIT STATUS (tag_20_44_14).
