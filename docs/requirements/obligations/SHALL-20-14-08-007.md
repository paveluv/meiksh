## Extracted Text

> The following environment variables shall affect the execution of cd:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

**Behavior:** This is the description of LANG's role for cd. LANG provides default values for all internationalization variables (LC_CTYPE, LC_MESSAGES, etc.) that are unset or null. The precedence order is: LC_ALL overrides everything; if LC_ALL is not set, each individual LC_* variable is used; if that is not set, LANG provides the default.

**Applies to:** All locale-dependent behavior of cd (diagnostic messages, character interpretation).

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), lines 185-186. See XBD 8.2 Internationalization Variables.
