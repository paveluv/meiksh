## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcsetattr():

## Context

**Behavior:** `<termios.h>` **shall** define symbolic constants for use as the `optional_actions` argument to `tcsetattr()`. These constants control when the attribute change takes effect: TCSANOW, TCSADRAIN, TCSAFLUSH.

**Conditions:** These constants must be defined and usable with `tcsetattr()`.

**Shell relevance:** meiksh must choose the appropriate action when changing terminal attributes:
- **TCSANOW**: change immediately (use when reading input in non-canonical mode)
- **TCSADRAIN**: change after output drains (use when switching modes that affect output)
- **TCSAFLUSH**: change after output drains and flush input (use when entering line editor to discard typeahead)

**Spec reference:** POSIX.1-2024, `<termios.h>` Attribute Selection (tag_14_77_03_08), `basedefs/termios.h.html` lines 456–457.
