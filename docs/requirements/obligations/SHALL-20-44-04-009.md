## Extracted Text

> The following options shall be supported:: -r

## Context

The `fc` builtin must support the `-r` option, which reverses the order of commands. When used with `-l`, the listed commands are displayed in reverse chronological order. When used in edit mode (without `-l` or `-s`), the commands are presented to the editor in reverse order. The `-r` option interacts with the `first`/`last` operands: if `first` represents a newer command than `last`, the commands are already in reverse order (equivalent to using `-r` with the operands swapped). Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
