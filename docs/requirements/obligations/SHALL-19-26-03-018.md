## Extracted Text

> The remaining arguments shall be assigned in order to the positional parameters. The special parameter '#' shall be set to reflect the number of positional parameters. All positional parameters shall be unset before any new values are assigned.

## Context

**Behavior:** When `set` is called with non-option arguments (after options are processed or after `--`):
1. All existing positional parameters must be **unset** first.
2. The remaining arguments are assigned in order to `$1`, `$2`, `$3`, etc.
3. The special parameter `$#` is updated to reflect the new count.

**Example:** `set -- a b c` sets `$1=a`, `$2=b`, `$3=c`, `$#=3`.

**Applies to:** All shell modes.
