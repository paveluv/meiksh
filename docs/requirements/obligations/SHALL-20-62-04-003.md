## Extracted Text

> The following options shall be supported:: -l

## Context

**Behavior:** The `jobs` utility must support the `-l` option (lowercase letter ell). This option provides more detailed information about each job, including process group IDs for job-control jobs or process IDs for non-job-control jobs. See SHALL-20-62-04-004 and SHALL-20-62-10-014 through SHALL-20-62-10-017 for the detailed output format.

**Mutual exclusivity:** `-l` and `-p` are mutually exclusive per the synopsis `[-l|-p]`.

**Spec reference:** POSIX.1-2024, `jobs` OPTIONS, line 54 of `utilities/jobs.html`.
