## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `unset` special builtin (section 2.30 in utilities/V3_chap02.html#tag_19_30) must use standard error **only** for diagnostic messages. It must not write any non-diagnostic output to stderr. The `unset` builtin produces no stdout output.

Diagnostic messages that may be written to stderr include:
- Attempting to unset a readonly variable
- Invalid variable/function name
- Invalid options

See 2.30 unset in utilities/V3_chap02.html#tag_19_30.
