## Extracted Text

> The value n shall be an unsigned decimal integer less than or equal to the value of the special parameter '#'. If n is not given, it shall be assumed to be 1. If n is 0, the positional and special parameters are not changed.

## Context

**Behavior:** The `n` argument to `shift`:
1. Must be an **unsigned decimal integer** less than or equal to `$#`.
2. If `n` is not given, it defaults to **1**.
3. If `n` is **0**, the positional parameters and `$#` are not changed (no-op).

**Error:** If `n` is greater than `$#`, it is an error.

**Applies to:** All shell modes.
