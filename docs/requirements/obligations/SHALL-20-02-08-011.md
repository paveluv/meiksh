## Extracted Text

> The following environment variables shall affect the execution of alias:: [XSI] Determine the location of messages objects and message catalogs.

## Context

(XSI) `NLSPATH` **shall** determine the location of message objects and message catalogs for `alias`. This is an XSI extension; when supported, the shell uses `NLSPATH` to locate translated message catalogs for diagnostic output. If not supporting XSI, this obligation may be deferred.
