## Extracted Text

> The <signal.h> header shall define the following macros, which shall expand to constant expressions with distinct values that have a type compatible with the second argument to, and the return value of, the signal() function, and whose values shall compare unequal to the address of any declarable function.

## Context

The shell must use or interact with three macros from `<signal.h>`: `SIG_DFL` (default signal handling), `SIG_ERR` (error return from `signal()`), and `SIG_IGN` (ignore signal). These macros must be constant expressions with distinct values, compatible with the `signal()` function signature, and must not equal any valid function pointer. For `meiksh`, these are used when setting up signal handlers (e.g., `SIG_IGN` for background jobs' SIGINT/SIGQUIT, `SIG_DFL` for restoring default handlers). The shell must correctly use these macros when calling `signal()` or `sigaction()`.
