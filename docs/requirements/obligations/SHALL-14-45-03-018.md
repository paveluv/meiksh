## Extracted Text

> The <signal.h> header shall define the following symbolic constants for the values of sigev_notify:: SIGEV_THREAD

## Context

`SIGEV_THREAD` must be defined in `<signal.h>`. When set as `sigev_notify`, a notification function is called in a new thread to perform notification when the event occurs. For `meiksh`, thread-based notification is generally not used (shells are typically single-threaded), but the constant must be available from the system header for completeness. This is a system header requirement.
