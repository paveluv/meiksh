## Extracted Text

> The following environment variables shall affect the execution of umask:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

**Behavior:** LANG shall provide default values for any internationalization variables (LC_CTYPE, LC_MESSAGES, etc.) that are unset or null during `umask` execution. The precedence order is: LC_ALL overrides everything, then specific LC_* variables, then LANG as a fallback.

**Conditions:** This is the standard i18n variable precedence defined in XBD 8.2. For `umask`, this primarily affects the formatting of diagnostic messages (via LC_MESSAGES) and character interpretation in arguments (via LC_CTYPE).

**Spec reference:** Section ENVIRONMENT VARIABLES (tag_20_132_08) in `utilities/umask.html`, lines 98-99. See XBD 8.2 Internationalization Variables.
