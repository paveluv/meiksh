## Extracted Text

> The shell shall format the output, including the proper use of quoting, so that it is suitable for reinput to the shell as commands that achieve the same exporting results, except:: Read-only variables with values cannot be reset.

## Context

**Behavior:** Exception to the reinput guarantee for `export -p`: read-only variables with values cannot be reset. If a variable is both exported and readonly, reinputting the `export -p` output may fail to reset it because the readonly attribute prevents reassignment.

**Applies to:** All shell modes.
