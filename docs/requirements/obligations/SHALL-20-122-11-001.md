## Extracted Text

> If the utility utility is invoked, the standard error shall be used to write the timing statistics and may be used to write a diagnostic message if the utility terminates abnormally; otherwise, the standard error shall be used to write diagnostic messages and may also be used to write the timing statistics.

## Context

**Behavior:** Stderr usage depends on whether the utility was successfully invoked:
1. **If utility is invoked:** stderr shall be used to write the timing statistics. It may also be used for diagnostics if the utility terminates abnormally.
2. **If utility is NOT invoked** (e.g., not found, cannot execute): stderr shall be used for diagnostic messages and may also be used for timing statistics.

**Key point:** Timing statistics always go to stderr, never stdout. The utility's own stdout/stderr are not affected.
