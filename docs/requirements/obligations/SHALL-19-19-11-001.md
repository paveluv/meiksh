## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The dot (`.`) special builtin (section 2.19 in utilities/V3_chap02.html#tag_19_19) must use standard error **only** for diagnostic messages. It must not write any non-diagnostic output to stderr. Note: the dot command executes commands from a file in the current environment; any stdout/stderr from those commands comes from the commands themselves, not from the dot builtin.

Diagnostic messages that may be written to stderr include:
- File not found
- File not readable
- Syntax errors in the sourced file

See 2.19 dot in utilities/V3_chap02.html#tag_19_19.
