## Extracted Text

> The <termios.h> header shall define the following data types through typedef:: cc_t

## Context

**Behavior:** `<termios.h>` **shall** define the type `cc_t` via typedef. This type is used for elements of the `c_cc` array in the termios structure (special control character values and MIN/TIME parameters).

**Conditions:** Must be available after including `<termios.h>`. Shall be an unsigned integer type (see SHALL-14-77-03-01-008).

**Shell relevance:** Used in meiksh for variables holding control character values (e.g., saved VEOF value, VMIN/VTIME settings). The `stty` builtin displays and modifies `cc_t` values.

**Spec reference:** POSIX.1-2024, `<termios.h>` The termios Structure (tag_14_77_03_01), `basedefs/termios.h.html` line 44.
