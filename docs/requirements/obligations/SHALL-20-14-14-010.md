## Extracted Text

> The following exit values shall be returned:: >1

## Context

**Behavior:** The cd built-in must return an exit status greater than 1 in the error case described in SHALL-20-14-14-011. This is the exit value label ">1" for when both -e and -P are in effect and an error occurred during cd (the chdir itself failed).

**Applies to:** `cd -eP` invocations where the directory change itself fails.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 253.
