> The <signal.h> header shall also define the following symbolic constants:

## Context

The `<signal.h>` header shall define a set of symbolic constants used as flags in the `sa_flags` member of `struct sigaction` and for alternate signal stack management. These constants include: `SA_NOCLDSTOP`, `SA_ONSTACK`, `SA_RESETHAND`, `SA_RESTART`, `SA_SIGINFO`, `SA_NOCLDWAIT`, `SA_NODEFER`, `SS_ONSTACK`, `SS_DISABLE`, `MINSIGSTKSZ`, and `SIGSTKSZ`. For `meiksh`, the relevant constants are those used with `sigaction()` to configure signal handling behavior (e.g., `SA_RESTART` for restartable system calls, `SA_NOCLDSTOP` to suppress `SIGCHLD` for stopped children, `SA_SIGINFO` for extended signal information). See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html line 513).
