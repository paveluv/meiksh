## Extracted Text

> The while loop shall continuously execute one compound-list as long as another compound-list has a zero exit status.

## Context

**Behavior:** The `while` loop continuously executes the body compound-list as long as the condition compound-list has a **zero** exit status. Syntax:

```
while compound-list-1 ; do compound-list-2 ; done
```

The condition is tested before each iteration; if it starts as non-zero, the body is never executed.

**Exit status:** The exit status of the `while` loop is the exit status of the last command executed in compound-list-2, or zero if compound-list-2 was never executed.

**Interaction with `-e`:** The condition compound-list (compound-list-1) is executed with `-e` ignored.

**Applies to:** All shell modes.
