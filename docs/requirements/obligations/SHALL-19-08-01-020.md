## Extracted Text

> Interactive Shell: Command not found -- shall not exit

## Context

An interactive shell SHALL NOT exit when a command is not found. See 2.8.1 Consequences of Shell Errors in utilities/V3_chap02.html.

**Behavior required:**
- When the shell is interactive and a command is not found, the shell SHALL write a diagnostic to stderr (e.g., "foo: command not found") but SHALL NOT exit.
- A shell diagnostic message IS required.
- The exit status of the failed command is 127.
- The shell returns to the prompt.
