## Extracted Text

> The following environment variables shall affect the execution of fc:: LC_MESSAGES

## Context

The `fc` builtin must recognize the `LC_MESSAGES` environment variable, which determines the locale for formatting diagnostic messages written to stderr. This affects error messages produced by `fc` (e.g., invalid option, history access failure). The standard `LC_ALL` > `LC_MESSAGES` > `LANG` precedence applies. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
