## Extracted Text

> The following commands shall be recognized in command mode:: Set the current command line to be the countth next command line in the shell command history. If count is not specified, it shall default to 1. The cursor shall be positioned on the first character of the new command. If a j or + command advances past the edit line, the current command line shall be restored to the edit line and the terminal shall be alerted.

## Context

**Behavior:** In vi command mode, `j` (or `+`) sets the current command line to the count-th next (more recent) command in the shell command history (default count = 1). The cursor shall be positioned on the first character of the new command.

**Edge cases:** If `j`/`+` would advance past the edit line (the current/newest line), the current command line is restored to the edit line and the terminal is alerted.

**Implementation:** Navigate forward in the history list by `count` entries. If the target goes beyond the edit line, restore the edit line, ring bell.
