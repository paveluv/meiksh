## Extracted Text

> The following environment variables shall affect the execution of fc:: LANG

## Context

The `fc` builtin must recognize the `LANG` environment variable. `LANG` provides a default value for the internationalization variables (`LC_CTYPE`, `LC_MESSAGES`, etc.) that are unset or null. The standard precedence applies: `LC_ALL` overrides all, then specific `LC_*` variables, then `LANG` as the fallback (per XBD 8.2). Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
