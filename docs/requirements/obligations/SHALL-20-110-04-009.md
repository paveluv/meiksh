## Extracted Text

> The following additional options shall be supported:: Read commands from the standard input.

## Context

The `-s` option directs the shell to read commands from **standard input**. This is the explicit form of the default behavior when no `command_file` and no `-c` are given.

When `-s` is specified:
- Commands are read from stdin (which may be a terminal, pipe, file redirect, etc.)
- If stdin is a terminal and stderr is also a terminal, the shell is interactive
- Any non-option operands become positional parameters (`$1`, `$2`, etc.)

See utilities/sh.html#tag_20_110_04.
