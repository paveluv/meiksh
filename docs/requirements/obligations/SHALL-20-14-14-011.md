## Extracted Text

> The following exit values shall be returned:: Both the -e and the -P options are in effect, and an error occurred.

## Context

**Behavior:** Exit status >1 must be returned when:
1. Both `-e` AND `-P` options are in effect, AND
2. An error occurred (the chdir itself failed, e.g., directory not found, permission denied).

This is distinct from exit status 1 (where chdir succeeded but PWD couldn't be determined). Here the directory change itself failed. The value must be strictly greater than 1.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 254.
