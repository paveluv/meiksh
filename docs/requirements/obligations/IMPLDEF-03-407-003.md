## Extracted Text

> If the initial working directory field is null, the interpretation of that field is implementation-defined.

## Context

**Definition — moderate shell relevance.** States that if the "initial working directory" field in the user database is null, interpretation is implementation-defined. The initial working directory is typically the user's `$HOME` directory.

**Shell impact:** The shell inherits its working directory from the login process. If the user database has no home directory, the shell's initial `$PWD` and `$HOME` may be undefined or set to an implementation-defined default (often `/`). The shell should handle a missing or inaccessible `$HOME` gracefully, particularly during login shell startup when reading `~/.profile`.
