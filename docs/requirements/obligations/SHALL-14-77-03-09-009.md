## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcflow():: TCIOFF

## Context

**Behavior:** `<termios.h>` **shall** define the constant `TCIOFF` for use with `tcflow()`. When used, a STOP character is transmitted to the terminal to request that the remote end suspend sending data.

**Conditions:** Used as the `action` parameter to `tcflow()`.

**Shell relevance:** TCIOFF is used for input flow control. The shell could use `tcflow(fd, TCIOFF)` to temporarily pause input from the terminal (e.g., while processing a large amount of already-buffered input). In practice, this is rarely needed by shells.

**Spec reference:** POSIX.1-2024, `<termios.h>` Line Control (tag_14_77_03_09), `basedefs/termios.h.html` line 483.
