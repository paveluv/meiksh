## Extracted Text

> The following environment variables shall affect the execution of fg:: LC_MESSAGES

## Context

The `fg` builtin must recognize the `LC_MESSAGES` environment variable. `LC_MESSAGES` determines the locale used for formatting diagnostic messages written to stderr by `fg` (e.g., "no such job", "job control not enabled"). The standard precedence applies. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
