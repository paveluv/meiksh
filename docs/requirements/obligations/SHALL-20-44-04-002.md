## Extracted Text

> The following options shall be supported:

## Context

The `fc` builtin must support all of the following options: `-e editor` (specify editor), `-l` (list mode), `-n` (suppress numbers in list), `-r` (reverse order), and `-s` (re-execute without editor). All of these options are mandatory; the implementation must recognize and correctly handle each one. See companion obligations SHALL-20-44-04-003 through SHALL-20-44-04-012 for individual option details. Reference: POSIX.1-2024 §fc, OPTIONS (tag_20_44_04).
