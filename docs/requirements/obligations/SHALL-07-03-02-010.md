## Extracted Text

> The following keywords shall be recognized in a collation sequence definition. They are described in detail in the following sections.

## Context

**Behavior:** The LC_COLLATE locale definition must recognize the following keywords: `copy`, `collating-element`, `collating-symbol`, `order_start`, `order_end`, and related directives. These are detailed in subsections 7.3.2.1 through 7.3.2.5.

**Shell relevance:** The shell does not parse locale definition files. These keywords are for `localedef` input syntax. The resulting compiled collation data is what the shell uses via `strcoll()` and the C library's regex/fnmatch functions.
