## Extracted Text

> The encoded values associated with <period>, <slash>, <newline>, and <carriage-return> shall be invariant across all locales supported by the implementation.

## Context

This is a duplicate of SHALL-06-01-004. The encoded values of `.`, `/`, newline, and carriage return must be the same in all locales. This is essential for the shell's pathname handling and line-oriented parsing. See SHALL-06-01-004 for full context.
