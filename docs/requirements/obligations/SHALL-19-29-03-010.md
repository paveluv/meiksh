## Extracted Text

> If an invalid signal name [XSI] or number is specified, the trap utility shall write a warning message to standard error.

## Context

**Behavior:** If an invalid signal name (or number, for [XSI] systems) is specified to `trap`, the utility must write a **warning message** to standard error. The trap command should not silently ignore invalid conditions.

**Applies to:** All shell modes.
