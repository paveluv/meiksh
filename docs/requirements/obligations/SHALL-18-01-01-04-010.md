## Extracted Text

> The user ID of the file shall be set to the effective user ID of the calling process.

## Context

Duplicate extraction of SHALL-18-01-01-04-003: when a new file is created, its user ID shall be set to the effective user ID of the calling process. For `meiksh`, this is enforced by the OS kernel when the shell creates files via `open()` for redirections. The shell process's effective user ID determines file ownership. See POSIX Utilities Chapter 1, Section 1.1.1.4 item 1 (utilities/V3_chap01.html line 89).
