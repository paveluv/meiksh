## Extracted Text

> Set by the shell to the decimal value of its parent process ID during initialization of the shell. In a subshell (see 2.13 Shell Execution Environment), PPID shall be set to the same value as that of the parent of the current shell. For example, echo $PPID and (echo $PPID ) would produce the same value.

## Context

**Duplicate of SHALL-19-05-03-026.** This obligation covers the same `PPID` variable text from the same source lines. See SHALL-19-05-03-026 for the full enriched context.

**Summary:** `PPID` is set to the parent process ID (decimal) during shell initialization. In a subshell, `PPID` retains the value from the parent of the current (top-level) shell. Both `echo $PPID` and `(echo $PPID)` produce the same value.

See 2.5.3 "Shell Variables" (`PPID`) and 2.13 "Shell Execution Environment" in `utilities/V3_chap02.html`.
