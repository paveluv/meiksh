## Extracted Text

> The following environment variables shall affect the execution of sh:: HOME

## Context

**Behavior:** The HOME environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-011).

**Applies to:** All shell modes (interactive and non-interactive).

**Details:** HOME determines the pathname of the user's home directory and is used in tilde expansion. See SHALL-20-110-08-011 for the complete behavior specification.
