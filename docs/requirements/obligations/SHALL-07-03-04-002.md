## Context

The LC_NUMERIC locale definition must recognize a set of mandatory keywords for non-monetary numeric formatting. Operand values follow the grammar in §7.4. Integer keywords set to -1 or omitted keywords indicate unavailable values. The shell's `printf` builtin is directly affected because it uses LC_NUMERIC's `decimal_point` for floating-point output and may use `thousands_sep`/`grouping` for the grouping flag.

See: `basedefs/V1_chap07.html` §7.3.4 LC_NUMERIC, §7.4 Locale Definition Grammar.
