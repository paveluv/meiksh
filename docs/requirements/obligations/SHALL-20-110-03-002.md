## Extracted Text

> Pathname expansion shall not fail due to the size of a file.

## Context

**Behavior:** Pathname expansion (globbing) must **not** fail due to the size of a file. The shell's glob implementation must not have limits that cause failures when matching against files of any size. This applies to the glob pattern matching itself, not reading file contents.

**Implementation:** Ensure the pathname expansion code does not have fixed buffer limits or size-based error conditions related to file metadata.

**Applies to:** All shell modes.
