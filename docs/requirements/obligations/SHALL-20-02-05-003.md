## Extracted Text

> The following operands shall be supported:: Write the alias definition to standard output.

## Context

When an `alias-name` operand (without `=`) is given to `alias`, the shell **shall** write that alias's definition to stdout in the format specified in the STDOUT section: `alias-name=value` with appropriate quoting. This is the "display" behavior — it outputs the alias definition so it can be reinput to the shell. Reference: alias utility OPERANDS, alias utility STDOUT.
