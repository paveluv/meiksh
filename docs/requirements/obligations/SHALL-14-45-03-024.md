## Extracted Text

> The <signal.h> header shall define the following symbolic constant. The value shall be suitable for use in #if preprocessing directives:: SIG2STR_MAX

## Context

`SIG2STR_MAX` must be defined in `<signal.h>` as a constant usable in `#if` directives. It represents the maximum buffer size for a signal name string from `sig2str()`, including the null terminator. For `meiksh`, signal name buffers (used in `kill -l`, `trap` output, job notifications) should be declared with `char buf[SIG2STR_MAX]` to ensure sufficient space for any signal name.
