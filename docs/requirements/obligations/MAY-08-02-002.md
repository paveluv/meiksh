## Extracted Text

> An implementation may support other formats.

## Context

**Behavior:** An implementation may support locale name formats beyond the standard `language[_territory][.codeset]` format. This is optional — the shell is not required to support additional formats.

**Shell relevance:** If `meiksh` delegates locale handling to the host C library (which it should), additional locale name formats are automatically supported if the host supports them. No special shell-side handling is needed.
