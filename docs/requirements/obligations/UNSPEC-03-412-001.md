## Extracted Text

> A character that in the output stream indicates that printing should start at the next vertical tabulation position. It is the character designated by '\v' in the C language. If the current position is at or past the last defined vertical tabulation position, the behavior is unspecified. It is unspecified whether this character is the exact sequence transmitted to an output device by the system to accomplish the tabulation.

## Context

**Definition — moderate shell relevance.** Defines the vertical-tab character `<vertical-tab>` as `\v` (0x0B). It advances to the next vertical tabulation position. Behavior at or past the last vertical tab stop is unspecified. Whether this byte is the exact terminal sequence is unspecified.

**Shell impact:** The shell must recognize `\v` in escape-sequence contexts: `printf '\v'`, `echo` with XSI interpretation, and `$'\v'` quoting. The shell writes byte 0x0B. Vertical tab is also a member of the `[:space:]` character class, which may affect pattern matching in bracket expressions.
