## Extracted Text

> The <signal.h> header shall define the following symbolic constants for the values of sigev_notify:

## Context

The `<signal.h>` header must define symbolic constants for `sigev_notify` values: `SIGEV_NONE`, `SIGEV_SIGNAL`, and `SIGEV_THREAD`. These control how asynchronous event notifications are delivered. For `meiksh`, these constants are primarily relevant if the shell uses POSIX timers or asynchronous I/O. The shell depends on the system header providing these constants.
