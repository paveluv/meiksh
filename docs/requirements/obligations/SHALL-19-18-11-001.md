## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `continue` special builtin (section 2.18 in utilities/V3_chap02.html#tag_19_18) must use standard error **only** for diagnostic messages. It must not write any non-diagnostic output to stderr. The `continue` builtin produces no stdout output.

Diagnostic messages that may be written to stderr include:
- Invalid operand (e.g., `continue foo` where `foo` is not a number)
- `continue` used outside a loop context
- Operand `n` is less than 1

See 2.18 continue in utilities/V3_chap02.html#tag_19_18.
