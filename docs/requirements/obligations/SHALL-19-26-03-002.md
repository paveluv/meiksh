## Extracted Text

> The value string shall be written with appropriate quoting; see the description of shell quoting in 2.2 Quoting. The output shall be suitable for reinput to the shell, setting or resetting, as far as possible, the variables that are currently set; read-only variables cannot be reset.

## Context

**Behavior:** The value string in `set` output must be written with appropriate quoting (per 2.2 Quoting rules). The output must be suitable for reinput to the shell to set or reset the variables that are currently set. Read-only variables cannot be reset by reinput.

**Implementation:** Values containing special characters (spaces, newlines, quotes, backslashes, etc.) must be quoted so they survive re-parsing.

**Applies to:** All shell modes.
