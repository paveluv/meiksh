## Extracted Text

> The following environment variables shall affect the execution of getopts:: [XSI] Determine the location of messages objects and message catalogs.

## Context

**Behavior:** Describes the effect of `NLSPATH` (XSI extension): determines the location of message objects and message catalogs used for internationalized diagnostic messages from `getopts`.

**Applicability:** This requirement is conditional on XSI conformance. A non-XSI shell implementation is not required to support `NLSPATH`.

**Spec reference:** POSIX.1-2024, `getopts` ENVIRONMENT VARIABLES, lines 152-153 of `utilities/getopts.html`.
