## Extracted Text

> The following environment variables shall affect the execution of sh:: LC_ALL

## Context

**Behavior:** The LC_ALL environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-015).

**Applies to:** All shell modes (interactive and non-interactive).

**Details:** LC_ALL, when set to a non-empty string, overrides all other internationalization variables (LANG, LC_COLLATE, LC_CTYPE, LC_MESSAGES). See SHALL-20-110-08-015 for the full description.
