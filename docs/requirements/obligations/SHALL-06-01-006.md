## Extracted Text

> A null character, NUL, which has all bits set to zero, shall be in the set of characters.

## Context

This is a duplicate of SHALL-06-01-003. The NUL character (all zero bits, byte value 0x00) must exist in the character set. The shell uses NUL as string terminators and cannot embed NUL in variable values. See SHALL-06-01-003 for full context.

See: `basedefs/V1_chap06.html` §6.1 Portable Character Set.
