## Extracted Text

> The <signal.h> header shall define the following symbolic constants for the values of sigev_notify:: No asynchronous notification is delivered when the event of interest occurs.

## Context

This describes the semantics of `SIGEV_NONE`: when an event of interest occurs, no asynchronous notification is delivered. The event is silently consumed. For `meiksh`, this could be used with POSIX timers if the shell wants to poll for timer expiration rather than receive a signal. This is a system header semantics requirement.
