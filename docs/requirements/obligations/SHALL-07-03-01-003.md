## Extracted Text

> Specify the name of an existing locale which shall be used as the definition of this category. If this keyword is specified, no other keyword shall be specified.

## Context

**Behavior:** The `copy` keyword in an LC_CTYPE locale definition imports the LC_CTYPE category from another named locale. When `copy` is used, no other keywords may appear in the LC_CTYPE category body.

**Shell relevance:** The shell does not parse locale definition files. The `copy` directive is for `localedef` to handle. The shell uses the resulting compiled locale data. No shell implementation action needed.
