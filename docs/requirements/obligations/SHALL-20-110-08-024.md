## Extracted Text

> The following environment variables shall affect the execution of sh:: MAILCHECK

## Context

**Behavior:** The MAILCHECK environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-025).

**Applies to:** Interactive shells (User Portability [UP] option).

**Details:** MAILCHECK specifies how often (in seconds) the shell checks for new mail in the files specified by MAILPATH or MAIL. Default is 600 seconds. See SHALL-20-110-08-025 for the full behavior specification.
