## Extracted Text

> The following environment variables shall affect the execution of fg:: LANG

## Context

The `fg` builtin must recognize the `LANG` environment variable. `LANG` provides a default value for the internationalization variables (`LC_CTYPE`, `LC_MESSAGES`, etc.) that are unset or null. The standard precedence applies per XBD 8.2. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08).
