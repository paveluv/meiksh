## Extracted Text

> The following exit values shall be returned:: 0

## Context

**Behavior:** The cd built-in must return exit status 0 in the first success case. This is the exit value label "0" corresponding to the description in SHALL-20-14-14-003 (directory changed successfully and PWD was set correctly).

**Applies to:** Successful cd invocations where PWD is correctly determined.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section EXIT STATUS (tag_20_14_14), line 242.
