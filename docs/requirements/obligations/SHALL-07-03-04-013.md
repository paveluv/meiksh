## Extracted Text

> The operand is a string containing the symbol that shall be used as a separator for groups of digits to the left of the decimal delimiter in numeric, non-monetary formatted monetary quantities. In contexts where standards limit the thousands_sep to a single byte, the result of specifying a multi-byte operand shall be unspecified.

## Context

The `thousands_sep` value in LC_NUMERIC is the string separating digit groups in non-monetary numeric output. Multi-byte separators have unspecified behavior in single-byte contexts. The shell is affected: the `printf` builtin's grouping flag (`%'d`) uses this separator. In the POSIX locale, `thousands_sep` is unavailable (empty), so no grouping occurs by default.
