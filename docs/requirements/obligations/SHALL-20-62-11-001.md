## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

**Behavior:** The `jobs` utility must use stderr exclusively for diagnostic/error messages (e.g., invalid `job_id`, write errors). All normal output (job status information) goes to stdout. No informational content may be written to stderr.

**Spec reference:** POSIX.1-2024, `jobs` STDERR, line 185 of `utilities/jobs.html`.
