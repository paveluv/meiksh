## Extracted Text

> The following environment variables shall affect the execution of cd:: NLSPATH

## Context

**Behavior:** The `NLSPATH` environment variable must affect `cd` execution. This is an XSI extension. NLSPATH determines the location of message catalogs for internationalized messages. For cd, this would affect where translated diagnostic messages are found.

**Applies to:** XSI-conformant systems only. Locale message catalog lookup for cd diagnostics.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 196.
