## Extracted Text

> The following exit values shall be returned:

## Context

**Behavior:** The `umask` utility shall return exit status 0 on success (mask changed or displayed) and >0 on error. This is an enumeration preamble for the exit status values.

**Exit codes:**
- 0: The file mode creation mask was successfully changed, or no mask operand was supplied (query mode succeeded).
- >0: An error occurred (e.g., invalid mask operand, invalid option).
