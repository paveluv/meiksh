## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `fg` builtin must never write normal output to stderr. Standard error must be used exclusively for diagnostic messages such as "no such job," "job not found," "job control not enabled," or other error conditions. All normal output (the job command line) goes to stdout. Reference: POSIX.1-2024 §fg, STDERR (tag_20_45_11).
