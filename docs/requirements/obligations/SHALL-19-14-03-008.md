## Extracted Text

> If a specified pattern does not contain any '*', '?' or '[' characters that will be treated as special, the pattern string shall be left unchanged.

## Context

**Duplicate of SHALL-19-14-03-004: pattern without special glob characters is left unchanged.**

Same obligation as SHALL-19-14-03-004. If a pattern has no unquoted, unescaped `*`, `?`, or `[`, the pattern string **shall** be left unchanged — no filesystem matching is performed. See SHALL-19-14-03-004 for full details.
