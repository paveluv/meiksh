## Extracted Text

> The <termios.h> header shall define the following symbolic constants for use with tcsetattr():: Change attributes when output has drained; also flush pending input.

## Context

**Behavior:** This is the description of TCSAFLUSH: **change attributes when output has drained; also flush pending input**. When `tcsetattr(fd, TCSAFLUSH, &termios)` is called, the function waits for all pending output to be transmitted, then discards any pending input that has not been read, and finally applies the new terminal attributes.

**Conditions:** Used as the `optional_actions` parameter to `tcsetattr()`.

**Shell relevance:** See SHALL-14-77-03-08-006. TCSAFLUSH combines output drain (like TCSADRAIN) with input flush, making it the most thorough mode-switch option.

**Spec reference:** POSIX.1-2024, `<termios.h>` Attribute Selection (tag_14_77_03_08), `basedefs/termios.h.html` line 465.
