## Extracted Text

> Any of the standard utilities may be implemented as regular built-in utilities within the command language interpreter. This is usually done to increase the performance of frequently used utilities or to achieve functionality that would be more difficult in a separate environment. The intrinsic utilities described in 1.7 Intrinsic Utilities below are frequently provided as regular built-ins.

## Context

Any standard utility (e.g., `cat`, `echo`, `printf`, `test`, `[`, `kill`, `true`, `false`) **may** be implemented as a regular built-in within the shell. This is typically done for performance or to achieve functionality difficult in a separate process. The "intrinsic utilities" listed in XCU 1.7 are frequently provided as regular builtins. A "regular builtin" means it participates in normal `PATH` search — if a file with the same name is found earlier in `PATH`, that file takes precedence (unlike special builtins which are always found first). Reference: XCU 1.6, XCU 1.7.
