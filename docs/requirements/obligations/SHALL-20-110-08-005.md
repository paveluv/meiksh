## Extracted Text

> [UP] This variable, when expanded by the shell, shall determine the default value for the -e editor option's editor option-argument. If FCEDIT is null or unset, ed shall be used as the editor.

## Context

**Behavior:** The FCEDIT variable, when expanded, shall determine the default editor for the `fc` built-in's `-e` option:
1. When `fc -e editor` is not explicitly specified, the shell shall expand FCEDIT and use the result as the editor command.
2. If FCEDIT is null (empty string) or unset, `ed` shall be used as the default editor.

**Applies to:** Interactive shells when the `fc` built-in is invoked without an explicit `-e editor` argument.

**Details:** The shell must expand FCEDIT (parameter expansion) and use the result as the pathname of the editor. The `fc` built-in creates a temporary file with the selected history commands, invokes the editor on it, then re-executes the edited commands.

**Feature gate:** This is a [UP] (User Portability) option feature.
