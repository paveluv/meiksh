## Extracted Text

> The until loop shall continuously execute one compound-list as long as another compound-list has a non-zero exit status.

## Context

**Behavior:** The `until` loop continuously executes the body compound-list as long as the condition compound-list has a **non-zero** exit status. Syntax:

```
until compound-list-1 ; do compound-list-2 ; done
```

This is the logical inverse of `while`: the loop continues when the condition **fails** and stops when it **succeeds**.

**Exit status:** The exit status of the `until` loop is the exit status of the last command executed in compound-list-2, or zero if compound-list-2 was never executed.

**Interaction with `-e`:** The condition compound-list (compound-list-1) is executed with `-e` ignored.

**Applies to:** All shell modes.
