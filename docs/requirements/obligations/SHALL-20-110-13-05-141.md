## Extracted Text

> Repeat the most recent f, F, t, or T command. Any number argument on that previous command shall be ignored. However, reverse the direction of that command.

## Context

**Behavior:** The `,` command shall repeat the most recent `f`, `F`, `t`, or `T` command but reverse the direction. The original count is ignored; the current count applies.

**Details:** Same normative content as SHALL-20-110-13-05-061. This is a motion command.
