## Extracted Text

> The following environment variables shall affect the execution of command:: Determine the search path used during the command search described in 2.9.1.4 Command Search and Execution, except as described under the -p option.

## Context

**Behavior:** PATH description for `command`. PATH determines the directories searched when resolving command_name to an executable, as described in the Command Search and Execution algorithm (section 2.9.1.4). When `-p` is specified, a system default PATH is used instead of the current PATH value. The PATH variable is searched in order, and the first matching executable is used.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section ENVIRONMENT VARIABLES (tag_20_22_08), lines 161-163. See also 2.9.1.4.
