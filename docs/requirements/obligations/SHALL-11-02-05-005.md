## Extracted Text

> If ECHONL and ICANON are set, the <newline> character shall be echoed even if ECHO is not set.

## Context

**Behavior:** When both ECHONL and ICANON are set in `c_lflag`, the newline character shall be echoed even if the ECHO flag is not set.

**Conditions:** Both ECHONL and ICANON must be set. Useful for password entry: the user can see line boundaries even when character echo is disabled.

**Relevance to meiksh:** The `stty` builtin must support `echonl`/`-echonl`.

**Spec reference:** POSIX.1-2024, §11.2.5 Local Modes (tag_11_02_05), `basedefs/V1_chap11.html` line 1181.
