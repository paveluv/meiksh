## Extracted Text

> Open FIFO. When attempting to create a regular file, and the existing file is a FIFO special file: If the FIFO is not already open for reading, the attempt shall block until the FIFO is opened for reading. Once the FIFO is open for reading, the utility shall open the FIFO for writing and continue with its operation.

## Context

Duplicate extraction of SHALL-18-01-01-04-020: action code "O" (Open FIFO) — when redirecting to a FIFO, the shell blocks until the FIFO is open for reading, then opens it for writing. For `meiksh`, this means `echo hello > myfifo` will block if no reader has opened `myfifo` yet. See POSIX Utilities Chapter 1, Section 1.1.1.4 action code O (utilities/V3_chap01.html lines 662–665).
