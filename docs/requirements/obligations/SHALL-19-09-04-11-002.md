## Extracted Text

> The compound-list-1 shall be executed, and if it has a zero exit status, the until command completes. Otherwise, the compound-list-2 shall be executed, and the process repeats.

## Context

**Behavior:** Execution of `until compound-list-1 ; do compound-list-2 ; done`:

1. Execute compound-list-1 (condition).
2. If its exit status is **zero** (success), the `until` command completes.
3. Otherwise, execute compound-list-2 (body).
4. Go to step 1 (repeat).

This is the inverse of `while`: the loop body runs when the condition has non-zero exit status.

**Applies to:** All shell modes.
