## Extracted Text

> The following environment variables shall affect the execution of sh:: LC_MESSAGES

## Context

**Behavior:** The LC_MESSAGES environment variable is one of the variables that shall affect the execution of `sh`. This is the variable name heading; the detailed semantics are specified in the subsequent description (see SHALL-20-110-08-021).

**Applies to:** All shell modes (interactive and non-interactive).

**Details:** LC_MESSAGES determines the locale for diagnostic messages written to standard error by the shell. See SHALL-20-110-08-021 for the full description.
