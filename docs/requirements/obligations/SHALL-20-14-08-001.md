## Extracted Text

> The following environment variables shall affect the execution of cd:

## Context

**Behavior:** The `cd` built-in must consult and be affected by the following environment variables: CDPATH, HOME, LANG, LC_ALL, LC_CTYPE, LC_MESSAGES, NLSPATH (XSI), OLDPWD, and PWD. The shell implementation of `cd` must read these variables from the current shell execution environment during its execution.

**Applies to:** Every invocation of the `cd` built-in.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section ENVIRONMENT VARIABLES (tag_20_14_08), line 175.
