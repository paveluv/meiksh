## Extracted Text

> The following environment variables shall affect the execution of fc:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

This is the description of `LC_MESSAGES`'s effect on `fc`. The implementation must use the locale determined by `LC_MESSAGES` when formatting any diagnostic messages written to stderr by the `fc` builtin. This includes error messages for invalid command ranges, history file access problems, missing history entries, and option parsing errors. Reference: POSIX.1-2024 §fc, ENVIRONMENT VARIABLES (tag_20_44_08).
