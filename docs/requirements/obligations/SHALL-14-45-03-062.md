## Context

The `<signal.h>` header shall define the `stack_t` type as a structure with at least the following members: `void *ss_sp` (stack base or pointer), `size_t ss_size` (stack size), and `int ss_flags` (flags such as `SS_ONSTACK` or `SS_DISABLE`). This type is used with `sigaltstack()` to describe alternate signal stacks. For `meiksh`, this type is provided by the platform; the shell does not typically use `sigaltstack()` directly. See POSIX `<signal.h>` DESCRIPTION (basedefs/signal.h.html lines 571–577).
