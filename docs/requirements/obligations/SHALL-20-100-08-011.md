## Context

**Behavior required:** Descriptive text for LC_MESSAGES: it determines the locale used for diagnostic messages written to stderr by `read`.

**Condition:** When `read` encounters errors and writes diagnostics.

**Implementation detail:** Diagnostic messages from `read` (e.g., readonly variable errors, invalid option errors) should be formatted according to the LC_MESSAGES locale. This controls the human-readable text of error messages.
