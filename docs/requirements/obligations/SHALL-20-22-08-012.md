## Extracted Text

> The following environment variables shall affect the execution of command:: PATH

## Context

**Behavior:** The `PATH` environment variable must affect `command` execution. PATH determines the search path used during command search (section 2.9.1.4), except when the `-p` option is specified (which uses a default system PATH instead).

**Applies to:** `command command_name` and `command -v/-V command_name` invocations without `-p`.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section ENVIRONMENT VARIABLES (tag_20_22_08), line 160.
