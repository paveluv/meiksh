## Extracted Text

> The following operands shall be supported:: name

## Context

**Behavior:** The `getopts` utility must accept a `name` operand as its second argument. This is the name of a shell variable that getopts will set to the found option character on each invocation. See SHALL-20-53-05-005/007 for detailed semantics.

**Spec reference:** POSIX.1-2024, `getopts` OPERANDS, line 118 of `utilities/getopts.html`.
