## Extracted Text

> The following environment variables shall affect the execution of read:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

**Behavior required:** This is the descriptive text for LANG's effect on `read`: LANG provides a default value for internationalization variables (LC_*) that are unset or null, per the precedence defined in XBD 8.2.

**Condition:** When specific LC_* variables are unset or null.

**Precedence (per XBD 8.2):**
1. LC_ALL (if non-empty) overrides everything.
2. Specific LC_* variable (if set and non-null).
3. LANG (as default fallback).
