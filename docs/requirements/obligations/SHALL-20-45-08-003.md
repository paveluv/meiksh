## Extracted Text

> The following environment variables shall affect the execution of fg:: Provide a default value for the internationalization variables that are unset or null. (See XBD 8.2 Internationalization Variables for the precedence of internationalization variables used to determine the values of locale categories.)

## Context

This is the description of `LANG`'s effect on `fg`. When any `LC_*` variable is unset or null, `LANG` provides the fallback locale value. The precedence is: `LC_ALL` > specific `LC_*` > `LANG` > POSIX locale default. This affects character interpretation and diagnostic message formatting within `fg`. Reference: POSIX.1-2024 §fg, ENVIRONMENT VARIABLES (tag_20_45_08); XBD 8.2.
