## Extracted Text

> The following options shall be supported:: -p

## Context

**Behavior:** The `jobs` utility must support the `-p` option. This option causes `jobs` to display only process IDs — one per line — rather than the full job status format. See SHALL-20-62-04-006 and SHALL-20-62-10-001 for the output format.

**Mutual exclusivity:** `-p` and `-l` are mutually exclusive per the synopsis `[-l|-p]`.

**Spec reference:** POSIX.1-2024, `jobs` OPTIONS, line 56 of `utilities/jobs.html`.
