## Extracted Text

> Otherwise, the following exit values shall be returned:: An error occurred in the command utility or the utility specified by command_name could not be found.

## Context

**Behavior:** Description for exit status 127: Either:
1. An internal error occurred in the `command` utility itself (e.g., invalid option usage), OR
2. The utility specified by command_name could not be found in PATH (or default PATH with -p).

The shell must return 127 in both cases — it combines "command not found" with "command utility error".

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section EXIT STATUS (tag_20_22_14), line 208.
