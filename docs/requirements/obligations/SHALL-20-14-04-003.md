## Extracted Text

> The following options shall be supported by the implementation:: -e

## Context

The `cd` builtin **shall** support the `-e` option. The `-e` option modifies behavior when used with `-P`: if the `-P` option is in effect, the working directory is successfully changed, but the correct value of `PWD` cannot be determined, `cd` exits with status 1 instead of 0. Without `-P`, `-e` has no specified effect. Reference: cd utility OPTIONS.
