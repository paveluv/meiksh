## Extracted Text

> The following operands shall be supported:: pid

## Context

**Behavior:** The `kill` built-in must accept one or more `pid` operands when not in `-l` (list) mode. Each `pid` operand identifies a target process or process group to receive the signal. See SHALL-20-64-05-003 and SHALL-20-64-05-006 for the detailed semantics of how `pid` values are interpreted.
