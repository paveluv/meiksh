## Context

Exit status >0 **shall** be returned by `alias` when an error occurred. This includes: one or more `alias-name` operands did not have an existing alias definition when querying, or some other error occurred during execution. The specific value >0 is not further specified. Reference: alias utility EXIT STATUS.
