## Extracted Text

> The environment in which the shell executes a trap action on EXIT shall be identical to the environment immediately after the last command executed before the trap action on EXIT was executed.

## Context

**Behavior:** The environment in which the shell executes an EXIT trap action must be **identical** to the environment immediately after the last command executed before the EXIT trap was invoked. This means all variable values, working directory, file descriptors, etc. are preserved as they were right before the exit sequence began.

**Applies to:** All shell modes.
