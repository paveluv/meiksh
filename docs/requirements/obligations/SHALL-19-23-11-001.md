## Extracted Text

> The standard error shall be used only for diagnostic messages.

## Context

The `export` special builtin (section 2.23 in utilities/V3_chap02.html#tag_19_23) must use standard error **only** for diagnostic messages. Note that `export` does write to stdout when invoked without arguments or with `-p` (listing exported variables), but stderr must be reserved for diagnostics.

Diagnostic messages that may be written to stderr include:
- Invalid variable name
- Attempting to export a readonly variable (if an assignment is also attempted)

See 2.23 export in utilities/V3_chap02.html#tag_19_23.
