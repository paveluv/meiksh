## Extracted Text

> [UP] Establish a decimal integer value that specifies how often (in seconds) the shell shall check for the arrival of mail in the files specified by the MAILPATH or MAIL variables. The default value shall be 600 seconds. If set to zero, the shell shall check before issuing each primary prompt.

## Context

**Behavior:** This is a duplicate extraction of the MAILCHECK variable description (same source lines as SHALL-20-110-08-025). The full behavior specification is identical:

1. **Value:** Decimal integer specifying the mail check interval in seconds.
2. **Default:** 600 seconds when unset.
3. **Zero value:** Check before every PS1 prompt.
