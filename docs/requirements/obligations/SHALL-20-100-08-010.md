## Extracted Text

> The following environment variables shall affect the execution of read:: LC_MESSAGES

## Context

**Behavior required:** The `LC_MESSAGES` environment variable must affect the execution of the `read` builtin. It determines the locale for diagnostic messages written to standard error.

**Condition:** When `read` writes diagnostic messages to stderr.

**Implementation detail:** LC_MESSAGES controls the language and format of error messages from `read` (e.g., "read: variable is readonly"). For a minimal POSIX/C locale implementation, English messages suffice.
