## Context

**Behavior:** The `c_iflag` field of the termios structure controls basic terminal input processing. Its values are composed of bitwise-OR of mask constants that shall be **bitwise-distinct** (no two masks share the same bit). The mask symbols are defined in `<termios.h>`. Standard input mode flags include: BRKINT, ICRNL, IGNBRK, IGNCR, IGNPAR, INLCR, INPCK, ISTRIP, IXANY, IXOFF, IXON, PARMRK.

**Conditions:** The flags are set/cleared via `tcgetattr()`/`tcsetattr()`.

**Shell relevance:** The shell must manage input mode flags when configuring the terminal:
- **ICRNL**: CR→NL translation (needed for Enter key to produce NL)
- **IXON/IXOFF**: XON/XOFF flow control (may need to disable for line editor)
- **ISTRIP**: Strip 8th bit (should generally be off for UTF-8)
- **INLCR/IGNCR**: NL/CR handling
The `stty` builtin must support setting/clearing all these flags.
