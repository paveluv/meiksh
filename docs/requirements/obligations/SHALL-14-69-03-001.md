## Context

The `<sys/times.h>` header shall define the `tms` structure, which is returned by `times()` and shall include at least the following members: `clock_t tms_utime` (user CPU time), `clock_t tms_stime` (system CPU time), `clock_t tms_cutime` (user CPU time of terminated child processes), and `clock_t tms_cstime` (system CPU time of terminated child processes). For `meiksh`, this structure is used by the `time` reserved word and the `times` builtin to report CPU time usage. The shell calls `times()` before and after command execution and reports the difference. The `tms_cutime` and `tms_cstime` fields accumulate CPU time from all waited-for child processes. See POSIX `<sys/times.h>` DESCRIPTION (basedefs/sys_times.h.html lines 37–44).
