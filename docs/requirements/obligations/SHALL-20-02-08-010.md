## Extracted Text

> The following environment variables shall affect the execution of alias:: NLSPATH

## Context

`NLSPATH` **shall** affect the execution of `alias`. This XSI extension variable determines the location of message catalogs for the processing of `LC_MESSAGES`. It is used to find translated diagnostic messages. Reference: alias utility ENVIRONMENT VARIABLES.
