## Extracted Text

> The exit status of the while loop shall be the exit status of the last compound-list-2 executed, or zero if none was executed.

## Context

The `while` loop (`while compound-list-1; do compound-list-2; done`) repeatedly executes compound-list-2 as long as compound-list-1 returns zero. The exit status rules are:

1. **If compound-list-2 was executed at least once**: the exit status is the exit status of the **last compound-list-2** that was executed (i.e., from the final iteration of the loop body).
2. **If compound-list-2 was never executed** (compound-list-1 returned non-zero on the first check): the exit status is **zero**.

Examples:
- `while false; do echo hi; done` — exit status is 0 (body never ran)
- `x=0; while [ $x -lt 3 ]; do x=$((x+1)); false; done` — exit status is 1 (last body execution returned 1)

See 2.9.4.9 The while Loop in utilities/V3_chap02.html#tag_19_09_04_09.
