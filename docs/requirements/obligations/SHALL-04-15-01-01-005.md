## Extracted Text

> The value of an atomic object M, as determined by evaluation B, shall be the value stored by some side effect A that modifies M, where B does not happen before A.

## Context

This is a C-level memory model requirement about atomic object value visibility. It is not relevant to shell language semantics. No shell-language-level action is required.

See: `basedefs/V1_chap04.html` §4.15.1.1 Data Races.
