## Extracted Text

> Otherwise, the following exit values shall be returned:: 127

## Context

**Behavior:** Exit status 127 must be returned by `command` (execution mode) when an error occurred in the `command` utility itself, or the utility specified by command_name could not be found. This is the "not found" exit code.

**Applies to:** `command command_name` where command_name does not resolve to any executable, built-in, or utility.

**Spec reference:** POSIX.1-2024, `utilities/command.html`, section EXIT STATUS (tag_20_22_14), line 207.
