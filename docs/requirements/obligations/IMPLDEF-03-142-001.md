## Extracted Text

> The property of a file indicating access permissions for a process related to the group identification of a process. A process is in the file group class of a file if the process is not in the file owner class and if the effective group ID or one of the supplementary group IDs of the process matches the group ID associated with the file. Other members of the class may be implementation-defined.

## Context

**Definition only — low shell relevance.** Defines "file group class" as the access permission property for processes whose effective GID or supplementary GID matches the file's GID, but which are not in the file owner class. Additional members may be implementation-defined.

**Shell impact:** The shell relies on the OS to evaluate file group class membership when checking access permissions (e.g., `test -r`, `test -w`, `test -x`, and command execution). The shell does not compute group class membership itself. No direct action required.
