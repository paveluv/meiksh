## Extracted Text

> The following operands shall be supported:: directory

## Context

**Behavior:** The `cd` builtin must support an optional `directory` operand, which is an absolute or relative pathname of the target directory. This is the primary operand for cd.

**Applies to:** The `cd` built-in when invoked with arguments.

**Spec reference:** POSIX.1-2024, `utilities/cd.html`, section OPERANDS (tag_20_14_05), line 155.
