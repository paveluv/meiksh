> The following environment variables shall affect the execution of unalias:: Determine the locale that should be used to affect the format and contents of diagnostic messages written to standard error.

## Context

**Behavior:** This is the description for LC_MESSAGES' effect on `unalias`. LC_MESSAGES determines the locale used for diagnostic messages written to stderr, such as error messages when an alias-name operand does not match a defined alias.
